# <a name="schema-rowsets"></a>Наборы строк схемы

Схема (или метаданные) — это любая информация о структуре или организации базы данных, например сведения о поставщике, наборе строк, таблице, столбцах или другой информации, помимо фактического содержимого базы данных. OLE DB позволяет получать такие сведения, не открывая набор строк схемы.

Дополнительные сведения об использовании наборов строк схемы см. в разделе [Получение метаданных с помощью наборов строк схемы](../../data/oledb/obtaining-metadata-with-schema-rowsets.md).
