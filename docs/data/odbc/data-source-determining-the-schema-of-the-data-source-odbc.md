# <a name="data-source-determining-the-schema-of-the-data-source-odbc"></a>Источник данных. Определение схемы источника данных (ODBC)

Этот раздел относится к классам ODBC библиотеки MFC.

Чтобы настроить элементы данных в `CRecordset` объектах, необходимо знать схему источника данных, к которому выполняется подключение. Определение схемы источника данных включает в себя получение списка таблиц в источнике данных, список столбцов в каждой таблице, тип данных каждого столбца и существование всех индексов.
