# <a name="work-with-odbc-and-drivers"></a>Работа с ODBC и драйверами

ODBC позволяет создавать приложения, не зависящие только от одной системы управления базами данных (СУБД). Можно написать одно приложение, которое может обрабатывать несколько СУБД, для которых пользователи имеют правильный драйвер ODBC для конкретной СУБД.
