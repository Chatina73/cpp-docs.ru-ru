# <a name="openmp-in-visual-c"></a>OpenMP в Visual C++

Интерфейс прикладного программирования OpenMP C и C++ позволяет создавать приложения, которые эффективно используют несколько процессоров. Visual C++ поддерживает стандарт OpenMP 2,0.
