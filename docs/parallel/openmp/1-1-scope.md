# <a name="11-scope"></a>1.1 Область

Эта спецификация охватывает параллелизации только управляемое пользователем, при котором пользователь явным образом указывает действия, которые необходимо предпринять компилятором и система времени выполнения для параллельного выполнения программы. Реализации API OpenMP C и C++ не требуются для проверки зависимости, конфликты, взаимоблокировки, гонки или других проблем, приводящих к возникновению выполнения неправильные программы. Пользователь несет ответственность за обеспечение правильности выполнения приложения с помощью конструкций API OpenMP C и C++ API. Созданный компилятором автоматическую параллелизацию и директивы компилятора для помощи такой параллелизации, не охваченных в этом документе.
