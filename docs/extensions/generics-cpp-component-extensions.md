# <a name="generics--ccli-and-ccx"></a>Универсальные шаблоны (C++/CLI и C++/CX)

Универсальные шаблоны — это параметризованные типы и методы. В этом разделе вы узнаете, какие универсальные функции поддерживаются как в среда выполнения Windows, так и в общеязыковой среде выполнения, а также какие поддерживаются только среды CLR. Вы также узнаете, как создавать собственные универсальные методы и типы в C++/CLI и как использовать универсальные типы, созданные на .NET Framework языке в C++/CLI. Наконец, в этом разделе сравниваются универсальные шаблоны и шаблоны C++.
