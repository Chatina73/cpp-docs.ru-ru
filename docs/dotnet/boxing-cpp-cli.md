# <a name="boxing-ccli"></a>упаковка-преобразование (C++/CLI)

Упаковка — это процесс преобразования типа значения в тип `object` или в любой тип интерфейса, реализуемый типом значения. Когда поле среды CLR имеет тип значения, оно заключает значение в `System.Object` и сохраняет его в управляемой куче. Операция распаковки извлекает тип значения из объекта. Упаковка является неявной; распаковка является явной.
