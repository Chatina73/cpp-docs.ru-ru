# <a name="how-to-create-clr-empty-projects"></a>Практическое руководство. Создание пустых проектов CLR

Для создания пустого проекта CLR используйте шаблон **CLR Empty Project,** который доступен в диалоговом окне **нового проекта.**

> [!NOTE]
> Появление функций в IDE может зависеть от ваших активных настроек или изданий и может отличаться от тех, которые описаны в Справке. Чтобы изменить параметры, выберите **Импорт и экспорт параметров** в меню **Сервис** . Дополнительные сведения см. в разделе [Персонализация интегрированной среды разработки Visual Studio](/visualstudio/ide/personalizing-the-visual-studio-ide).

### <a name="to-create-a-clr-empty-project"></a>Создание пустого проекта CLR

1. В меню **Файл** выберите **Создать**, а затем пункт **Проект**.

   Раскроется диалоговое окно **Новый проект**.

1. Под **установленными шаблонами**щелкните визуальный узл **C';** затем нажмите на узла **CLR.** Выберите значок **CLR Пустой проект.**

1. В поле **Имя** введите уникальное имя для приложения.

    > [!NOTE]
    >  Вы также можете указать другие параметры проекта и решения из окна диалога **Нового проекта,** но эти настройки не требуются.

1. Нажмите кнопку **ОК**.
