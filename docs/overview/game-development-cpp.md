# <a name="game-development-with-c"></a>Разработка игр на C++

При создании игры на Windows 10 у вас есть возможность привлечь миллионы игроков по всему миру через телефон, ПК и Xbox One. С помощью Xbox в Windows, Xbox Live, функции многопользовательской игры на разных устройствах, потрясающего сообщества игроков и невероятных новых функций, например универсальной платформы Windows (UWP) и DirectX 12, игры на Windows 10 увлекут игроков всех возрастов и жанровых предпочтений. Новая универсальная платформа Windows (UWP) обеспечивает совместимость игры на разных устройствах с Windows 10 за счет единого API для телефона, ПК и Xbox One, а также средств и параметров для адаптации игры к каждому устройству.

Разработка игр описана в [Центре разработки для Windows](https://docs.microsoft.com/windows/uwp/gaming/getting-started).
