# <a name="container-classbegin"></a>Класс контейнера::begin

> [!NOTE]
> Этот раздел находится в Visual Studio C++ документации в качестве нефункционального примера контейнеров, используемых в C++ стандартной библиотеки. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).

Возвращает итератор, указывающий на первый элемент последовательности (или на место сразу за концом пустой последовательности).

## <a name="syntax"></a>Синтаксис

```

    const_iterator begin() const;

iterator begin();
```
