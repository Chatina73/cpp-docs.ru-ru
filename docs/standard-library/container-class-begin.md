# <a name="container-classbegin"></a>Класс контейнера::begin

> [!NOTE]
> Эта статья находится в документации по Visual C++ Studio как нефункциональный пример контейнеров, C++ используемых в стандартной библиотеке. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).

Возвращает итератор, указывающий на первый элемент последовательности (или на место сразу за концом пустой последовательности).

## <a name="syntax"></a>Синтаксис

```cpp
const_iterator begin() const;

iterator begin();
```
