# <a name="ltciso646gt"></a>&lt;ciso646&gt;

Включает заголовок \<iso646.h> стандартной библиотеки C и добавляет связанные имена в пространство имен `std`.

> [!NOTE]
> Использовать этот заголовок необязателен в C++.

## <a name="requirements"></a>Требования

**Заголовок:** \<ciso646 >

**Пространство имен:** std

## <a name="remarks"></a>Примечания

Включение этого заголовка гарантирует, что имена, объявленные с помощью внешней компоновки в заголовке стандартной библиотеки C, объявляются в пространстве имен `std`.
