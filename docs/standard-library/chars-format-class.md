# <a name="charsformat-class"></a>Класс chars_format

Формат с плавающей запятой для простые числовые преобразования.

## <a name="syntax"></a>Синтаксис

```cpp
enum class chars_format {
    scientific = unspecified ,
    fixed = unspecified ,
    hex = unspecified ,
    general = fixed | scientific
};
```
