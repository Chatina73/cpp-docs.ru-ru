# <a name="ltcwchargt"></a>&lt;cwchar&gt;

Включает заголовок стандартной библиотеки C \<uchar.h > и добавляет связанные имена в `std` пространства имен.

## <a name="requirements"></a>Требования

**Заголовок:** \<cuchar >

**Пространство имен:** std

## <a name="member-functions"></a>Функции-члены

|||
|-|-|
|[c16rtomb](../standard-library/cuchar-functions.md#c16rtomb)|Нет описания.|
|[c32rtomb](../standard-library/cuchar-functions.md#c32rtomb)|Нет описания.|
|[mbrtoc16](../standard-library/cuchar-functions.md#mbrtoc16)|Нет описания.|
|[mbrtoc32](../standard-library/cuchar-functions.md#mbrtoc32)|Нет описания.|

## <a name="remarks"></a>Примечания

Включение этого заголовка гарантирует, что имена, объявленные с помощью внешней компоновки в заголовке стандартной библиотеки C, объявляются в пространстве имен `std`.
