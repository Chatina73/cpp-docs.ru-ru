# <a name="container-classrend"></a>Класс контейнера::rend

> [!NOTE]
> Эта статья содержится в документации по Microsoft C++ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C++. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).

Функция-член возвращает обратный итератор, указывающий на первый элемент последовательности (или на место сразу за концом пустой последовательности), обозначая конец обратной последовательности.

## <a name="syntax"></a>Синтаксис

```cpp
const_reverse_iterator rend() const;

reverse_iterator rend();
```
