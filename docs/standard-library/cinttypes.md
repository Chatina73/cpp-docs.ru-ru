# <a name="ltcinttypesgt"></a>&lt;cinttypes&gt;

Включает заголовок \<inttypes.h> стандартной библиотеки C. При включении этого заголовка также включается \<cstdint>.

## <a name="syntax"></a>Синтаксис

```cpp
#include <cinttypes>

```

## <a name="remarks"></a>Примечания

Включение этого заголовка гарантирует, что имена, объявленные путем использования внешней компоновки в заголовке стандартной библиотеки C, объявляются в пространстве имен `std`.
