# <a name="container-classconst_iterator"></a>Класс контейнера::const_iterator

> [!NOTE]
> Эта статья содержится в документации по Microsoft C++ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C++. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).

Описывает объект, который можно использовать в качестве постоянного итератора для управляемой последовательности.

## <a name="syntax"></a>Синтаксис

```cpp
typedef T6 const_iterator;
```

## <a name="remarks"></a>Remarks

Он описан здесь как синоним для неопределенного типа `T6` .
