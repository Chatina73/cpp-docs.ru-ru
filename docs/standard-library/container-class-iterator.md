# <a name="container-classiterator"></a>Класс контейнера::iterator

> [!NOTE]
> Эта статья содержится в документации по Microsoft C++ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C++. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).

Описывает объект, который можно использовать в качестве итератора для управляемой последовательности.

## <a name="syntax"></a>Синтаксис

```cpp
typedef T5 iterator;
```

## <a name="remarks"></a>Remarks

Он описан здесь как синоним для неопределенного типа `T5` . Объект типа `iterator` можно привести к объекту типа [const_iterator](../standard-library/container-class-const-iterator.md).
