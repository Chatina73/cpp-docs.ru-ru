# <a name="try_to_lock_t-structure"></a>Структура try_to_lock_t

Представляет **`struct`** , определяющий объект [try_to_lock](../standard-library/mutex-functions.md#try_to_lock) . Служит для выбора одного из перегруженных конструкторов [unique_lock](../standard-library/unique-lock-class.md).

## <a name="syntax"></a>Синтаксис

```cpp
struct try_to_lock_t;
```

## <a name="requirements"></a>Требования

**Заголовок:**\<mutex>

**Пространство имен:** std
