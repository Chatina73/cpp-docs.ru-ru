# <a name="container-classreverse_iterator"></a>Класс контейнера::reverse_iterator

> [!NOTE]
> Эта статья содержится в документации по Microsoft C++ как нефункциональный пример контейнеров, используемых в стандартной библиотеке C++. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).

Описывает объект, который можно использовать в качестве обратного итератора для управляемой последовательности.

## <a name="syntax"></a>Синтаксис

```cpp
typedef T7 reverse_iterator;
```

## <a name="remarks"></a>Remarks

Он описан здесь как синоним для неопределенного типа `T7` (обычно `reverse_iterator` **\<**[iterator](../standard-library/container-class-iterator.md)**>** ).
