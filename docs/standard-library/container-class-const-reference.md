# <a name="container-classconstreference"></a>Класс контейнера::const_reference

> [!NOTE]
> Этот раздел находится в Microsoft C++ документации в качестве нефункционального примера контейнеров, используемых в C++ стандартной библиотеки. Дополнительные сведения см. в разделе [Контейнеры стандартной библиотеки C++](../standard-library/stl-containers.md).

Описывает объект, который можно использовать в качестве постоянной ссылки на элемент управляемой последовательности.

## <a name="syntax"></a>Синтаксис

```

typedef T3 const_reference;
```

## <a name="remarks"></a>Примечания

Он описан здесь как синоним для неопределенного типа `T3` (обычно `Alloc::const_reference`).
