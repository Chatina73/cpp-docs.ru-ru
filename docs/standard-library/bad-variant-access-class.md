# <a name="bad_variant_access-class"></a>Класс bad_variant_access

Объекты типа bad_variant_access создаются для передачи недопустимых обращений к значению объекта Variant.

## <a name="syntax"></a>Синтаксис

```cpp
class bad_variant_access;
```
