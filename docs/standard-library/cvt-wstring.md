# <a name="ltcvtwstringgt"></a>&lt;cvt/wstring&gt;

Заголовок `<cvt/wstring>` в предыдущих версиях Visual Studio определил класс шаблона класса [wstring_convert](../standard-library/wstring-convert-class.md) в пространстве имен stdext:: CVT. Заголовок сохраняется для обеспечения обратной совместимости. В новом коде следует использовать версию класса, определенную в параметре [\<locale>](../standard-library/locale.md) в пространстве имен `std`.

## <a name="syntax"></a>Синтаксис

```cpp
#include <cvt/wstring>
```
