# <a name="safeint-library"></a>Библиотека SafeInt

**SafeInt** — переносимая библиотека, которую можно использовать с MSVC GCC и Clang, чтобы предотвратить переполнение при работе с целочисленными переменными, что может возникнуть при выполнении математических операций приложением. Последняя версия этой библиотеки находится по адресу [https://github.com/dcleblanc/SafeInt](https://github.com/dcleblanc/SafeInt) .
