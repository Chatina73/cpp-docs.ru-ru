# <a name="support-for-ansi"></a>Поддержка ANSI

Большинство классов и методов MFC поддерживают кодировку ANSI, хотя платформа MFC в целом постепенно развивается к поддержке только набора символов Юникода. Из-за текущих улучшений в Windows Vista и стандартных элементах управления Windows версии 6,1 Поддержка нескольких классов и методов ANSI не рекомендуется.  Дополнительные сведения см. в статьях [устаревшие API ANSI](../mfc/deprecated-ansi-apis.md) и [Поддержка Юникода](../text/support-for-unicode.md).
