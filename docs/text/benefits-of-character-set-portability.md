# <a name="benefits-of-character-set-portability"></a>Преимущества переносимости кодировки

Вы можете воспользоваться преимуществами переносимых функций MFC и среды выполнения C, даже если сейчас вы не планируете интернатионализе приложение:

- Программирование с помощью перепрограммированной базы кода делает ее более гибкой. Позднее его можно легко переместить в Юникод или MBCS.

- Использование Юникода делает приложения для Windows более эффективными. Так как в Windows используются Юникод, строки, не поддерживающие Юникод, передаются в операционную систему и из нее должны быть переведены, что влечет за собой издержки.
