# <a name="incrementing-and-decrementing-pointers"></a>Увеличение и уменьшение значений указателей

Используйте следующие советы.

- Наведите указатель на ведущий байт, а не на конечные байты. Обычно небезопасна наличие указателя на конечный байт. Обычно более безопасно проверять строку вперед, а не в обратную.

- Существуют функции инкремента и уменьшения указателя, доступные для перемещения по целому символу:

    ```cpp
    sz1++;
    ```

   обретает

    ```cpp
    sz1 = _mbsinc( sz1 );
    ```

   `_mbsinc`Функции и `_mbsdec` увеличиваются и уменьшаются в `character` единицах, независимо от размера символов.

- Для декремента требуется указатель на заголовок строки, как показано ниже:

    ```cpp
    sz2--;
    ```

   обретает

    ```cpp
    sz2 = _mbsdec( sz2Head, sz2 );
    ```

   В качестве альтернативы можно, чтобы указатель HEAD был допустимым символом в строке, таким образом:

    ```cpp
    sz2Head < sz2
    ```

   Необходимо иметь указатель на известный допустимый старший байт.

- Может потребоваться сохранить указатель на предыдущий символ для ускорения вызовов `_mbsdec` .
