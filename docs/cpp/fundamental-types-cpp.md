# <a name="fundamental-types--c"></a>Фундаментальные типы (C++)

Основные типы в С++ делятся на три категории: целочисленные, с плавающей запятой и void. Целочисленные типы позволяют обрабатывать целые числа. Типы с плавающей запятой позволяют задавать значения, которые могут иметь дробные части.

Типом [void](../cpp/void-cpp.md) описывается пустой набор значений. Переменная типа не **void** можно указать, он используется в основном для объявления функций, которые не возвращают никаких значений или универсальных указателей на нетипизированные или произвольно типизированные данные. Любое выражение можно явно преобразовать или приведенное к типу **void**. Однако такие выражения можно использовать только в следующих операторах и операндах:

- в операторе выражения (дополнительные сведения см. в разделе [Выражения](../cpp/expressions-cpp.md));

- в левом операнде оператора запятой (дополнительные сведения см. в разделе [Оператор "запятая"](../cpp/comma-operator.md) );

- во втором и третьем операндах условного оператора (`? :`). (дополнительные сведения см. в разделе [Выражения с условным оператором](../cpp/conditional-operator-q.md) );

В следующей таблице объясняются ограничения на размеры типов. Эти ограничения не зависят от реализации Microsoft.

### <a name="fundamental-types-of-the-c-language"></a>Основные типы языка C++

|Категория|Тип|Описание|
|--------------|----------|--------------|
|Целые числа|**char**|Тип **char** это целочисленный тип, обычно содержащий члены кодировки выполнения — по умолчанию, это ASCII в Microsoft C++.<br /><br /> Компилятор C++ обрабатывает переменные типа **char**, **автоматический char**, и **unsigned char** как переменные разных типов. Переменные типа **char** повышаются до **int** как если бы они имели тип **автоматический char** по умолчанию, если не используется параметр компиляции/j. В этом случае они рассматриваются как тип **unsigned char** и повышаются до **int** без расширения знака.|
||**bool**|Тип **bool** это целочисленный тип, который может иметь одно из двух значений **true** или **false**. Его размер не определен.|
||**short**|Тип **короткое целочисленное** (или просто **короткие**) — это целочисленный тип, размер которого больше или равен размеру типа **char**и меньше или равен размеру типа **int**.<br /><br /> Объекты типа **короткие** могут быть объявлены как **автоматический говоря** или **unsigned short**. **Говоря автоматический** является синонимом **короткие**.|
||**int**|Тип **int** — это целочисленный тип, размер которого больше или равен размеру типа **короткое целочисленное**и меньше или равен размеру типа **long**.<br /><br /> Объекты типа **int** могут быть объявлены как **целочисленное число со знаком** или **unsigned int**. **Целочисленное число со знаком** является синонимом **int**.|
||**__int8**, **__int16**, **__int32**, **__int64**|Целое число с указанием размера `__int n`, где `n` — размер в битах целочисленной переменной. **__int8**, **__int16**, **__int32** и **__int64** ключевые слова для систем Microsoft. Не все типы доступны для всех архитектур. (**__int128** не поддерживается.)|
||**long**|Тип **long** (или **long int**) — это целочисленный тип, размер которого больше или равен размеру типа **int**.<br /><br /> Объекты типа **long** могут быть объявлены как **автоматический долго** или **unsigned long**. **Автоматический долго** является синонимом **long**.|
||**long long**|Больше, чем unsigned **long**.<br /><br /> Объекты типа **long long** могут быть объявлены как **со знаком длинное длинное** или **long long без знака**. **со знаком длинное длинное** является синонимом **long long**.|
||**wchar_t**, **__wchar_t**|Переменная типа **wchar_t** обозначает расширенный символьный или Многобайтовый символьный тип. По умолчанию **wchar_t** является машинным типом, но можно использовать [/Zc:wchar_t-](../build/reference/zc-wchar-t-wchar-t-is-native-type.md) вносить **wchar_t** typedef для **unsigned short**. **__Wchar_t** тип является синонимом характерные для Майкрософт для машинного кода **wchar_t** типа.<br /><br /> Чтобы задать расширенный символьный тип, перед символьным или строковым литералом следует использовать префикс L.|
|С плавающей запятой|**float**|Тип **float** — это наименьшее число с плавающей запятой.|
||**double**|Тип **двойные** — это тип с плавающей запятой, которое больше или равен размеру типа **float**, но меньше или равен размеру типа **long double**.<br /><br /> Системам Майкрософт: Представление **long double** и **двойные** идентичны. Тем не менее **long double** и **двойные** — это отдельные типы.|
||**long double**|Тип **long double** — с плавающей запятой тип, размер которых превышает или равен размеру типа **двойные**.|

**Блок, относящийся только к системам Microsoft**
