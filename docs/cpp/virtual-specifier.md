# <a name="virtual-specifier"></a>Спецификатор virtual

Ключевое слово [Virtual](../cpp/virtual-cpp.md) может применяться только к нестатическим функциям члена класса. Это означает, что привязка вызовов функции откладывается до времени выполнения. Дополнительные сведения см. в разделе [виртуальные функции](../cpp/virtual-functions.md).
