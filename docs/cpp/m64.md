# <a name="__m64"></a>__m64

**Блок, относящийся только к системам Microsoft**

Тип данных **__m64** предназначен для использования с технологией MMX и 3DNow! встроенные функции и определяются в \<ксмминтрин. h >.

```cpp
// data_types__m64.cpp
#include <xmmintrin.h>
int main()
{
   __m64 x;
}
```

## <a name="remarks"></a>Remarks

Не следует напрямую обращаться к полям **__m64** . Однако можно просматривать эти типы в отладчике. Переменная типа **__m64** сопоставляется с регистрами mm [0-7].

Переменные типа **_m64** автоматически согласовываются с границами в 8 байт.

Тип данных **__m64** не поддерживается на процессорах x64. Приложения, использующие тип __m64 в составе встроенных инструкций MMX, необходимо переписать с использованием эквивалентных встроенных инструкций SSE и SSE2.
