# <a name="naked-c"></a>naked (C++)

**Блок, относящийся только к системам Microsoft**

Для функций, объявленных с атрибутом **naked** , компилятор создает код без кода пролога и эпилога. Эту возможность можно использовать, чтобы создавать свой собственный код на языке ассемблера и вставлять его в качестве пролога и эпилога. Функции с атрибутом naked особенно полезны для написания драйверов виртуальных устройств.  Обратите внимание, что атрибут **naked** допустим только для x86 и ARM и недоступен в x64.

## <a name="syntax"></a>Синтаксис

```
__declspec(naked) declarator
```

## <a name="remarks"></a>Remarks

Поскольку атрибут **naked** относится только к определению функции и не является модификатором типа, функции с атрибутом naked должны использовать синтаксис расширенных атрибутов и ключевое слово [__declspec](../cpp/declspec.md) .

Компилятор не может создать встроенную функцию для функции, помеченной атрибутом naked, даже если эта функция также помечена ключевым словом [__forceinline](inline-functions-cpp.md) .

Компилятор выдает ошибку, если атрибут **naked** применяется к любому другому, кроме определения метода, не являющегося членом.

## <a name="examples"></a>Примеры

Этот код определяет функцию с атрибутом **naked** :

```
__declspec( naked ) int func( formal_parameters ) {}
```

Другой пример.

```
#define Naked __declspec( naked )
Naked int func( formal_parameters ) {}
```

Атрибут **naked** влияет только на характер создания кода компилятора для последовательностей пролога и эпилога функции. Код, который создается для вызова таких функций, не зависит от этого атрибута. Таким словами, атрибут **naked** не считается частью типа функции, а указатели функций не могут иметь атрибут **naked** . Кроме того, атрибут **naked** не может применяться к определению данных. Например, следующий код вызывает ошибку:

```
__declspec( naked ) int i;
// Error--naked attribute not permitted on data declarations.
```

Атрибут **naked** относится только к определению функции и не может быть указан в прототипе функции. Например, следующее объявление создает ошибку компилятора:

```
__declspec( naked ) int func();  // Error--naked attribute not permitted on function declarations
```
