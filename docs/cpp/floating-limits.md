# <a name="floating-limits"></a>Пределы констант с плавающей запятой

**Microsoft Специфический**

В следующей таблице представлены ограничения на значения констант с плавающей запятой. Эти ограничения также определены в стандартном файле \<заголовка float.h>.

## <a name="limits-on-floating-point-constants"></a>Ограничения на константы с плавающей запятой

|Константа|Значение|Значение|
|--------------|-------------|-----------|
|`FLT_DIG`<br/>`DBL_DIG`<br/>`LDBL_DIG`|Количество цифр q, при котором число с плавающей запятой с q десятичными цифрами можно округлить в представление с плавающей запятой и обратно без потери точности.|6<br/>15<br/>15|
|`FLT_EPSILON`<br/>`DBL_EPSILON`<br/>`LDBL_EPSILON`|Наименьшее положительное число x, при котором x + 1,0 не равно 1,0.|1,192092896e-07F<br/>2,2204460492503131e-016<br/>2,2204460492503131e-016|
|`FLT_GUARD`||0|
|`FLT_MANT_DIG`<br/>`DBL_MANT_DIG`<br/>`LDBL_MANT_DIG`|Количество цифр в радиксе, указанном `FLT_RADIX` в плавающей точке significand. Радикс 2; следовательно, эти значения указывают биты.|24<br/>53<br/>53|
|`FLT_MAX`<br/>`DBL_MAX`<br/>`LDBL_MAX`|Максимальное представленное плавающее значение.|3,402823466e+38F<br/>1.7976931348623158e+308<br/>1.7976931348623158e+308|
|`FLT_MAX_10_EXP`<br/>`DBL_MAX_10_EXP`<br/>`LDBL_MAX_10_EXP`|Максимальное целых числа так, что 10 поднятыдок до этого числа является представляющим плавающей точки номер.|38<br/>308<br/>308|
|`FLT_MAX_EXP`<br/>`DBL_MAX_EXP`<br/>`LDBL_MAX_EXP`|Максимальный целый `FLT_RADIX` ряд, такой, что подняли до этого числа является представляющим плавающей точки номер.|128<br/>1024<br/>1024|
|`FLT_MIN`<br/>`DBL_MIN`<br/>`LDBL_MIN`|Минимальное положительное значение.|1,175494351e-38F<br/>2.2250738585072014e-308<br/>2.2250738585072014e-308|
|`FLT_MIN_10_EXP`<br/>`DBL_MIN_10_EXP`<br/>`LDBL_MIN_10_EXP`|Минимальный отрицательный целый ряд таков, что 10 поднятых до этого числа является представляющим плавающей точки номер.|-37<br/>-307<br/>-307|
|`FLT_MIN_EXP`<br/>`DBL_MIN_EXP`<br/>`LDBL_MIN_EXP`|Минимальный отрицательный целый ряд, такой, что `FLT_RADIX` поднял до этого числа является представляющим плавающей точки числа.|-125<br/>-1021<br/>-1021|
|`FLT_NORMALIZE`||0|
|`FLT_RADIX`<br/>`_DBL_RADIX`<br/>`_LDBL_RADIX`|Основание экспоненциальной формы представления.|2<br/>2<br/>2|
|`FLT_ROUNDS`<br/>`_DBL_ROUNDS`<br/>`_LDBL_ROUNDS`|Режим округления для добавления плавающей точки.|1 (приблизительно)<br/>1 (приблизительно)<br/>1 (приблизительно)|

> [!NOTE]
> В будущих версиях продукта информация из приведенной выше таблицы может отличаться.
