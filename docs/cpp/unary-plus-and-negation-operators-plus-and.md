# <a name="unary-plus-and-negation-operators--and--"></a>Унарные операторы «плюс» и «отрицание»: + и -

## <a name="syntax"></a>Синтаксис

```
+ cast-expression
- cast-expression
```

## <a name="-operator"></a>+ - оператор

Результатом оператора унарного сложения ( **+** ) является значение его операнда. Операнд оператора унарного оператора сложения должен иметь арифметический тип.

Над целочисленными операндами выполняется восходящее приведение целого типа. Результирующим типом является тип, до которого повышается уровень операнда. Поэтому выражение `+ch` , где `ch` имеет тип **`char`** , приводит к типу. **`int`** значение не изменяется. Дополнительные сведения о том, как выполняется продвижение, см. в разделе [стандартные преобразования](standard-conversions.md) .

## <a name="--operator"></a>- - оператор

Унарный оператор отрицания ( **-** ) создает отрицательное значение операнда. Операнд оператора унарного отрицания должен быть арифметическим типом.

Над целочисленными операндами выполняется восходящее приведение целого типа, и результирующим типом является тип, до которого повышается уровень операнда. Дополнительные сведения о том, как выполняется продвижение, см. в разделе [стандартные преобразования](standard-conversions.md) .

**Блок, относящийся только к системам Microsoft**

Унарное отрицание величин без знака выполняется путем вычитания значения операнда из числа 2^n, где n — количество битов в объекте заданного типа без знака.
