# <a name="special-member-functions"></a>Специальные функции-члены

*Специальных функций-членов* являются класс (или структура), что в некоторых случаях компилятор автоматически создает для вас функций-членов. Эти функции являются [конструктор по умолчанию](constructors-cpp.md#default_constructors), [деструктор](destructors-cpp.md), [конструктор копии и оператор присваивания копии](copy-constructors-and-copy-assignment-operators-cpp.md)и [конструктор перемещения и оператор присваивания перемещения](move-constructors-and-move-assignment-operators-cpp.md). Если ваш класс определяет один или несколько специальных функций-членов, затем компилятор может неявно объявления и определения функций, которые используются. Реализации компилятором вызываются *по умолчанию* специальных функций-членов. Компилятор не создает функции, если они не нужны.

По умолчанию специальную функцию-член можно явно объявить с помощью **= значение по умолчанию** ключевое слово. В этом случае компилятор для определения функции только в том случае, если требуется, так же как, если функция не была объявлена вообще.

В некоторых случаях компилятор может создать *удалены* специальные функции-члены, которые не определены и поэтому не вызываемую. Это может произойти в случаях, где вызова определенной специальная функция-член класса не имеет смысла, учитывая других свойств класса. Чтобы явно запретить автоматическое создание специальная функция-член, его можно объявить как удаленные с помощью **= удаление** ключевое слово.

Компилятор создает *конструктор по умолчанию*, конструктор, который не принимает аргументы, только в том случае, если вы не объявили любого другого конструктора. Если вы объявили только конструктор, принимающий параметры, код, который пытается вызвать конструктор по умолчанию компилятор для создания сообщения об ошибке. По умолчанию, созданный компилятором конструктор выполняет простой member-wise [инициализация по умолчанию](initializers.md#default_initialization) объекта. Инициализация по умолчанию оставляет все переменные-члены в неопределенном состоянии.

Деструктор по умолчанию выполняет Поэлементное уничтожения объекта. Он является виртуальным, только в том случае, если деструктор базового класса является виртуальным.

По умолчанию и конструкции перемещения операций копирования и присваивания выполнять одним битовым шаблоном копирует или перемещает членов нестатических данных. Перемещение операций создаются только в том случае, когда объявляются нет деструктора или операции перемещения или копирования. Конструктор копии по умолчанию создается только в том случае, когда конструктора копии не объявлен. Оно удаляется неявно при объявлении операции перемещения. Оператор присваивания копии по умолчанию создается только в том случае, когда нет оператор присваивания копии был объявлен явным образом. Оно удаляется неявно при объявлении операции перемещения.
