# <a name="remainders"></a>Остатки от деления

**ANSI 3.3.5** Знак остатка при целочисленном делении

Знак остатка совпадает со знаком делимого. Например, примененная к объекту директива

```
50 / -6 == -8
50 % -6 ==  2
-50 /  6 == -8
-50 %  6 == -2
```
