# <a name="walkthrough-creating-a-standard-c-program-c"></a>Пошаговое руководство. Создание стандартной программы C++ (C++)

Вы можете использовать Visual Studio для создания стандартных программ СЗ. Следуя шагам в этом пошаговом шаге, вы можете создать проект, добавить новый файл в проект, изменить файл, чтобы добавить код C, а затем компилировать и запускать программу с помощью Visual Studio.

Вы можете ввести свою собственную программу C или использовать одну из примерок программ. Образец программы в этом пошаговочном пошаговочном приложении является консольным приложением. Это приложение `set` использует контейнер в стандартной библиотеке СЗ.

> [!NOTE]
> Если требуется соблюдение определенной версии языкового стандарта СЗЗ (т.е. СЗ14 или `/std:c++14` `/std:c++17` СЗ17), используйте опцию или компилятор. (Visual Studio 2017 и более поздний срок.)

## <a name="prerequisites"></a>Предварительные требования

Для выполнения этого пошагового руководства читатель должен владеть основами языка C++.

### <a name="to-create-a-project-and-add-a-source-file"></a>Создание проекта и добавление исходного файла

Приведенные ниже инструкции немного отличаются в зависимости от используемой версии Visual Studio. Чтобы просмотреть документацию для предпочтительной версии Visual Studio, используйте элемент управления селектора **версии.** Он находится в верхней части таблицы содержимого на этой странице.

::: moniker range="vs-2019"

### <a name="to-create-a-c-project-in-visual-studio-2019"></a>Создание проекта СЗЗ в Visual Studio 2019

1. В главном меню выберите **Файл ** > **Создать ** > **Проект**, чтобы открыть диалоговое окно **Создание проекта**.

1. В верхней части диалогового окна для параметра **Язык** выберите значение **C++**, для параметра **Платформа** — значение **Windows**, а для параметра **Тип проекта** — значение **Консоль**.

1. В отфильтрованном списке типов проектов щелкните **Консольное приложение**, а затем нажмите кнопку **Далее**. На следующей странице введите имя проекта и при желании укажите местоположение проекта.

1. Нажмите кнопку **Создать**, чтобы создать проект.

::: moniker-end

::: moniker range="vs-2017"

### <a name="to-create-a-c-project-in-visual-studio-2017"></a>Для создания проекта СЗЗ в Visual Studio 2017

1. Создайте проект, указав на **новое** меню **файла,** а затем нажав **на проект**.

1. В типе типов визуальных типов **C',** нажмите **Windows Desktop**, а затем нажмите Windows **Console Application**.

1. Введите имя для проекта. По умолчанию решение, содержащее проект, имеет то же имя, что и проект, но можно ввести другое имя. Вы также можете ввести другое место для проекта.

1. Нажмите кнопку **ОК**, чтобы создать проект.

::: moniker-end

::: moniker range="vs-2015"

### <a name="to-create-a-c-project-in-visual-studio-2015"></a>Для создания проекта СЗЗ в Visual Studio 2015

1. Создайте проект, указав на **новое** меню **файла,** а затем нажав **на проект**.

1. В типе типов визуальных типов **C',** нажмите **Windows Desktop**, а затем нажмите Windows **Console Application**.

1. В диалоговом окне **нового проекта** расширьте **установленные** > **шаблоны** > **Visual C ,** а затем выберите **Win32**. В центральной области выберите **Консольное приложение Win32**.

1. Введите имя для проекта. По умолчанию решение, содержащее проект, имеет то же имя, что и проект, но можно ввести другое имя. Вы также можете ввести другое место для проекта.

1. Нажмите кнопку **ОК**, чтобы создать проект.

1. Завершите **Win32 Приложение Мастер**.

1. Нажмите **далее,** затем убедитесь, что **консоль приложение** выбрано и отменить **предварительно собранные заголовки** поле.

1. Нажмите кнопку **Готово**.

::: moniker-end

## <a name="add-a-new-source-file"></a>Добавление нового исходного файла

1. Если **Solution Explorer** не отображается в меню **View,** нажмите **Solution Explorer.**

1. Добавление нового исходного файла в проект следует следующим образом.

   1. В **Solution Explorer**, правой кнопкой кнопку **Исходфайлы** папку, указать, чтобы **добавить,** а затем нажмите **Новый пункт**.

   1. В узло **кода** щелкните **файл C '(.cpp)**, введите имя для файла, а затем нажмите **Добавить**.

   Файл .cpp отображается в папке **Исходные файлы** в **Solution Explorer,** и файл открывается в редакторе Visual Studio.

1. В файле в редакторе введите действительную программу СЗЗ, которая использует Стандартную библиотеку СЗ, или скопируйте одну из примеров программ и вставьте ее в файл.

1. Сохраните файл.

1. В меню **Сборка** выберите **Построить решение**.

   Окно **вывода** отображает информацию о ходе компиляции, например, расположение журнала сборки и сообщение, указававщее состояние сборки.

1. В меню **Отладка** выберите команду **Запуск без отладки**.

   Если вы использовали образец программы, отображается окно команды и показывает, находятся ли определенные ряды в наборе.
