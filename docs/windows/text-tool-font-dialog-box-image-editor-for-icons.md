# <a name="text-tool-font-dialog-box-c-image-editor-for-icons"></a>Текст "Шрифт диалогового окна" (C++) (редактор изображений для значков)

Указывает параметры шрифта, при использовании [диалоговое окно "текст"](../windows/text-tool-dialog-box-image-editor-for-icons.md) из [редактор изображений](../windows/image-editor-for-icons.md).

Используйте это диалоговое окно для:

- [Изменение шрифта текста на изображении](../windows/changing-the-font-of-text-on-an-image-image-editor-for-icons.md)

Чтобы открыть это диалоговое окно, щелкните **шрифта** кнопку **текст** диалоговое окно.

- **Шрифт**

   Список доступных шрифтов.

- **Стиль шрифта**

   Перечисляются доступные стили для заданного шрифта.

- **Size**

   Список доступных размеров для выбранного шрифта.

- **Пример**

   Показан пример того, как будет выглядеть текст с заданными параметрами шрифта.

- **Сценарий**

   Список доступных языков сценариев, для заданного шрифта. При выборе другого языка, набор символов для этого языка станет доступен при создании многоязычных документов.

## <a name="requirements"></a>Требования
