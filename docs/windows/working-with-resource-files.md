# <a name="working-with-resource-files"></a>Работа с файлами ресурсов

> [!WARNING]
> Этот раздел относится к классическим приложениям Windows, написанным на C++. Сведения о ресурсах в приложениях универсальной платформы Windows, написанного на языке C++ см. в разделе [определение ресурсов приложения](/windows/uwp/app-resources/).
>
> Сведения о добавлении ресурсов в C + +/ CLI (управляемые) проектов, см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*.

Ресурсы могут состоять из широкий спектр элементов, включая элементы интерфейса, которые предоставляют сведения пользователю (например точечного рисунка, значка или курсора), настраиваемые ресурсы, которые содержат данные приложению, ресурсов версии, которые используются API-интерфейсами, настройки а также ресурсов меню и диалоговому окну.

Вы можете добавить новые ресурсы в проект и изменить их с помощью соответствующего редактора ресурсов. Большинство мастеров Visual C++ автоматически создают RC-файл для проекта.

Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).
