# <a name="designing-a-dialog-box-c"></a>Проектирование диалоговое окно (C++)

Расположение и размер в диалоговом окне C++ и расположение и размер элементов управления внутри него, измеряются в диалоговых единицах. Значения для отдельных элементов управления и диалоговом окне отображаются в правом нижнем углу строки при их выборе состояния Visual Studio.

При разработке диалоговое окно, можно смоделировать и протестировать его поведение во время выполнения, не компилируя программу. В этом режиме можно выполнять следующие действия:

- вводить текст, выбирать пункты в полях со списками, включать и отключать параметры, выбирать команды;

- тестировать последовательность табуляции;

- проверять группировку элементов управления, например переключателей и флажков;

- тестировать сочетания клавиш, используемые для доступа к элементам управления в диалоговом окне.

   > [!NOTE]
   > Подключения к коду диалогового окна, выполненные с помощью мастеров, не включаются в моделирование.

При тестировании диалогового окна его положение обычно определяется относительно главного окна программы. Если вы задали диалогового окна **абсолютное выравнивание** свойства **True**, отображает диалоговое окно с позиции, относительно верхнего левого угла экрана.

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index).

## <a name="to-specify-the-location-and-size-of-a-dialog-box"></a>Чтобы указать расположение и размер в диалоговом окне

Существует три свойства, которые можно установить в [окно "Свойства"](/visualstudio/ide/reference/properties-window) для указания, где диалоговое окно будет отображаться на экране. **Center** свойство является логическое значение; Если значение равно **True**, диалоговое окно всегда будет отображаться в центре экрана. Если ему присвоено **False**, затем можно задать **XPos** и **YPos** свойства, чтобы явно определить, когда на экране появится диалоговое окно. Позиция свойства, значения смещения в верхнем левом углу области просмотра, который определен как `{X=0, Y=0}`. Позиция также основана на **абсолютное выравнивание** свойство: Если **True**, координаты указываются относительно экрана; Если **False**, координаты указываются относительно диалогового окна окно владельца.

## <a name="to-test-a-dialog-box"></a>Проверка диалогового окна

1. Когда **диалоговое окно** редактора является активным окном, в строке меню, выберите **формат** > **проверить диалоговое окно**.

1. Чтобы завершить тестирование, нажмите клавишу **Esc**, или просто нажмите **закрыть** кнопку в диалоговом окне вы тестируете.
