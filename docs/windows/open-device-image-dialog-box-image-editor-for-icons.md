# <a name="open-ltdevicegt-image-dialog-box-c-image-editor-for-icons"></a>Откройте &lt;устройства&gt; изображения диалоговое окно (C++) (редактор изображений для значков)

Это диалоговое окно используется для открытия изображений устройств в проектах C++. В ней перечислены существующие образы устройство в текущий ресурс (изображения, которые являются частью текущего ресурса).

### <a name="current-images"></a>Текущих образов

Список образов, включенных в ресурсе. Выберите тип образа, который вы хотите открыть.

## <a name="requirements"></a>Требования

Нет
