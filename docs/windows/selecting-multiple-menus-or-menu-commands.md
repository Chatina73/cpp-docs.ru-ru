# <a name="selecting-multiple-menus-or-menu-commands"></a>Выбор нескольких меню или команд меню

Для выполнения массовых операций, таких как удаление или изменение свойств, можно выбрать несколько меню или команд меню.

### <a name="to-select-multiple-menu-commands"></a>Выбор нескольких команд меню

1. Удерживая нажатой **Ctrl** ключ, нажмите кнопку меню или подменю команды.

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*.

## <a name="requirements"></a>Требования

Win32
