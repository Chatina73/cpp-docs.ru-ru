# <a name="creating-moving-and-editing-toolbar-buttons-c"></a>Создание, перемещение и редактирование кнопок панели инструментов (C++)

Можно легко создать, переместить, скопировать и изменить кнопки панели инструментов.

По умолчанию новая или пустая кнопка отображается в правом конце панели инструментов. Можно переместить эту кнопку, прежде чем изменять его. При создании новой кнопки, еще одна пустая кнопка справа от нее. При сохранении панели инструментов пустая кнопка не сохраняется.

Существуют следующие свойства кнопки панели инструментов.

|Свойство.|Описание|
|--------------|-----------------|
|**ID**|Определяет идентификатор для кнопки. Выберите в раскрывающемся списке предоставляет Общие **идентификатор** имена.|
|**Width**|Задает ширину кнопки. рекомендуется 16 пикселей.|
|**Height**|Задает высоту кнопки. Высота кнопки изменяет высоту всех кнопок на панели инструментов. Рекомендуется 15 пикселей.|
|**Запрашивать**|Определяет сообщение, отображаемое в строке состояния. При добавлении \n и именем добавляются всплывающей подсказки для кнопки панели инструментов. Дополнительные сведения см. в разделе [Создание всплывающей подсказки](../windows/creating-a-tool-tip-for-a-toolbar-button.md).|

**Ширина** и **высота** применяются ко всем кнопкам. Битовая карта, используется для создания панели инструментов имеет максимальную ширину 2048. Поэтому если ширина кнопки значение 512, может быть только четыре кнопки, и если ширина присвоено 513, может быть только три кнопки.

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).

См. следующие действия:

## <a name="to-create-a-new-toolbar-button"></a>Чтобы создать новую кнопку панели инструментов

1. В [представление ресурсов](../windows/resource-view-window.md) разверните папку ресурсов (например, *Project1.rc*).

   > [!NOTE]
   > Если в проекте еще нет RC-файла, см. раздел [Создание нового файла описания ресурсов](../windows/how-to-create-a-resource-script-file.md).

1. Разверните **инструментов** папку и выберите панель инструментов для редактирования.

1. Назначение идентификатора пустую кнопку в правом конце панели инструментов. Это можно сделать, изменив **идентификатор** свойство в [окно "Свойства"](/visualstudio/ide/reference/properties-window). Например можно предоставить один и тот же идентификатор как пункт меню для кнопки панели инструментов. В этом случае используйте поле с раскрывающимся списком для выбора **идентификатор** пункта меню.

   - или -

   Выберите пустую кнопку в правом конце панели инструментов (в **инструментов представления** области) и начала рисования. Идентификатор команды кнопки по умолчанию назначается (ID_BUTTON\<n >).

Кроме того, можно скопировать и вставить изображение на панель инструментов в качестве новой кнопки.

## <a name="to-add-an-image-to-a-toolbar-as-a-button"></a>Добавление изображения на панель инструментов, как кнопка

1. В [представление ресурсов](../windows/resource-view-window.md), откройте панель инструментов, дважды щелкнув его.

1. Затем откройте образ, который вы хотите добавить на панель инструментов.

   > [!NOTE]
   > При открытии изображения в Visual Studio, он открывается в **изображение** редактора. Можно также открыть изображение в других графических программ.

1. Из **изменить** меню, выберите **копирования**.

1. Переключитесь на панель инструментов, выбрав его вкладки в верхней части окна исходного кода.

1. Из **изменить** меню, выберите **вставить**.

   Изображение будет отображаться на панели инструментов, как новая кнопка.

## <a name="to-move-a-toolbar-button"></a>Перемещение кнопки на панели инструментов

В **инструментов представления** панели, перетащите кнопку, вы хотите переместить в новое место на панели инструментов.

## <a name="to-copy-buttons-from-a-toolbar"></a>Копирование кнопок из панели инструментов

1. Удерживая нажатой **Ctrl** ключ.

1. В **инструментов представления** панели, перетащите кнопку либо новое место на панели инструментов или в расположение на другой панели инструментов.

## <a name="to-delete-a-toolbar-button"></a>Чтобы удалить кнопку панели инструментов

Выберите кнопку панели инструментов и перетащите его с панели инструментов.

## <a name="to-insert-or-remove-space-between-buttons-on-a-toolbar"></a>Чтобы вставить или удалить промежутка между кнопками на панели инструментов

Как правило чтобы вставить промежутка между кнопками, перетащите их друг от друга на панели инструментов. Чтобы удалить пространство, перетащите их к друг с другом.

### <a name="to-insert-a-space-before-a-button-that-isnt-followed-by-a-space"></a>Чтобы вставить пробел перед кнопкой, не разделенных пробелами

Перетащите кнопку вправо или вниз, пока он перекрывает "Далее" о посередине.

### <a name="to-insert-a-space-before-a-button-that-is-followed-by-a-space-and-to-keep-the-trailing-space"></a>Вставлять пробел перед кнопкой, за которой следуют пробел и сохранить этот пробел

Перетащите кнопку до правого или нижнего края коснулся "Далее", или даже перекрыл ее.

### <a name="to-insert-a-space-before-a-button-that-is-followed-by-a-space-and-close-up-that-following-space"></a>Вставлять пробел перед кнопкой, за которой следуют пробел и закрыть следующие пространство

Перетащите кнопку вправо или вниз, пока он перекрывает "Далее" о посередине.

### <a name="to-remove-a-space-between-buttons-on-a-toolbar"></a>Удаление промежутка между кнопками на панели инструментов

Перетащите кнопку на одной стороне пространства к кнопке на другой стороне пространства, чтобы он перекрывает приблизительно половина "Далее".

   Если нет места перетаскиваемой кнопки перетаскивания из и перетащить ее более чем на равном расстоянии за кнопку рядом, **инструментов** редактора также вставляет символ пробела на противоположной стороне, кнопки, которую вы перетаскивание.

## <a name="to-change-the-properties-of-a-toolbar-button"></a>Изменение свойств кнопки панели инструментов

1. В проекте C++ нажмите кнопку панели инструментов.

1. Введите новый идентификатор в **идентификатор** свойство в [окно "Свойства"](/visualstudio/ide/reference/properties-window), или используйте стрелку раскрывающегося списка, чтобы выбрать новую **идентификатор**.
