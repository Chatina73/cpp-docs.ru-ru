# <a name="creating-moving-and-editing-toolbar-buttons"></a>Создание, перемещение и редактирование кнопок в панели инструментов

Можно легко создать, переместить, скопировать и изменить кнопки панели инструментов.

По умолчанию новая или пустая кнопка отображается в правом конце панели инструментов. Можно переместить эту кнопку, прежде чем изменять его. При создании новой кнопки, еще одна пустая кнопка справа от нее. При сохранении панели инструментов пустая кнопка не сохраняется.

- [Перемещение кнопки на панели инструментов](moving-a-toolbar-button.md)

- [Копирование кнопок из панели инструментов](../windows/copying-buttons-from-a-toolbar.md)

- [Удаление кнопки панели инструментов](deleting-a-toolbar-button.md)

- [Вставка промежутка между кнопками на панели инструментов](../windows/inserting-a-space-between-buttons-on-a-toolbar.md)

- [Крупным промежутка между кнопками на панели инструментов](../windows/removing-space-between-buttons-on-a-toolbar.md)

- [Измените страницу свойств на панели инструментов](../windows/changing-the-properties-of-a-toolbar-button.md)

Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps).
