# <a name="dialog-editor-c"></a>Редактор диалогов (КЗ)

**Редактор Dialog** позволяет создавать или редотировать ресурсы диалогового окна.

- Чтобы открыть редактор, дважды нажмите на файл .rc диалога в окне **просмотра ресурсов** или перейдите в меню **View** > **Other Windows** > **Resource View.**

Одним из первых шагов в создании нового шаблона диалогового окна или диалогового окна является добавление элементов управления. В **редакторе Dialog**вы можете организовать элементы управления в соответствии с определенным размером, формой или выравниванием, или переместить их для работы в диалоговом поле. Кроме того, любой элемент управления можно легко удалить.

Диалоговое окно можно сохранить как шаблон и использовать повторно. Вы можете легко переключаться между проектированием диалогового окна и редактированием кода, на котором оно основано,

Также можно отредактировать свойства одного или нескольких элементов управления в **редакторе Dialog.** Вы можете изменить порядок вкладок, то есть порядок, в котором элементы управления получить фокус при нажатии ключа **Вкладк,** или вы можете определить ключ доступа или комбинацию ключа, которая позволяет пользователям выбирать элемент управления с помощью клавиатуры.

**Редактор Dialog** также позволяет использовать пользовательские элементы управления, включая элементы управления ActiveX. Вы также можете отодвить [представление формы,](../mfc/reference/cformview-class.md) [запись представлений](../data/record-views-mfc-data-access.md)или [диалоговые бары.](../mfc/dialog-bars.md)

Начиная с Visual Studio 2015, вы можете использовать **редактор Dialog** для определения динамических макетов, в которых указывается, как элементы управления перемещаются и изменяются при перестроении диалога. Для получения дополнительной информации см. [Dynamic Layout](../mfc/dynamic-layout.md).

Для получения дополнительной информации о ресурсах, [Dialog Box Controls](../windows/controls-in-dialog-boxes.md) [см.](../windows/creating-a-new-dialog-box.md)

> [!TIP]
> При использовании **редактора Dialog**во многих случаях можно выбрать с помощью правой кнопки мыши для отображения меню ярлыка часто используемых команд.

## <a name="switch-between-dialog-box-controls-and-code"></a>Переключение между управлениями и кодом Dialog Box

В приложениях MFC можно дважды нажать на элементы управления диалоговым полем, чтобы перейти к коду обработчика или быстро создать функции обработчика заглушки.

С выбранным элементом управления выберите кнопку **ControlEvents** или кнопку **«Сообщения»** в [окне Свойств,](/visualstudio/ide/reference/properties-window) чтобы просмотреть полный список сообщений ИWindows и событий, доступных для выбранного элемента. Выберите из списка для создания или отсвачения функций обработчика.

- Чтобы перейти к коду из **редактора Dialog,** дважды щелкните элемент управления в диалоговом поле, чтобы перейти к декларации для самой недавно реализованной функции обработки сообщений.

   Для классов диалога на основе ATL вы всегда переходите к определению конструктора.

- Для просмотра событий для управления с выбранным элементом управления выберите кнопку **ControlEvents** в окне **Свойств.**

   Когда один элемент управления фокусируется в диалоговом поле, вы можете нажать правой кнопкой мыши и выбрать **Обработчик событий.** Это позволяет указать класс, в который добавляется обработчик. Для получения дополнительной информации [см.](../ide/adding-an-event-handler-visual-cpp.md)

   > [!NOTE]
   > Выбор кнопки **ControlEvents,** когда диалоговая коробка имеет фокус, предоставляет список всех элементов управления в диалоговом поле, который затем можно расширить, чтобы отсеивать события для отдельных элементов управления.

- Для просмотра сообщений для диалогового окна с выбранным полем диалога выберите кнопку **«Сообщения»** в окне **Свойств.**

## <a name="accelerator-keys"></a>Сочетания клавиш

Ниже приведены клавиши ускорителя по умолчанию для команд **редактора Dialog.**  

|Get-Help|Ключи|Описание|
|-------------|----------|-----------------|
|Format.AlignBottoms|**Ctrl** + **Сдвиг** + **вниз стрелка**|Выравнивает нижние края выбранных элементов управления с доминирующим элементом управления.|
|Format.AlignCenters|**Сдвиг** + **F9**|Выравнивает вертикальные центры выбранных элементов управления с доминирующим управлением.|
|Format.AlignLefts|**Ctrl** + **Shift** + **Левая стрелка**|Выравнивает левые края выбранного элемента управления с доминирующим управлением.|
|Format.AlignMiddles|**F9**|Выравнивает горизонтальные центры выбранных элементов управления с доминирующим управлением.|
|Format.AlignRights|**Ctrl** + **Shift** + **Правая стрелка**|Выравнивает правые края выбранных элементов управления с доминирующим элементом управления.|
|Format.AlignTops|**Ctrl** + **Сдвиг** + **вверх стрелка**|Выравнивает верхние края выбранных элементов управления с доминирующим управлением.|
|Format.ButtonBottom|**Ctrl** + **B**|Разместите выбранные кнопки вдоль нижнего центра диалогового окна.|
|Format.ButtonRight|**Ctrl** + **R**|Размещает выбранные кнопки в правом верхнем углу диалогового окна.|
|Format.CenterHorizontal|**Ctrl** + **Shift** + **F9**|Центры управления горизонтально в диалоговом поле.|
|Format.CenterVertical|**Ctrl** + **F9**|Центры управления вертикально в диалоговом поле.|
|Format.CheckMnemonics|**Ctrl** + **M**|Проверяет уникальность мнемоники.|
|Формат.SizetoContent|**Сдвиг** + **F7**|Изображает выбранный элемент управления (ы) в соответствии с текстом заголовка.|
|Format.SpaceAcross|**Альт** + **Левая стрелка**|Равномерное пространство выбранных элементов управления горизонтально.|
|Format.SpaceDown|**Стрелка** + **Альт-Даун**|Равномерное пространство выбранных элементов управления по вертикали.|
|Format.TabOrder|**Ctrl** + **D**|Устанавливает порядок управления в диалоге.|
|Format.TestDialog|**Ctrl** + **T**|Запускает диалоговый ящик для проверки внешнего вида и поведения.|
|Format.ToggleGuides|**Ctrl** + **G**|Циклы между сеткой, руководящими принципами и сеткой для редактирования диалогов.|

- Чтобы изменить клавиши ярлыка, перейдите в меню **Параметры инструментов** > **Options**и выберите **клавиатуру** под папкой **«Окружающая среда».**

   Дополнительные сведения см. в разделе [Определение и настройка сочетаний клавиш в Visual Studio](/visualstudio/ide/identifying-and-customizing-keyboard-shortcuts-in-visual-studio).

- Чтобы изменить настройки, перейдите в меню **Инструменты** > **Импорт и Экспорт Настройки**.

   Варианты, доступные в диалоговых коробках, а также имена и места расположения команд меню, которые вы видите, могут отличаться от того, что описано в **Справке** в зависимости от активных настроек или издания.  Дополнительные сведения см. в разделе [Персонализация интегрированной среды разработки Visual Studio](/visualstudio/ide/personalizing-the-visual-studio-ide).
