# <a name="saving-and-loading-different-color-palettes-image-editor-for-icons"></a>Сохранение и загрузка различных цветовых палитр (редактор изображений для значков)

Можно сохранять и загружать **цвета** палитру, которая содержит [настраивать цвета](../windows/customizing-or-changing-colors-image-editor-for-icons.md). (По умолчанию **цвета** последняя использованная палитра автоматически загружается при запуске Visual Studio.)

> [!TIP]
> Так как **изображение** редактор имеет отсутствуют средства для восстановления по умолчанию **цвета** палитры, следует сохранить значение по умолчанию **цвета** палитру под именем, например  *Standard.PAL* или *default.pal* таким образом, можно легко восстановить параметры по умолчанию.

Используйте **Загрузка цветов палитры** диалоговое окно, чтобы загрузить специальные цветовые палитры для использования в проекте C++. Будут включены следующие свойства:

|Свойство.|Описание:|
|---|---|
|**Искать в**|Указывает расположение, где вы хотите найти файл или папку. Щелкните стрелку, чтобы выбрать и другое расположение или выберите значок папки на панели инструментов для перемещения вверх уровней.|
|**Имя файла**|Место для ввода имени файла, который требуется открыть. Чтобы быстро найти файл, который вы ранее открыли, выберите имя файла в раскрывающемся списке, если он доступен.<br/><br/>Если выполняется поиск файла, можно использовать звездочки (*) как символы-шаблоны. Например, можно ввести \*.\* для просмотра списка всех файлов. Можно также ввести полный путь файла, например, C:\My Documents\MyColorPalette.pal или \\\NetworkServer\MyFolder\MyColorPalette.pal.|
|**Тип файлов**|Список типов файлов для вывода. Палитра (* .pal) — это тип файла по умолчанию для палитры цветов.|

## <a name="to-save-a-custom-colors-palette"></a>Сохранение пользовательской палитры цветов

1. Из **изображение** меню, выберите **сохранить палитру**.

1. Перейдите в каталог, в который требуется сохранить палитру, и введите имя палитры.

1. Нажмите кнопку **Сохранить**.

## <a name="to-load-a-custom-colors-palette"></a>Загрузка пользовательской палитры цветов

1. Из **изображение** меню, выберите **Загрузка палитры**.

1. В **Загрузка палитры цветов** диалоговое окно, перейдите в нужный каталог и выберите палитру, которую нужно загрузить. **Цвет** палитры сохраняются с расширением PAL.

## <a name="requirements"></a>Требования

Нет
