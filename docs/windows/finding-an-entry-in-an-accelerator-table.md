# <a name="finding-an-entry-in-an-accelerator-table"></a>Поиск записи в таблице сочетаний клавиш

### <a name="to-find-an-entry-in-an-open-accelerator-table"></a>Поиск записи в открытой таблице сочетаний клавиш

1. Откройте таблицу сочетаний клавиш, дважды щелкнув его значок в [представление ресурсов](../windows/resource-view-window.md).

   > [!NOTE]
   > Если в проекте еще нет RC-файла, см. раздел [Создание нового файла описания ресурсов](../windows/how-to-create-a-resource-script-file.md).

2. Щелкните заголовок столбца, чтобы отсортировать содержимое столбца по алфавиту. Например, щелкните **идентификатор** для отображения всех идентификаторов в таблице в алфавитном порядке.

Затем просмотрите список и найдите запись.

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*.

## <a name="requirements"></a>Требования

Win32
