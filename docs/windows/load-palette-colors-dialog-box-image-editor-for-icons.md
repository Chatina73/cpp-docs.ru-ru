# <a name="load-palette-colors-dialog-box-c-image-editor-for-icons"></a>Загрузка цветов палитры-диалоговое окно (C++) (редактор изображений для значков)

Это диалоговое окно используется для загрузки специальные цветовые палитры для использования в проекте C++. Дополнительные сведения см. в разделе [сохранение и загрузка различных цветовых палитр](../windows/saving-and-loading-different-color-palettes-image-editor-for-icons.md).

### <a name="look-in"></a>Область поиска

Указывает расположение, где вы хотите найти файл или папку. Щелкните стрелку, чтобы выбрать другое расположение, или щелкните значок папки на панели инструментов для перемещения вверх уровней.

### <a name="file-name"></a>Имя файла

Место для ввода имени файла, который требуется открыть. Чтобы быстро найти файл, который вы ранее открыли, щелкните имя файла, в раскрывающемся списке, если он доступен.

Если для поиска файла, можно использовать звездочки (*) как символы-шаблоны. Например, можно ввести \*.\* для просмотра списка всех файлов. Можно также ввести полный путь файла, например, C:\My Documents\MyColorPalette.pal или \\\NetworkServer\MyFolder\MyColorPalette.pal.

### <a name="files-of-type"></a>Тип файлов

Список типов файлов для вывода. Палитра (* .pal) — это тип файла по умолчанию для палитры цветов.

## <a name="requirements"></a>Требования

Нет
