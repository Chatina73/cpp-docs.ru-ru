# <a name="editing-accelerator-tables-c"></a>Редактирование таблиц сочетаний клавиш (C++)

В проект C++, в таблицу сочетаний клавиш можно изменить непосредственно с редактирование на месте в **Accelerator** редактора.

Приведенные ниже процедуры относятся к страницам стандартных свойств, тем не менее, как редактирование на месте, так и метод страницы свойств обеспечивают одинаковый результат. Изменения, внесенные с помощью страницы свойств или редактирование на месте немедленно отражаются в таблице сочетаний клавиш.

- [Редактирование в таблице сочетаний клавиш](../windows/editing-in-an-accelerator-table.md)

- [Поиск записи в таблице сочетаний клавиш](../windows/finding-an-entry-in-an-accelerator-table.md)

- [Добавление записи в таблицу сочетаний клавиш](../windows/adding-an-entry-to-an-accelerator-table.md)

- [Удаление записи из таблицы сочетаний клавиш](../windows/deleting-an-entry-from-an-accelerator-table.md)

- [Перемещение или копирование элемента таблицы сочетаний клавиш](../windows/moving-or-copying-an-accelerator-table-entry-to-another-resource-script-file.md)

- [Изменение свойств для нескольких сочетаний клавиш](../windows/changing-the-properties-of-multiple-accelerator-keys.md)
