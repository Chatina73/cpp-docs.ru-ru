# <a name="toolbar-editor-c"></a>Редактор панелей инструментов (C++)

**Редактор панелей инструментов** позволяет создавать ресурсы панелей инструментов и преобразовывать растровые изображения в ресурсы панелей инструментов. **Редактор панелей инструментов** используется графическое изображение для отображения панели инструментов и кнопок, в значительной степени напоминают как они будут выглядеть в готовое приложение.

**Редактор панелей инструментов** окне отображаются два представления изображение кнопки, так же, как **редактор изображений** окна. Две области разделены разделителем и вправо, чтобы изменить соотношение областей, можно перетащить разделителе со стороны. Активную область отображает граница выделения и над представлениями изображения тематическая панель инструментов.

![Редактор панелей инструментов](../mfc/media/vctoolbareditor.gif "vcToolbarEditor")<br/>
**Редактор панелей инструментов**

**Редактор панелей инструментов** аналогичен **редактор изображений** в функциональные возможности и элементы меню, инструменты рисования и сетка растрового между ними отличаются. В есть команда меню **изображение** меню для переключения между **редактор панелей инструментов** и **редактор изображений**. Дополнительные сведения об использовании **графики** инструментов **цвета** палитре, или **изображение** меню, см. в разделе [редактор изображений](../windows/image-editor-for-icons.md).

Новая панель инструментов можно создать в проекте C++ путем преобразования растрового изображения. Рисунок из точечного рисунка преобразует изображениями кнопок панели инструментов. Обычно растровое изображение содержит несколько изображений кнопок отдельной с один образ для каждой кнопки. Образы могут быть любого размера, по умолчанию, 16 пикселей в ширину и высоту изображения. Можно указать размер изображения кнопок в **Создание ресурса панели инструментов** диалоговое окно при выборе **редактор панелей инструментов** из **изображение** меню во время в  **Редактор изображений**.

**Создание ресурса панели инструментов** диалоговое окно позволяет указать ширину и высоту кнопки, вы добавляете к ресурса панели инструментов в проекте C++. Значение по умолчанию — 16 х 15 пикселей.

Точечный рисунок, который используется для создания панели инструментов имеет максимальную ширину 2048, поэтому, если вы установите **Ширина кнопки** для *512*, может иметь только четыре кнопки. Если задано значение ширины *513*, может иметь только три кнопки.

**Создание ресурса панели инструментов** диалоговое окно имеет следующие свойства:

|Свойство.|Описание:|
|---|---------------|
|**Ширина кнопки**|Место для ввода ширину для кнопок панели инструментов, преобразовании ресурса точечного рисунка панели инструментов.|
|**Высота кнопки**|Место для ввода высоту для кнопок панели инструментов, преобразовании ресурса точечного рисунка панели инструментов.|

> [!NOTE]
> Изображения обрезаются до ширины и высоты, заданной и цвета корректируются для использования стандартной палитры панели инструментов (16 цветов).

По умолчанию новая или пустая кнопка отображается в правом конце панели инструментов. Можно переместить эту кнопку, прежде чем изменять его. При создании новой кнопки, еще одна пустая кнопка справа от нее. При сохранении панели инструментов пустая кнопка не сохраняется.

Кнопки панели инструментов имеет следующие свойства:

|Свойство.|Описание:|
|--------------|-----------------|
|**ID**|Определяет идентификатор для кнопки. Выберите в раскрывающемся списке предоставляет Общие **идентификатор** имена.|
|**Width**|Задает ширину кнопки. рекомендуется 16 пикселей.|
|**Height**|Задает высоту кнопки. Высота кнопки изменяет высоту всех кнопок на панели инструментов. Рекомендуется 15 пикселей.|
|**Запрашивать**|Определяет сообщение, отображаемое в строке состояния. Добавление *\n* и добавляет имя **подсказки** для этой кнопки панели инструментов. Дополнительные сведения см. в разделе [Создание всплывающей подсказки](../windows/creating-a-tool-tip-for-a-toolbar-button.md).|

**Ширина** и **высота** применяются ко всем кнопкам. Точечный рисунок, который используется для создания панели инструментов имеет максимальную ширину 2048, поэтому если ширину кнопки *512*, может иметь только четыре кнопки, и если вам для настройки ширины *513*, может иметь только три кнопки.

## <a name="how-to"></a>Практические руководства

**Редактор панелей инструментов** позволяет:

### <a name="to-create-new-toolbars"></a>Создание новых панелей инструментов

1. В **ресурсов** просмотра, щелкните правой кнопкой мыши RC-файл и выберите **добавить ресурс**. При наличии существующую панель инструментов в RC-файл, можно щелкнуть **инструментов** папку и выберите **вставить панель инструментов**.

1. В **добавить ресурс** выберите **инструментов** в **тип ресурса** списке, а затем выберите **New**.

   Если знак плюс (**+**) отображается рядом **инструментов** тип ресурса, это означает, что доступны шаблоны. Выберите знак «плюс», чтобы развернуть список шаблонов, выберите шаблон и выберите **New**.

### <a name="to-convert-bitmaps-to-toolbar-resources"></a>Чтобы преобразовывать растровые изображения в ресурс панели инструментов

1. Откройте существующий ресурс растрового изображения в [редактор изображений](../windows/image-editor-for-icons.md). Если растровое изображение не в RC-файл, щелкните правой кнопкой мыши RC-файл и выберите **импорта**, перейдите к растровому изображению, необходимо добавить в RC-файл и выберите **откройте**.

1. Перейдите к меню **изображение** > **редактор панелей инструментов**.

   **Создание ресурса панели инструментов** откроется диалоговое окно. Можно изменить ширину и высоту изображения значков в соответствии с растрового изображения. Изображение панели инструментов отображается в **редактор панелей инструментов**.

1. Чтобы завершить преобразование, измените команду **идентификатор** кнопки с помощью [окно "Свойства"](/visualstudio/ide/reference/properties-window). Введите новое *идентификатор* или выберите **идентификатор** из раскрывающегося списка.

   > [!TIP]
   > **Свойства** окне есть кнопка в строке заголовка и при выборе этого включает или отключает **автоматическое скрытие** для окна. Для циклического перебора все свойства кнопок панели инструментов без необходимости повторно открыть windows отдельного свойства, включить **автоматическое скрытие** off поэтому **свойства** окно остается стационарным.

   Можно также изменить идентификаторы команд кнопок на новой панели инструментов с помощью [окно "Свойства"](/visualstudio/ide/reference/properties-window).

### <a name="to-manage-toolbar-buttons"></a>Для управления кнопки панели инструментов

Чтобы создать новую кнопку панели инструментов:

1. В [представление ресурсов](../windows/resource-view-window.md) разверните папку ресурсов (например, *Project1.rc*).

1. Разверните **инструментов** папку и выберите панель инструментов, чтобы изменить, выполните одно из следующих:

   - Назначение идентификатора пустую кнопку в правом конце панели инструментов. Это можно сделать, изменив **идентификатор** свойство в [окно "Свойства"](/visualstudio/ide/reference/properties-window). Например можно предоставить один и тот же идентификатор как пункт меню для кнопки панели инструментов. В этом случае используйте поле с раскрывающимся списком для выбора **идентификатор** пункта меню.

   - Выберите пустую кнопку в правом конце панели инструментов (в **инструментов представления** области) и начала рисования. Идентификатор команды кнопки по умолчанию назначается (ID_BUTTON\<n >).

Кроме того, можно скопировать и вставить изображение на панель инструментов в качестве новой кнопки.

Добавление изображения на панель инструментов, как кнопка:

1. В [представление ресурсов](../windows/resource-view-window.md), откройте панель инструментов, дважды щелкнув его.

1. Затем откройте образ, который вы хотите добавить на панель инструментов.

   > [!NOTE]
   > При открытии изображения в Visual Studio, он открывается в **изображение** редактора. Можно также открыть изображение в других графических программ.

1. Из **изменить** меню, выберите **копирования**.

1. Переключитесь на панель инструментов, выбрав его вкладки в верхней части окна исходного кода.

1. Из **изменить** меню, выберите **вставить**.

   Изображение будет отображаться на панели инструментов, как новая кнопка.

Перемещение кнопки на панели инструментов:

В **инструментов представления** панели, перетащите кнопку, вы хотите переместить в новое место на панели инструментов.

Копирование кнопок из панели инструментов, удерживая нажатой **Ctrl** ключа и в **инструментов представления** панели, перетащите кнопку либо новое место на панели инструментов или в расположение на другой панели инструментов.

Чтобы удалить кнопку панели инструментов, выберите кнопку панели инструментов и перетащите его с панели инструментов.

Чтобы вставить или удалить промежутка между кнопками на панели инструментов, либо перетащите их из или к друг с другом на панели инструментов.

|Действие|Шаг|
|------|------|
|Чтобы вставить пробел перед кнопкой, не разделенных пробелами|Перетащите кнопку вправо или вниз, пока он перекрывает "Далее" о посередине.|
|Вставлять пробел перед кнопкой, за которой следуют пробел и сохранить этот пробел|Перетащите кнопку до правого или нижнего края коснулся "Далее", или даже перекрыл ее.|
|Вставлять пробел перед кнопкой, за которой следуют пробел и закрыть следующие пространство|Перетащите кнопку вправо или вниз, пока он перекрывает "Далее" о посередине.|
|Удаление промежутка между кнопками на панели инструментов|Перетащите кнопку на одной стороне пространства к кнопке на другой стороне пространства, чтобы он перекрывает приблизительно половина "Далее".|

> [!NOTE]
> Если нет места перетаскиваемой кнопки перетаскивания из и перетащить ее более чем на равном расстоянии за кнопку рядом, **редактор панелей инструментов** вставляет символ пробела на противоположной стороне кнопки, перетаскивания.

Изменение свойств кнопки панели инструментов:

1. В проекте C++ нажмите кнопку панели инструментов.

1. Введите новый идентификатор в **идентификатор** свойство в [окно "Свойства"](/visualstudio/ide/reference/properties-window), или используйте стрелку раскрывающегося списка, чтобы выбрать новую **идентификатор**.

Создание всплывающей подсказки для кнопки панели инструментов:

1. Нажмите кнопку панели инструментов.

1. В [окно "Свойства"](/visualstudio/ide/reference/properties-window)в **Prompt** поле, добавьте описание кнопки для строки состояния, а после сообщения, добавить `\n` и имя всплывающей подсказки.

Например, см. в разделе всплывающая подсказка для **печати** кнопку **WordPad**:

1. Откройте **WordPad**.

1. Наведите указатель мыши **печати** кнопки панели инструментов и обратите внимание, что слово `Print` теперь располагается под указателем мыши.

1. Посмотрите на строке состояния в нижней части **WordPad** и обратите внимание, что теперь отображается текст `Prints the active document`.

`Print` — Это имя всплывающей подсказки и `Prints the active document` представляет собой описание кнопки для строки состояния.

Если этот эффект с помощью функции **редактор панелей инструментов**, задайте **Prompt** свойства `Prints the active document\nPrint`.
