# <a name="removing-space-between-buttons-on-a-toolbar-c"></a>Удаление промежутка между кнопками на панели инструментов (C++)

Ниже показано, как удалить промежутка между кнопками на панели инструментов.

### <a name="to-remove-a-space-between-buttons-on-a-toolbar"></a>Удаление промежутка между кнопками на панели инструментов

1. Перетащите кнопку на одной стороне пространства к кнопке на другой стороне пространства, чтобы он перекрывает приблизительно половина "Далее".

   Если нет места перетаскиваемой кнопки при перетаскивании из и перетащить ее более чем на равном расстоянии за кнопку рядом, **инструментов** редактора также вставляет символ пробела на противоположной стороне, кнопки, которую вы являетесь перетаскивание.
