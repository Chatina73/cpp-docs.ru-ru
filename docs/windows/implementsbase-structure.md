# <a name="implementsbase-structure"></a>ImplementsBase - структура

Поддерживает инфраструктуру WRL и не предназначен для использования непосредственно из программного кода.

## <a name="syntax"></a>Синтаксис

```cpp
struct ImplementsBase;
```

## <a name="remarks"></a>Примечания

Используется для проверки типов параметров шаблона в [Implements-структура](../windows/implements-structure.md).

**ImplementsBase** структура пуста намеренно.

## <a name="inheritance-hierarchy"></a>Иерархия наследования

`ImplementsBase`

**Пространство имен:** Microsoft::wrl:: Details
