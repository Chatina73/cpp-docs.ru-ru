# <a name="displaybind"></a>displaybind

Указывает свойство, которое должно быть отображено пользователю как связываемое.

## <a name="syntax"></a>Синтаксис

```cpp
[displaybind]
```

## <a name="remarks"></a>Remarks

Атрибут **дисплайбинд** C++ имеет те же функциональные возможности, что и атрибут [дисплайбинд](/windows/win32/Midl/displaybind) MIDL.

## <a name="example"></a>Пример

Пример использования **дисплайбинд** см. в примере для [привязки](bindable.md) .

## <a name="requirements"></a>Требования

| Контекст атрибута | Значение |
|-|-|
|**Относится к**|Метод интерфейса|
|**REPEATABLE**|Нет|
|**Требуемые атрибуты**|Нет|
|**Недопустимые атрибуты**|Нет|

Дополнительные сведения см. в разделе [Контексты атрибутов](cpp-attributes-com-net.md#contexts).
