# <a name="array-attributes"></a>Атрибуты массивов

Следующие атрибуты применяются к массивам или `SAFEARRAY`s.

|attribute|Description|
|---------------|-----------------|
|[library_block](library-block.md)|Помещает конструкцию внутрь блока библиотеки IDL-файла.|
|[satype](satype.md)|Указывает тип данных структуры `SAFEARRAY`.|
|[строка](string-cpp.md)|Указывает, что одномерный массив **char**, **wchar_t**, `byte` (или эквивалентный) или указатель на такой массив должен рассматриваться как строка.|
