# <a name="array-attributes"></a>Атрибуты массивов

Следующие атрибуты применяются к массивам или `SAFEARRAY` .

|attribute|Описание:|
|---------------|-----------------|
|[library_block](library-block.md)|Помещает конструкцию внутрь блока библиотеки IDL-файла.|
|[satype](satype.md)|Указывает тип данных `SAFEARRAY` структуры.|
|[строка](string-cpp.md)|Указывает, что одномерный **`char`** массив, **`wchar_t`** `byte` (эквивалентный) или указатель на такой массив должен рассматриваться как строка.|
