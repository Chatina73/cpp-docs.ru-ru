# <a name="how-to-layout-controls-c"></a>Как выполнить: Элементы управления макета (C++)

**Редактор диалоговых окон** предоставляет средства разметки, выравнивания и размера элементов управления автоматически. Для выполнения большинства задач можно использовать [редактор диалоговых окон инструментов](../windows/showing-or-hiding-the-dialog-editor-toolbar.md). Все **редактор диалоговых окон** команды панели инструментов, также доступны в **формат** меню, а большинство имеют [сочетания клавиш](../windows/accelerator-keys-for-the-dialog-editor.md).

Многие команды макета по диалоговым окнам, доступны только в том случае, если выбрано несколько элементов управления. Можно выбрать один или несколько элементов управления, и при выборе нескольких элементов управления, первый выбранный по умолчанию главного элемента управления.

Расположение, высоту и ширину текущего элемента управления отображаются в правом нижнем углу строки состояния. При выборе всего диалоговое окно, в строке состояния отображается положение диалогового окна как единое целое и его высоту и ширину.

## <a name="arrange-controls"></a>Размещение элементов управления

Можно упорядочить элементы управления в диалоговых окнах с **редактор диалоговых окон** в одном из трех разных состояний:

- С помощью полей и направляющих на установите по умолчанию.

- С помощью сетки расположения.

- Без привязки или выравнивать их функции.

[Редактор диалоговых окон инструментов](../windows/showing-or-hiding-the-dialog-editor-toolbar.md) содержит кнопки, управления состоянием.

- Чтобы изменить состояние, выберите соответствующий значок, или перейдите к меню **формат** > **параметры направляющих**.

**Параметры направляющих** диалоговое окно имеет следующие свойства:

|Свойство|Описание|
|---|---|
|**Направляющие разметки**|Параметры для направляющих макета.|
|**None**|Скрывает макета.|
|**Линейки и направляющие**|Если этот параметр включен, добавляет линейки средство «layout», и позволяет руководства должно быть помещено в линейки. В руководствах по умолчанию – это поля.|
|**Сетка**|Создание сетки макета. Новые элементы управления автоматически будет соответствовать сетки.|
|**Шаг сетки**|Параметры для шаг сетки в единицах диалогового (окна DLU).|
|**Ширина: DLU**|Задает ширину сетки макета в единицах диалогового окна. Горизонтальная Единица диалогового окна — среднюю ширину шрифта диалогового поделенное на 4.|
|**Высота: DLU**|Задает высоту сетки макета в единицах диалогового окна. Вертикальная Единица диалогового окна — это средняя высота шрифта диалогового разделенное на 8.|

### <a name="guides-and-margins"></a>Полей и направляющих

Ли вы перемещаете элементы управления, добавление элементов управления или изменения порядка текущий макет, полей и направляющих могут помочь выравнивания элементов управления точно в диалоговом окне.

При создании диалоговое окно, четыре измененный направляющие, именем поля предоставляются и отображаются в виде синей пунктирными линиями.

- Чтобы переместить поля, перетащите поле на новое место.

- Чтобы удалить поле, переместите поля нулевой позиции.

- Чтобы вернуть поля, наведите указатель на поле нулевой позиции и перетащите поле в позиции.

Направляющие отображаются пунктирными линиями, как синий в диалоговом окне отображаются в окне редактора и соответствующий стрелки на линейки в верхней и левой части **редактор диалоговых окон**. Маркеры изменения размера элементов управления привязка к описаниям, при перемещении элементов управления, а направляющие привязываются к элементам управления, если нет элементов управления, ранее привязываются к структуре. При перемещении по элементам управления, привязанные к нему переместить также. Элементы управления привязаны к нескольких структур изменяется при перемещении одно из руководств.

- Для создания структуры внутри линейки, выберите один раз, чтобы создать структуру, или двойным щелчком запустите **параметры направляющих** диалоговое окно, где можно указать параметры направляющих.

- Чтобы задать направляющую в диалоговом окне, выберите руководство по и перетащите его в новую позицию или выберите стрелку на линейке перетащите направляющую в связанные.

   Координаты структуры, отображаются в строке состояния в нижней части окна и на линейке или переместите указатель над стрелкой на линейке, чтобы отобразить точную позицию руководства.

- Удаление структуры, перетащите направляющую в соответствующем диалоговом окне или соответствующие стрелка за пределы линейки.

Деления линейки, которые определяют интервал направляющих и элементов управления определяются единицы диалогового (окна DLU). Единица основан на размер шрифта диалогового, обычно 8 пунктов MS Shell Dlg. Горизонтальная Единица диалогового окна — среднюю ширину деления четыре на шрифт диалогового окна. Вертикальная Единица диалогового окна — это средняя высота шрифта, разделенное на 8.

- Чтобы изменить интервалы меток делений, перейдите к меню **формат** > **параметры направляющих**, затем в **шаг сетки** укажите новая ширина и высота в единицах диалогового окна.

### <a name="layout-grid"></a>Макет сетки

При помещении или упорядочение элементов управления в диалоговом окне, используйте сетки макета для более точного размещения. Если сетка включена, как намагниченный элементов управления будет привязана пунктирными линиями сетки.

- Чтобы выключать сетки макета, перейдите к меню **формат** > **параметры направляющих** и установите или снимите **сетки** кнопки.

   Можно управлять сетки в отдельных **редактор диалоговых окон** windows с помощью **сетка** кнопку [редактор диалоговых окон инструментов](../windows/showing-or-hiding-the-dialog-editor-toolbar.md).

- Чтобы изменить размер сетки макета, перейдите к меню **формат** > **параметры направляющих** и высоту и ширину в единицах диалогового окна для ячейки в сетке. Минимальные значения высоты или ширины — 4.

### <a name="disable-guides"></a>Отключение структур

Отключение эффекта привязки руководства можно использовать специальные ключи в сочетании с помощью мыши. С помощью **Alt** ключ отключает эффект привязки для выбранной направляющей. Перемещение направляющей при **Shift** ключа не позволяет перемещать со структурой привязанных элементов управления.

- Отключение эффекта привязки из руководств, перетащите элемент управления, удерживая нажатой **Alt** ключ.

- Чтобы переместить направляющие без перемещения привязанных элементов управления, перетащите направляющую при нажатой **Shift** ключ.

- Чтобы отключить направляющие, перейдите к меню **формат** > **параметры направляющих**. Затем в разделе **направляющих макета**выберите **None**.

   > [!TIP]
   > Можно также использовать ярлык в меню **формат** > **направляющие**.

## <a name="select-controls"></a>Выберите элементы управления

Выберите элементы управления до размера, выравнивание, переместить, скопировать, или удалите их и завершите нужную операцию. В большинстве случаев необходимо выбрать более одного элемента управления, чтобы использовать средства изменения размера и выравнивания на [редактор диалоговых окон инструментов](../windows/showing-or-hiding-the-dialog-editor-toolbar.md).

При выборе элемента управления, он имеет затененную границу вокруг его с использованием (активный) или пустыми (неактивными) маркеры, небольшие квадраты, которые отображаются в границы выделения. Если выбрано несколько элементов управления, главного элемента управления имеет сплошной маркеры и все другие выбранные элементы управления имеют без заливки маркеров.

- Чтобы выбрать элементы управления, в [окно панели элементов](/visualstudio/ide/reference/toolbox)выберите **указатель** средство и используйте один из следующих действий для выбора:

  - Перетащите указатель, чтобы нарисовать рамку выделения вокруг элементов управления, которые вы хотите выбрать в диалоговом окне. Если отпустить кнопку мыши, все элементы управления внутри и пересекающиеся выбраны поле выбора.

  - Удерживая нажатой **Shift** ключ и выбрать элементы управления, которые вы хотите включить в список.

  - Удерживая нажатой **Ctrl** ключ и выбрать элементы управления, которые вы хотите включить в список.

- Чтобы добавить или удалить элемент управления из группы выбранные элементы управления, удерживая нажатой **Shift** ключ и выбрать элемент управления, нужно добавить или удалить.

### <a name="dominant-controls"></a>Главные элементы управления

При изменения размера или выравнивание нескольких элементов управления, **редактор диалоговых окон** использует главного элемента управления, чтобы определить, как размер или выровнены других элементов управления. По умолчанию главного элемента управления является первого выбранного элемента управления.

- Чтобы указать главного элемента управления, удерживая нажатой **Ctrl** ключа и выберите элемент управления, вы хотите использовать, чтобы повлиять на размер или расположение других элементов управления *первый*. Удерживая нажатой **Ctrl** ключа и выбора элемента управления в составе также сделает, которые управляют главного элемента управления в указанной области.

- Чтобы изменить главного элемента управления, очистить текущее выделение, выбрав за пределами всех выбранных элементов управления и повторите описанные выше действия, выбрав другой элемент управления *первый*.

> [!NOTE]
> Маркеры изменения размера главного элемента управления показываются сплошными, а маркеры подчиненных элементов управления являются пустыми. Все дальнейшее изменение размера или выравнивание зависит от главного элемента управления.

## <a name="size-controls"></a>Размер элементов управления

Используйте маркеры размер элемента управления. Когда указатель находится на маркер, меняет форму для указания направления, в которых можно изменить размер элемента управления. Активные маркеры solid и если маркер является пустой, невозможно изменить размер элемента управления по этой оси.

- Изменение размера элемента управления, выберите элемент управления и перетащите маркеры изменения размера.

  - Размер маркеров в верхней и сторон изменить размер горизонтальную или вертикальную.

  - Размер маркеров в углах изменить горизонтальный и вертикальный размер.

   > [!TIP]
   > Можно изменить размер элемента управления одного диалога единица (Единица) за раз, удерживая нажатой **Shift** ключа и используя **справа** и **вниз** клавиш со стрелками.

- Автоматическое изменение размера элемента управления в соответствии с текстом внутри него, перейдите к меню **формат** или щелкните правой кнопкой мыши элемент управления и выберите **размер по содержимому**.

- Чтобы настроить элементы управления на тот же размер, выберите элементы управления, необходимо изменить размер и перейдите к меню **формат** > **установить тот же размер**, затем выберите либо **оба**, **Высота**, или **ширины**.

   Вы измените размер группы элементов управления в зависимости от размера главного элемента управления, являющегося первый выбранный в ряду. Окончательный размер элементов управления в группе зависит от размера главного элемента управления.

- Размера группы элементов управления с помощью руководств, привязать к структуре одной стороны элемента управления (или элементов управления), а затем перетащите направляющую другой стороне элемента управления (или элементов управления). Теперь вы можете перейти либо руководство для определения размера элемента управления (или элементов управления).

   При необходимости с несколькими элементами управления, размер каждого, чтобы привязать к второй направляющей.

## <a name="align-controls"></a>Выравнивание элементов управления

- Для выравнивания элементов управления, выберите элементы управления, которые требуется выровнять. Перейдите к меню **формат** > **выровнять** и выберите один из следующих вариантов выравнивания:

   |Выравнивание|Описание|
   |-----|-----------|
   |**Левых границ**|Выравнивание элементов управления левой границы.|
   |**Центры**|Выравнивание элементов управления по горизонтали по центру.|
   |**Права**|Выравнивает правые границы выбранных элементов управления.|
   |**Верхних границ**|Выравнивание по верхнему краю выбранных элементов управления.|
   |**По середине**|Выравнивание выделенных элементов управления по вертикали их средней точки.|
   |**Нижние границы**|Выравнивание элементов управления по нижнему краю.|

   Не забудьте выбрать элемент управления, который вы хотите сначала преобладать или установке его главного элемента управления перед выполнением выравнивание или изменения размера команды, а последняя позиция группа элементов управления, зависит от позиции главного элемента управления.

- Чтобы равномерно элементами управления выберите элементы управления, которые нужно изменить. Перейдите к меню **формат** > **пространство равномерно** и выберите один из следующих вариантов выравнивания:

   |Интервал|Описание|
   |---|---|
   |**Между**|Элементы управления пространство равномерно между крайней левой и правой выбранного элемента управления.|
   |**Вниз**|Элементы управления пространство равномерно между верхним и нижним элементом управления.|

- Центрирование элементов управления, выберите элемент управления или элементы управления, которые вы хотите изменить. Перейдите к меню **формат** > **Выровнять по центру** и выберите один из следующих порядков:

   |Упорядочение|Описание|
   |---|---|
   |**По вертикали**|Центрирование элементов управления по вертикали в диалоговом окне.|
   |**По горизонтали**|Центрирование элементов управления по горизонтали в диалоговом окне.|

- Чтобы выровнять кнопки, выберите один или несколько кнопок. Перейдите к меню **формат** > **упорядочить кнопки**, затем выберите один из следующих порядков:

   |Упорядочение|Описание|
   |---|---|
   |**Right**|Выравнивание кнопок по правому краю диалогового окна.|
   |**Снизу**|Выравнивает кнопки вдоль нижней границы окна.|

   Если выбрать элемент управления не является кнопкой, его положение не влияет.
