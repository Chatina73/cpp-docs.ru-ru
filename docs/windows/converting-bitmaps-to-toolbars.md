# <a name="converting-bitmaps-to-toolbars-c"></a>Преобразование растровых изображений в панели инструментов (C++)

Новая панель инструментов можно создать в проекте C++ путем преобразования растрового изображения. Рисунок из точечного рисунка преобразует изображениями кнопок панели инструментов. Обычно растровое изображение содержит несколько изображений кнопок отдельной с один образ для каждой кнопки. Образы могут быть любого размера; значение по умолчанию — 16 пикселей в ширину и высоту изображения. Можно указать размер изображения кнопок в **Создание ресурса панели инструментов** диалоговое окно при выборе **редактор панелей инструментов** из **изображение** меню в редакторе изображений.

**Создание ресурса панели инструментов** диалоговое окно позволяет указать ширину и высоту кнопки, вы добавляете к ресурса панели инструментов в проекте C++. Значение по умолчанию — 16 х 15 пикселей.

Битовая карта, используется для создания панели инструментов имеет максимальную ширину 2048. Поэтому, если задать **Ширина кнопки** 512, может быть только четыре кнопки. Если вам для настройки ширины 513, может иметь только три кнопки.

Диалоговое окно имеет следующие свойства:

|Свойство.|Описание:|
|---|---|
|**Ширина кнопки**|Место для ввода ширину для кнопок панели инструментов, преобразовании ресурса точечного рисунка панели инструментов. Изображения обрезаются до ширины и высоты, заданной и цвета корректируются для использования стандартной палитры панели инструментов (16 цветов).|
|**Высота кнопки**|Место для ввода высоту для кнопок панели инструментов, преобразовании ресурса точечного рисунка панели инструментов. Изображения обрезаются до ширины и высоты, заданной и цвета корректируются для использования стандартной палитры панели инструментов (16 цветов).|

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).

## <a name="to-convert-bitmaps-to-a-toolbar"></a>Чтобы преобразовывать растровые изображения на панель инструментов

1. Откройте существующий ресурс растрового изображения в [редактор изображений](../windows/image-editor-for-icons.md). (Если растровое изображение не в RC-файл, щелкните правой кнопкой мыши RC-файл, выберите **импорта** в контекстном меню, перейдите к растровому изображению, вы хотите добавить в RC-файл, а затем выберите **откройте**.)

1. Из **изображение** меню, выберите **редактор панелей инструментов**.

   **Создание ресурса панели инструментов** откроется диалоговое окно. Можно изменить ширину и высоту изображения значков в соответствии с растрового изображения. Изображение панели инструментов отображается в редакторе панелей инструментов.

1. Чтобы завершить преобразование, измените команду **идентификатор** кнопки с помощью [окно "Свойства"](/visualstudio/ide/reference/properties-window). Введите новое **идентификатор** или выберите **идентификатор** из раскрывающегося списка.

   > [!TIP]
   > **Свойства** окно содержит кнопку канцелярской кнопки в заголовке окна. Нажатия этой кнопки включает или отключает **автоматическое скрытие** для окна. Чтобы быстро пролистать все свойства кнопок панели инструментов без необходимости повторно открыть windows отдельного свойства, включить **автоматическое скрытие** off поэтому **свойства** окно остается стационарным.

Можно также изменить идентификаторы команд кнопок на новой панели инструментов с помощью [окно "Свойства"](/visualstudio/ide/reference/properties-window). Сведения о редактировании новой панели инструментов см. в разделе [создание, перемещение и редактирование кнопок панели инструментов](../windows/creating-moving-and-editing-toolbar-buttons.md).
