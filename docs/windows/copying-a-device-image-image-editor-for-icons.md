# <a name="copying-a-device-image-image-editor-for-icons"></a>Копирование изображения устройства (редактор изображений для значков)

### <a name="to-copy-a-device-image"></a>Чтобы скопировать изображение устройства

1. На **изображение** меню, щелкните **открыть изображение устройства** и выбрать образ из текущего списка изображений. Например можно выберите 32 × 32, 16 цветов версию значка.

2. Скопируйте изображение в настоящее время значка (**Ctrl**+**C**).

3. Откройте другое изображение значка в другом **редактор изображений** окна. Например откройте версия значка в 16 цветов, 16 × 16.

4. Вставьте изображение значка (**Ctrl**+**V**) из одного **редактор изображений** окна к другому. При вставке большего размера в меньшего размера, можно использовать значок маркеры изменения размера изображения.

Сведения о добавлении ресурсов в управляемые проекты см. в разделе [ресурсы в приложениях для настольных систем](/dotnet/framework/resources/index) в *руководства разработчика .NET Framework*. Сведения о вручную добавлять файлы ресурсов в управляемые проекты, осуществлять доступ к ресурсам, отображать статические ресурсы и присваивать строки ресурсов свойствам, см. в разделе [Создание файлов ресурсов для приложений рабочего стола](/dotnet/framework/resources/creating-resource-files-for-desktop-apps). Сведения о глобализации и локализации ресурсов в управляемых приложениях, см. в разделе [Globalizing and Localizing .NET Framework Applications](/dotnet/standard/globalization-localization/index).
