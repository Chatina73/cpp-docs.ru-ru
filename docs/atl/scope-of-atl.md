# <a name="scope-of-atl"></a>Область применения ATL

ATL позволяет легко создавать объекты COM, серверы автоматизации и элементы управления ActiveX. ATL предоставляет встроенную поддержку многих основных COM-интерфейсов.

ATL поставляется в качестве исходного кода, включаемого в приложение. Кроме того, Библиотека ATL предоставляет доступ к библиотеке DLL (atl90.dll), которая содержит код, который можно совместно использовать в компонентах. Однако эта библиотека DLL не является обязательной.
