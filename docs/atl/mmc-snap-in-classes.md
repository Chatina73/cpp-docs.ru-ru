# <a name="mmc-snap-in-classes"></a>Классы Snap-In MMC

Следующие классы обеспечивают поддержку разработки компонентов оснастки консоли управления (MMC).

- [Кснапинитемимпл](../atl/reference/csnapinitemimpl-class.md) Реализует объект узла оснастки, например добавление пунктов меню и панелей инструментов, а также команды перенаправления для узла оснастки в соответствующую функцию обработчика.

- [Кснапинпропертипажеимпл](../atl/reference/csnapinpropertypageimpl-class.md) Реализует объект страницы свойств оснастки.
