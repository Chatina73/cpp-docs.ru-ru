# <a name="stock-properties-atl-control-wizard"></a>Свойства стеков, мастер элементов управления ATL

На этой странице мастера определяются стандартные свойства, поддерживаемые элементом управления. По умолчанию свойства не идентифицируются.

- **Не поддерживается**

   Указывает доступные свойства, которые в настоящее время не поддерживаются для элемента управления.

- **Поддерживается**

   Указывает свойства, которые в настоящее время поддерживаются для элемента управления.

|Кнопка "Перенести"|Описание|
|---------------------|-----------------|
|**>**|Добавляет в список **поддерживаемых** , имя свойства, которое в данный момент выбрано в списке **не поддерживается** .|
|**>>**|Добавляет в список **поддерживаемых** , все имена свойств, доступные в списке **не поддерживается** .|
|**\<**|Удаляет имя свойства, которое в данный момент выбрано в списке **поддерживаемых** .|
|**\<\<**|Удаляет все имена свойств, которые в настоящее время перечислены в списке **поддерживаемых** .|
