# <a name="creating-an-atl-project"></a>Создание проекта ATL

Проще всего создавать проекты ATL с помощью мастера проектов ATL, который располагается в папке **Проекты Win32** в диалоговом окне **Создание проекта**.

## <a name="to-create-an-atl-project-using-the-atl-project-wizard"></a>Создание проекта ATL с помощью мастера проектов ATL

1. В Visual Studio последовательно выберите **Файл > Создать > Проект**.

1. Щелкните значок **Проект ATL** в области **Шаблоны**, чтобы открыть **мастер проектов ATL**.

1. Задайте параметры приложения на странице [Параметры приложения](../../atl/reference/application-settings-atl-project-wizard.md) в **мастере проектов ATL**.

   > [!NOTE]
   > Для сохранения параметров, заданных в мастере по умолчанию, пропустите этот шаг.

1. Нажмите кнопку **Готово** для завершения работы мастера и откройте новый проект в среде разработки.

После создания проекта можно просмотреть его файлы при помощи **обозревателя решений**. Дополнительные сведения о файлах, создаваемых мастером для проекта, см. в созданном для проекта файле ReadMe.txt. Дополнительные сведения о типах файлов см. в разделе [Типы файлов, создаваемых для проектов Visual Studio C++](../../build/reference/file-types-created-for-visual-cpp-projects.md). Дополнительные сведения о конфигурациях для нового проекта ATL, а также об их изменении см. в разделе [Конфигурации проектов ATL по умолчанию](../../atl/reference/default-atl-project-configurations.md).
