# <a name="default-atl-project-configurations"></a>Конфигурации проектов ATL по умолчанию

По умолчанию мастер проектов ATL создает две конфигурации проекта:

|Конфигурация|Набор знаков|Использование ATL|
|-------------------|-------------------|----------------|
|Release|MBCS|DLL|
|Отладка|MBCS|DLL|

**Кодировка**, **Использование ATL** и все это можно изменить в диалоговом окне " **Параметры проекта** " на вкладке " **Общие** ". Можно также добавить собственные конфигурации с помощью Configuration Manager. Дополнительные сведения см. в разделе [конфигурации сборки](/visualstudio/ide/understanding-build-configurations).
