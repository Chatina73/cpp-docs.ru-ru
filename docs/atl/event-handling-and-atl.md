# <a name="event-handling-and-atl"></a>Обработка событий и ATL

В этом разделе показано, как присвоить события с помощью ATL. В нем рассматриваются принципы обработки событий COM и особенности приемника событий с помощью поддержки, предоставляемой библиотекой ATL.

Сведения о том, как запускать события и реализовывать точки подключения, см. в статье [точки подключения ATL](../atl/atl-connection-points.md).
