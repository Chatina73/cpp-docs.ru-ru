# <a name="using-task-manager"></a>С помощью диспетчера задач

Одним из самых простых способов отладки службы является при помощи диспетчера задач. Пока служба запущена, запустите диспетчер задач и нажмите кнопку **процессы** вкладки. Щелкните правой кнопкой мыши имя EXE-файла и нажмите кнопку **Отладка**. Запустится Visual C++, подключенных к этой выполняющемуся процессу. Теперь щелкните **прервать** на **Отладка** меню, чтобы можно было задать точки останова в коде. Нажмите кнопку **запуска** Чтобы выполнить код до выбранной точки останова.
