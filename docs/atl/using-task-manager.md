# <a name="using-task-manager"></a>Использование диспетчера задач

Одним из самых простых способов отладки службы является использование диспетчера задач. Пока служба запущена, запустите диспетчер задач и перейдите на вкладку **процессы** . Щелкните правой кнопкой мыши имя EXE-файла и выберите пункт **Отладка**. Запустится Visual C++, прикрепленный к этому выполняющемуся процессу. Теперь нажмите кнопку **прервать** в меню **Отладка** , чтобы задать точки останова в коде. Нажмите кнопку **выполнить** , чтобы выполнить до выбранных точек останова.
