# <a name="thread-pooling-classes"></a>Классы для группировки потоков

Следующие классы поддерживают объединение потоков в пул:

- [Ккомаутосреадмодуле](../atl/reference/ccomautothreadmodule-class.md) Реализует модуль EXE с поддержкой нескольких апартаментов в пуле потоков.

- [Ккомапартмент](../atl/reference/ccomapartment-class.md) Управляет подразделением в модуле EXE в пуле потоков.

- [Ккомсимплесреадаллокатор](../atl/reference/ccomsimplethreadallocator-class.md) Управляет выбором потоков для модуля EXE.
