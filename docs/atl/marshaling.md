# <a name="marshaling"></a>Маршалинг

Техника кома маршалинга позволяет использовать интерфейсы, подвергающиеся воздействию объекта в одном процессе, в другом процессе. При маршализации COM предоставляет код (или использует код, предоставляемый исполнителем интерфейса), как для упаковки параметров метода в формат, который может быть перемещен через процессы (а также, по всей проволоке к процессам, работающим на других машинах), и для распаковки этих параметров на другом конце. Аналогичным образом, COM должен выполнить те же шаги по возвращению из вызова.

> [!NOTE]
> Маршалинг обычно не требуется, когда интерфейс, предоставляемый объектом, используется в том же процессе, что и объект. Тем не менее, маршалинг может быть необходим между потоками.
