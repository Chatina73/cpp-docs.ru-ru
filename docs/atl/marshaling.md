# <a name="marshaling"></a>Маршалинг

COM-методика упаковки позволяет использовать интерфейсы, предоставляемые объектом в одном процессе, для использования в другом процессе. В процессе упаковки COM предоставляет код (или использует код, предоставляемый интерфейсом реализации интерфейса) для упаковки параметров метода в формат, который можно перемещать между процессами (а также между процессами, работающими на других компьютерах) и распаковать эти параметры на другом конце. Аналогичным образом, COM должен выполнить те же действия для возврата из вызова.

> [!NOTE]
> Маршалирование обычно не требуется, если интерфейс, предоставляемый объектом, используется в том же процессе, что и объект. Однако для потоков может потребоваться маршалирование.
