# <a name="testing-the-atl-dhtml-control"></a>Тестирование элемента управления DHTML ATL

После создания проекта можно выполнить сборку и тестирование примера элемента управления. Перед этим используйте **представление классов** и **Обозреватель решений** для изучения проекта. Элементы проекта более подробно описаны в разделе [Определение элементов проекта элемента управления DHTML](../atl/identifying-the-elements-of-the-dhtml-control-project.md).

## <a name="to-build-and-test-the-atl-dhtml-control"></a>Построение и тестирование элемента управления ATL DHTML

1. Выполните построение проекта. В меню **Сборка** выберите **Построить решение**.

1. По завершении сборки откройте **тестовый контейнер**. Сведения о том, как получить доступ к **тестовому контейнеру**, см. в разделе [Тестирование свойств и событий с помощью тестового контейнера](../mfc/testing-properties-and-events-with-test-container.md) .

1. В **тестовом контейнере** в меню **Правка** выберите пункт **Вставить новый элемент управления**.

1. В диалоговом окне **Вставка элемента управления** выберите элемент управления из списка. Помните, что его имя основано на кратком имени, которое вы указали в мастере элементов управления ATL. Нажмите кнопку **ОК**.

1. Изучите элемент управления. Обратите внимание, что у него есть полоса прокрутки. Используйте дескрипторы элемента управления, чтобы изменить размер элемента управления для активации полосы прокрутки.

1. Проверьте кнопки элемента управления. Цвет фона изменится на цвет, указанный кнопкой.

1. Закройте **тестовый контейнер**.

Затем попытайтесь [изменить элемент управления DHTML](../atl/modifying-the-atl-dhtml-control.md).
