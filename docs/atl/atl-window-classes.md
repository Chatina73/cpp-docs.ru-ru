# <a name="atl-window-classes"></a>Классы окон ATL

ATL включает несколько классов, позволяющих использовать и реализовывать Windows. Эти классы, как и другие классы ATL, предоставляют эффективную реализацию, которая не накладывает никаких издержек на код.

В этом разделе описываются классы окон ATL и объясняется, как их использовать.

[Общие сведения о классах окон ATL](../atl/introduction-to-atl-window-classes.md)<br/>
Кратко описывает каждый класс окна ATL и предоставляет ссылки на справочные материалы по ним.

[Использование окна](../atl/using-a-window.md)<br/>
Описывает использование `CWindow` для управления окном.

[Реализация окна](../atl/implementing-a-window.md)<br/>
Описывает обработчики сообщений, схемы сообщений и использование `CWindowImpl` . Содержит подробные сведения о классе и подклассах.

[Реализация диалогового окна](../atl/implementing-a-dialog-box.md)<br/>
Обсуждаются два метода добавления класса диалогового окна и показан пример кода.

[Использование автономных окон](../atl/using-contained-windows.md)<br/>
Содержит описание содержащихся в библиотеке ATL окон, которые делегируют свои сообщения объекту-контейнеру, а не обрабатывают их в своем собственном классе.

[Основные сведения о характеристиках окна](../atl/understanding-window-traits.md)<br/>
Описание классов признаков использования окон в ATL. Эти классы предоставляют простой способ стандартизации стилей, используемых для создания объекта Window.
