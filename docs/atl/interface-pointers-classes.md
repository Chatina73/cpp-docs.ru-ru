# <a name="interface-pointers-classes"></a>Классы указателей интерфейса

Следующие классы управляют заданным указателем интерфейса:

- [CComPtr](../atl/reference/ccomptr-class.md) Выполняет автоматический подсчет ссылок.

- [CComQIPtr](../atl/reference/ccomqiptr-class.md) Аналогично `CComPtr` , но также выполняет автоматические запросы интерфейсов.

- [Цинтерфацеаррай](../atl/reference/cinterfacearray-class.md) Предоставляет методы, полезные при построении массива указателей на COM-интерфейс.

- [Цинтерфацелист](../atl/reference/cinterfacelist-class.md) Предоставляет методы, полезные при создании списка указателей на COM-интерфейс.

- [Ккомгитптр](../atl/reference/ccomgitptr-class.md) Предоставляет методы для работы с указателями интерфейса и глобальной таблицей интерфейса (GIT).
