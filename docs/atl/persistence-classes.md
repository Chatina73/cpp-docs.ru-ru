# <a name="persistence-classes"></a>Классы сохраняемости

Следующие классы реализуют сохраняемость объектов:

- [Иперсистпропертибагимпл](../atl/reference/ipersistpropertybagimpl-class.md) Позволяет клиенту загружать и сохранять свойства объекта в контейнере свойств.

- [IPersistStreamInitImpl](../atl/reference/ipersiststreaminitimpl-class.md) Позволяет клиенту загружать и сохранять постоянные данные объекта в поток.

- [Иперсистсторажеимпл](../atl/reference/ipersiststorageimpl-class.md) Позволяет клиенту загружать и сохранять постоянные данные объекта в хранилище.
