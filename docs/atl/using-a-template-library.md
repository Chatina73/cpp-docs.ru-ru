# <a name="using-a-template-library"></a>Использование библиотеки шаблонов

Шаблон является несколько похожим на макрос. Как и в случае с макросом, вызов шаблона приводит к его расширению (с подстановкой соответствующего параметра) коду, который вы написали. Однако шаблон становится больше, чтобы разрешить создание новых классов на основе типов, передаваемых в качестве параметров. Эти новые классы реализуют строго типизированные способы выполнения операции, выраженной в коде шаблона.

Библиотеки шаблонов, такие как ATL, отличаются от традиционных библиотек классов C++ тем, что они обычно предоставляются только в качестве исходного кода (или в виде исходного кода с небольшим, поддерживающим временем выполнения) и не являются по сути или являются иерархическими. Вместо создания производного от класса для получения требуемой функциональности можно создать экземпляр класса из шаблона.
