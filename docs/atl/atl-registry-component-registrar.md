# <a name="atl-registry-component-registrar"></a>Компонент реестра ATL (регистратор)

Регистратор ATL предоставляет оптимизированный доступ к системному реестру через пользовательский интерфейс. Регистратор является свободным потоком и позволяет выполнять статическую компоновку кода для клиентов C++.

> [!NOTE]
> Исходный код регистратора ATL можно найти в атлмфк\инклуде\атлифаце.х.
