# <a name="dual-interfaces-and-atl"></a>Сдвоенные интерфейсы и ATL

Сдвоенный интерфейс позволяет обращаться к своим методам как к методам disp-интерфейсов или как к методам vtable. В этом разделе рассматриваются некоторые функции сдвоенных интерфейсов с точки зрения библиотеки ATL.

## <a name="in-this-section"></a>в этом разделе

[Реализация сдвоенного интерфейса](../atl/implementing-a-dual-interface.md)<br/>
Обсуждаются классы и мастера, используемые при реализации сдвоенного интерфейса.

[Несколько сдвоенных интерфейсов](../atl/multiple-dual-interfaces.md)<br/>
Описывает, как предоставить несколько сдвоенных интерфейсов для одного объекта.

[Нерасширяемый атрибут](../atl/nonextensible-attribute.md)<br/>
Описывает, когда следует использовать **нерасширяемый** атрибут в определении интерфейса.

[Сдвоенные интерфейсы и события](../atl/dual-interfaces-and-events.md)<br/>
Описывает причины, по которым создание интерфейса событий не является сдвоенным интерфейсом.
