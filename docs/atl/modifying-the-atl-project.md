# <a name="modifying-the-atl-project"></a>Изменение проекта ATL

На этом этапе проект составного элемента управления реализует необходимые объекты для составного элемента управления. Следующим шагом является добавление элементов управления, которые будут содержаться в составном элементе управления, и обработку всех необходимых событий.

Чтобы добавить дополнительные элементы управления ActiveX или Windows, добавьте новый скрипт ресурсов, а затем используйте редактор диалоговых окон. Дополнительные сведения о добавлении элементов управления (и связанных задач) см. в разделе [Редактор диалоговых окон](../windows/dialog-editor.md).

Сведения о обработке необходимых событий из элементов управления ActiveX см. в разделе [Добавление функциональных возможностей в составной элемент управления](../atl/adding-functionality-to-the-composite-control.md).
