# <a name="building-and-testing-the-atl-project"></a>Сборка и тестирование проекта ATL

Как упоминалось при [вставке составного элемента управления](../atl/inserting-a-composite-control.md), один из первоначальных компонентов проекта представляет собой HTML-страницу по умолчанию, в которой размещен новый составной элемент управления. После завершения изменения составного элемента управления щелкните **построить решение** или **Перестроить решение** в меню **Сборка** . После успешного построения проекта загрузите HTML-страницу, расположенную в корневом каталоге проекта составного элемента управления, в Internet Explorer или в другой браузер и протестируйте функциональные возможности элемента управления.

Вы также можете проверить составной элемент управления с помощью средства тестового контейнера или любого другого приложения, которое может разместить элемент управления ActiveX. Сведения о том, как получить доступ к Контейнеру для тестирования, см. в разделе [Тестирование свойств и событий в Контейнере для тестирования](../mfc/testing-properties-and-events-with-test-container.md) .
