# <a name="site-information-classes"></a>Классы сведений о сайте

Следующие классы позволяют объекту взаимодействовать со своим сайтом:

- [Иобжектвисситеимпл](../atl/reference/iobjectwithsiteimpl-class.md) Извлекает и задает указатель на сайт объекта. Используется для объектов, которые не являются элементами управления.

- [Иолеобжектимпл](../atl/reference/ioleobjectimpl-class.md) Извлекает и задает указатель на сайт объекта. Используется для элементов управления.
