# <a name="inserting-a-composite-control"></a>Вставка составного элемента управления

Диалоговое окно **Добавление класса** позволяет вставить объект ATL в проект. Чтобы открыть это диалоговое окно, щелкните правой кнопкой мыши имя проекта в обозреватель решений, наведите указатель на пункт **Добавить** и выберите пункт **Добавить класс**.

В диалоговом окне **Добавление класса** выберите **элемент управления ATL**. Запустится [Мастер элементов управления ATL](../atl/reference/atl-control-wizard.md). Чтобы создать составной элемент управления, перейдите на вкладку **Параметры** и установите флажок **составной элемент управления** .

Для просмотра элемента управления будет создана HTML-страница по умолчанию.
