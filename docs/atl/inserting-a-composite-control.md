# <a name="inserting-a-composite-control"></a>Вставка составного элемента управления

**Добавление класса** диалоговое окно позволяет вставить объект ATL в проект. Это диалоговое окно можно открыть, щелкнув правой кнопкой мыши имя проекта в обозревателе решений, указывающий на **добавить**и выбрав **Добавление класса**.

В **Добавление класса** диалоговое окно, выберите **управления ATL**. После этого будет запущена [мастер элементов управления ATL](../atl/reference/atl-control-wizard.md). Чтобы создать составной элемент управления, выберите **параметры** , а щелкните **составного элемента управления** "флажок".

Для просмотра элемент управления будет создан на HTML-страницу по умолчанию.
