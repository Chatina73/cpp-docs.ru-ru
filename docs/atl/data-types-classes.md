# <a name="data-types-classes"></a>Классы типов данных

Следующие классы являются оболочкой для типов данных C++:

- [CComBSTR](../atl/reference/ccombstr-class.md) Заключает тип данных в оболочку `BSTR` .

- [CComVariant](../atl/reference/ccomvariant-class.md) Заключает тип данных в оболочку `VARIANT` .

- [Ккомкурренци](../atl/reference/ccomcurrency-class.md) Содержит методы и операторы для создания объекта и управления им `CURRENCY` .
