# <a name="creating-an-atl-dhtml-control"></a>Создание элемента управления DHTML ATL

Мастер элементов управления ATL автоматизирует процесс создания элемента управления DHTML. Он создает необходимые файлы ресурсов, включая HTML-файл с примером кода.

## <a name="to-create-an-atl-dhtml-control"></a>Создание элемента управления ATL DHTML

1. Выполните действия, описанные в разделе [Создание проекта ATL](../atl/reference/creating-an-atl-project.md).

1. В **представление классов** щелкните правой кнопкой мыши узел проекта, наведите указатель на пункт **Добавить** и в контекстном меню выберите пункт **Добавить класс** . В диалоговом окне **Добавление класса** дважды щелкните [Мастер элементов управления ATL](../atl/reference/atl-control-wizard.md). В **мастере элементов управления ATL** перейдите на вкладку **Параметры** и выберите **элемент управления DHTML**.

Теперь можно [протестировать элемент управления по умолчанию](../atl/testing-the-atl-dhtml-control.md).
