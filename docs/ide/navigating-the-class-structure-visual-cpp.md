# <a name="navigate-the-class-structure"></a>Навигация по структуре классов

Вы можете перемещаться по структуре классов проекта с помощью **представления классов** или **обозревателя объектов**. Структура классов проекта может включать функции, наследование и интерфейсы. Дополнительные сведения см. в разделе [Просмотр структуры кода](/visualstudio/ide/viewing-the-structure-of-code).

Чтобы перейти к объявлению функции, дважды щелкните ее имя в представлении классов. Чтобы перейти к определению класса, дважды щелкните имя класса в представлении классов.
