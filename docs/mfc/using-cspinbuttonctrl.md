# <a name="using-cspinbuttonctrl"></a>Использование CSpinButtonCtrl

*Кнопка "Счетчик"* управления (также известный как *вверх вниз* управления) предоставляет пару кнопок со стрелками для корректировки значения пользователем. Это значение называется *текущей позиции*. Позиция остается в пределах диапазона кнопка "Счетчик". Когда пользователь щелкает стрелку вверх, перемещает позицию увеличивается до максимума; и когда пользователь щелкает стрелку вниз, положение минимума.

Представления управления "Счетчик" в MFC, [CSpinButtonCtrl](../mfc/reference/cspinbuttonctrl-class.md) класса.

> [!NOTE]
>  По умолчанию допустимый диапазон для кнопка "Счетчик" имеет максимальное значение ноль (0), а также минимальное, равным 100. Так как максимальное значение меньше минимального значения, нажав кнопку со стрелкой вверх уменьшает позицию и щелкните стрелку вниз увеличивает его. Используйте [CSpinButtonCtrl::SetRange](../mfc/reference/cspinbuttonctrl-class.md#setrange) Чтобы изменить эти значения.

Как правило отображается текущее положение, в сопутствующим элементом управления. Сопутствующим элементом управления называется *связанным окном*. Иллюстрация управления "Счетчик", см. в разделе [о элементы вверх-вниз управления](/windows/desktop/Controls/up-down-controls) в пакете Windows SDK.

Чтобы создать элемент управления "Счетчик" и окно buddy элемента управления редактирования, в Visual Studio, сначала перетащите элемент управления редактированием в диалоговое окно или окно, а затем перетащите элемент управления "Счетчик". Выберите элемент управления "Счетчик" и задайте его **автоматический выбор родственного** и **задать целое для родственного** свойства **True**. Также задайте **выравнивание** свойство; **По правому краю** является наиболее типичных. С этими параметрами поле ввода будет задан в связанном окне так, как он непосредственно предшествующий элемент управления в последовательности табуляции. Элемент управления отображает целые числа и управления "Счетчик" внедряется в правой части элемента управления редактирования. При необходимости можно задать с помощью допустимого диапазона от управления "Счетчик" [CSpinButtonCtrl::SetRange](../mfc/reference/cspinbuttonctrl-class.md#setrange) метод. Обработчики событий необходимы для обмена данными между управления "Счетчик" и связанным окном, так как они напрямую обмениваться данными. При использовании управления "Счетчик", для каких-либо других целей, например, для постраничного просмотра последовательность windows или диалоговые окна, добавьте обработчик для сообщения UDN_DELTAPOS и выполнения вашего настраиваемого действия.
