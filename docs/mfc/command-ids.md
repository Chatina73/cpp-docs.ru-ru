# <a name="command-ids"></a>Идентификаторы команд

Команда полностью описывается его только идентификатор команды (в кодировке **WM_COMMAND** сообщения). Этот идентификатор присваивается объект пользовательского интерфейса, который формирует команды. Как правило идентификаторы именуются для функциональных возможностей объекта пользовательского интерфейса, который им назначены.

Например, элемент "Очистить все" в меню "Правка" могут быть назначены идентификатор например **ID_EDIT_CLEAR_ALL**. Библиотека классов предопределяет некоторые идентификаторы, особенно для команд, что платформа обрабатывает, такие как **ID_EDIT_CLEAR_ALL** или **ID_FILE_OPEN**. Вы создадите другие идентификаторы команд самостоятельно.

При создании собственных меню в Visual C++ редактор меню, это рекомендуется следовать библиотеки классов соглашение об именовании проиллюстрированная **ID_FILE_OPEN**. [Стандартные команды](../mfc/standard-commands.md) описание стандартных команд, определенных в библиотеке классов.
