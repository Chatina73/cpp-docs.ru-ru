# <a name="command-ids"></a>Идентификаторы команд

Команда полностью описывается только ИДЕНТИФИКАТОРом команды (кодируется в **WM_COMMAND** сообщении). Этот идентификатор назначается объекту пользовательского интерфейса, который создает команду. Обычно идентификаторы именуются для функций объекта пользовательского интерфейса, которым они назначены.

Например, элементу очистить все в меню Правка может быть назначен идентификатор, например **ID_EDIT_CLEAR_ALL**. Библиотека классов предопределяет некоторые идентификаторы, особенно для команд, которые платформа обрабатывает самостоятельно, например **ID_EDIT_CLEAR_ALL** или **ID_FILE_OPEN**. Вы создадите другие идентификаторы команд.

При создании собственных меню в редакторе меню Visual C++ рекомендуется следовать соглашению об именовании библиотеки классов, как показано в **ID_FILE_OPEN**. [Стандартные](standard-commands.md) команды объясняют стандартные команды, определенные библиотекой классов.
