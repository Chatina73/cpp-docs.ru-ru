# <a name="using-your-old-toolbars"></a>Использование старых панелей инструментов

Если вы использовали предыдущие версии Visual C++ для создания настраиваемых панелей инструментов, то новая реализация класса [CToolBar](../mfc/reference/ctoolbar-class.md) может привести к проблемам. Так что вам не нужно предоставлять старые панели инструментов для использования новых функций, старая реализация по-прежнему поддерживается.

В примере ДОККТУЛ не используются панели инструментов старого стиля, а только панели инструментов нового стиля.

Изменять панели инструментов старого стиля с помощью редактора ресурсов панели инструментов нельзя.
