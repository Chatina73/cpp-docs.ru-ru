# <a name="creating-the-dialog-resource"></a>Создание ресурса диалогового окна

Для разработки [диалоговое окно](../mfc/dialog-boxes.md) и Создание ресурса диалогового окна, использовать [редактор диалоговых окон](../windows/dialog-editor.md). В редактор диалоговых окон вы можете:

- Измените размер и расположение, будут иметь диалогового окна, когда он появится.

- Перетащите различные типы элементов управления из элементов управления палитры и разместить их в нужное место в диалоговом окне.

- Положение элементов управления с помощью выравнивание кнопок на панели инструментов.

- Проверка диалогового окна, имитируя внешний вид и поведение, она получит в программе. В тестовом режиме можно управлять элементами управления диалогового окна путем ввода текста в текстовых полях, щелкнув кнопок и так далее.

Когда вы закончите, ресурсе шаблона диалогового окна хранится в файле описания ресурсов приложения. Его можно изменить позже при необходимости. Полное описание того, как создавать и редактировать ресурсы диалогового окна, см. в разделе [редактор диалоговых окон](../windows/dialog-editor.md) разделы. Этот метод также используется для создания ресурсов шаблона диалогового окна для [CFormView](../mfc/reference/cformview-class.md) и [CRecordView](../mfc/reference/crecordview-class.md) классы.

Если внешний вид диалогового окна вам подходит, создание класса диалогового окна и сопоставить его сообщения, как описано в [создание класса диалогового окна с помощью мастеров кода](../mfc/creating-a-dialog-class-with-code-wizards.md).
