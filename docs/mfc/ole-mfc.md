# <a name="ole-mfc"></a>OLE (MFC)

Реализация функций OLE в программе влияет на пользовательский интерфейс несколькими способами:

- Визуальное редактирование (встроенная активация) отображает пользовательский интерфейс другой программы в окнах программы и изменяет меню программы на элементы из другой программы.

- Перетаскивание позволяет пользователям перетаскивать объекты в окнах и между окнами и даже между программами.

- Средства отслеживания предоставляют визуальным подсказкам состояние объектов во время визуального редактирования и перетаскивания.
