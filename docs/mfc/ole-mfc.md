# <a name="ole-mfc"></a>OLE (MFC)

Реализация функциональности OLE в программе влияет на пользовательском интерфейсе несколькими способами:

- Визуального редактирования (Активация на месте) отображает пользовательский интерфейс другой программы в windows программы и изменяет программы меню с элементами других приложений.

- Перетаскивание позволяет перетаскивать объекты внутри и между окнами и даже между программами.

- Средства отслеживания предоставляют визуальные подсказки для состояния объектов во время визуального редактирования и перетащите.
