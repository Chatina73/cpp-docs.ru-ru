# <a name="serialization-serialization-vs-database-inputoutput"></a>Сериализация. Сериализация и ввод-вывод баз данных

В этой статье объясняется, когда использовать объекты документов и выполнять сериализацию для файлового ввода-вывода, а также когда подходят другие методы ввода-вывода, поскольку приложение считывает и записывает данные на основе транзакций, как в приложениях баз данных.
