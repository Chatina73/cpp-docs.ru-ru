# <a name="setting-the-images-for-an-individual-item"></a>Установка изображений для отдельного элемента

Различные типы изображений, используемые расширенным элементом поля со списком, определяются значениями в элементах *иимаже*, *иселектедимаже*и *иоверлай* структуры [комбобоксекситем](/windows/win32/api/commctrl/ns-commctrl-comboboxexitemw) . Каждое значение — это индекс изображения в связанном списке изображений элемента управления. По умолчанию для этих членов задано значение 0, в результате чего элемент управления не будет отображать изображение для элемента. Если вы хотите использовать изображения для определенного элемента, можно соответствующим образом изменить структуру при вставке элемента поля со списком или при изменении существующего элемента поля со списком.

## <a name="setting-the-image-for-a-new-item"></a>Задание изображения для нового элемента

При вставке нового элемента инициализируйте элементы структуры *иимаже*, *иселектедимаже*и *иоверлай* с соответствующими значениями, а затем вставьте элемент с вызовом [CComboBoxEx:: InsertItem](../mfc/reference/ccomboboxex-class.md#insertitem).

## <a name="setting-the-image-for-an-existing-item"></a>Задание изображения для существующего элемента

При изменении существующего элемента необходимо работать с элементом *Mask* структуры **комбобоксекситем** .

#### <a name="to-modify-an-existing-item-to-use-images"></a>Изменение существующего элемента для использования изображений

1. Объявите структуру **комбобоксекситем** и задайте для элемента данных *маски* значения, которые нужно изменить.

1. Используя эту структуру, выполните вызов [CComboBoxEx::-Item](../mfc/reference/ccomboboxex-class.md#getitem).

1. Измените члены *Mask*, *иимаже*и *иселектедимаже* вновь возвращенной структуры, используя соответствующие значения.

1. Выполните вызов [CComboBoxEx:: сетитем](../mfc/reference/ccomboboxex-class.md#setitem), передав измененную структуру.
