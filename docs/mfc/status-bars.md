# <a name="status-bars"></a>Строки состояния

Строки состояния предоставляют приложению место для вывода сообщений и полезной информации пользователю, не прерывая работу пользователя. Обычно в нижней части окна строки состояния имеют "области", которые включают "Индикаторы" и "строка сообщения". Индикаторы дают состояние таких объектов, как блокировка прокрутки, включена ли запись макросов и т. д. Строка сообщения в строке состояния может отображать сведения о состоянии программы или о кнопке панели инструментов или пункте меню, на которые пользователь указывает мышью.

Создайте строку состояния в программе, выбрав параметр **Начальная строка состояния** в мастере приложений MFC.
