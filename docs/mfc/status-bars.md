# <a name="status-bars"></a>Строки состояния

Строки состояния предоставить приложению место отображения сообщения и полезные сведения для пользователя, не прерывая работы пользователя. Как правило, отображается в нижней части окна, строки состояния имеют «области», среди которых «индикаторы» и «строка сообщения». Индикаторы отображают состояние, что SCROLL LOCK ли запись макросов включена, и т. д. Строка сообщения, в строке состояния может отображать сведения о состоянии программы или панели инструментов кнопки или меню элемента, который указывает пользователь с помощью мыши.

Создать строку состояния в программе, выбрав **начальной строки состояния** параметр в мастера приложений MFC.
