# <a name="tn049-mfcole-mbcs-to-unicode-translation-layer-mfcans32"></a>TN049. Слой трансляции MFC/OLE MBCS в Юникод (MFCANS32)

В этом заметке изначально описано, как MFCANS32.DLL предоставляет интерфейсы ANSI в основном мире Юникода 32-разрядной технологии OLE. Эта библиотека DLL больше не используется библиотекой MFC.
