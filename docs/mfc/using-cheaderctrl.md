# <a name="using-cheaderctrl"></a>Использование CHeaderCtrl

Используйте элемент управления заголовком, представленный классом [CHeaderCtrl,](../mfc/reference/cheaderctrl-class.md)для отображения заголовков столбцов для списка столбцов. Например, элемент управления заголовками будет полезен для реализации элементов управления столбиками в электронной таблице.

Элемент заголовка обычно делится на части, называемые «элементами заголовка», каждый из которых имеет заголовок для связанного столбца текста или чисел. В зависимости от установленных стилей можно предоставить пользователям несколько прямых способов манипулирования элементами заголовка.

> [!NOTE]
> [CListCtrl](../mfc/reference/clistctrl-class.md) обеспечивает встроенный элемент управления заголовком, а `CListCtrl` [CListView](../mfc/reference/clistview-class.md) инкапсулирует в классе MFC. В общем, подумайте об использовании `CHeaderCtrl` для обозначения списков, которые вы собираетесь нарисовать самостоятельно.
