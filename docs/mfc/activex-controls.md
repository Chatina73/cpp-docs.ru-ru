# <a name="activex-controls"></a>Элементы управления ActiveX

В Visual C++ можно создавать элементы управления ActiveX с помощью MFC или ATL.

>[!IMPORTANT]
> ActiveX — это устаревшая технология, которую не следует использовать для новой разработки. Многие возможности элементов управления ActiveX могут быть выполнены более простым и более безопасным способом с помощью современных технологий, таких как HTML5 и JavaScript, современные расширения браузера или модули веб-сборки. Дополнительные сведения см. [в разделе перерыв в прошлом, часть 2. как и в случае с ActiveX, VBScript, attachEvent](https://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/) и [собственными сообщениями](/microsoft-edge/extensions/guides/native-messaging) и [расширениями Microsoft ребра и веб](/microsoft-edge/extensions) - [сборками](https://webassembly.org/).
