# <a name="activex-controls"></a>Элементы управления ActiveX

В Visual C++ можно создавать элементы управления ActiveX с помощью MFC или ATL.

>[!IMPORTANT]
> ActiveX — это устаревшая технология, которую не следует использовать для новой разработки. Многие возможности элементов управления ActiveX могут быть выполнены более простым и более безопасным способом с помощью современных технологий, таких как HTML5 и JavaScript, современные расширения браузера или модули веб-сборки.
