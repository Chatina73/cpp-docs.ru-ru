Элементы управления "список" ([CListCtrl](../mfc/reference/clistctrl-class.md)) можно прокручивать по умолчанию. Дополнительные сведения см. в разделе [прокрутка](/windows/win32/Controls/using-list-view-controls) в Windows SDK и функция [прокрутки](../mfc/reference/clistctrl-class.md#scroll) .

Дополнительные сведения см. в разделе [Использование элементов управления ListView](/windows/win32/Controls/using-list-view-controls) в Windows SDK и [CListCtrl](../mfc/reference/clistctrl-class.md) Members [упорядочивает](../mfc/reference/clistctrl-class.md#arrange), [сортитемс](../mfc/reference/clistctrl-class.md#sortitems)и [FindItem](../mfc/reference/clistctrl-class.md#finditem).
