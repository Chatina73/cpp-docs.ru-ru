# <a name="scrolling-arranging-sorting-and-finding-in-list-controls"></a>Прокрутка, расположение, сортировка и поиск в элементах управления "Список"

Список элементов управления ([CListCtrl](../mfc/reference/clistctrl-class.md)) являются прокручиваемыми по умолчанию. Дополнительные сведения см. в разделе [позиции прокрутки](/windows/desktop/Controls/using-list-view-controls) в пакете SDK для Windows и [прокрутки](../mfc/reference/clistctrl-class.md#scroll) функция-член.

Вы можете вызвать `CListCtrl` функции-члены для упорядочения элементов списка в элементе управления, отсортировать элементы и найти определенные товары. Дополнительные сведения см. в разделе [с использованием элементов управления ListView](/windows/desktop/Controls/using-list-view-controls) в пакете SDK для Windows и [CListCtrl](../mfc/reference/clistctrl-class.md) члены [расположение](../mfc/reference/clistctrl-class.md#arrange), [SortItems](../mfc/reference/clistctrl-class.md#sortitems), и [FindItem](../mfc/reference/clistctrl-class.md#finditem).
