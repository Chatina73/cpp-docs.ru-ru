# <a name="adding-an-mfc-message-handler"></a>Добавление обработчика сообщения MFC

Можно использовать окно [класса «Мастер»](mfc-class-wizard.md) или окно **Свойств** в **CLass View** для добавления обработчика сообщений (функция члена, обрабатывая сообщения Windows) в класс и отображение сообщений Windows обработчику сообщений. Вы также можете добавить [обработчик событий для любого управления диалоговым полем.](../../windows/adding-event-handlers-for-dialog-box-controls.md)

С помощью окна **Class Wizard** или **Properties** (в **классе View)** для определения функций обработки сообщений и событий можно автоматически обновлять таблицу отправки сообщений (или карту сообщений) и файл заголовка класса.

> [!NOTE]
> Обработчик сообщений можно добавить в класс ATL с помощью класса Wizard; однако некоторые результаты могут варьироваться. Для получения дополнительной информации [Adding an ATL Message Handler](../../atl/adding-an-atl-message-handler.md)см.
