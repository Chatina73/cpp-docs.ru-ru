# <a name="standard-command-and-window-ids"></a>Стандартная команда и идентификаторы окон

Библиотека класса Фонда Майкрософт определяет ряд стандартных идентифицированных команд и окон в Afxres.h. Эти идентифицировать наиболее часто используются в редакторах ресурсов и [мастере класса](mfc-class-wizard.md) для отображения сообщений в функции обработчика. Все стандартные команды имеют **ID_** префикс. Например, при использовании редактора меню обычно привязывается элемент меню File Open к стандартному идентификатору ID_FILE_OPEN команды.

Для большинства стандартных команд код приложения не должен ссылаться на идентификатор команды, потому что`CWinThread`сама `CWinApp` `CView`инфраструктура `CDocument`обрабатывает команды через карты сообщений в своих основных классах фреймворка (, , и так далее).

В дополнение к стандартным итоговым иждивеникам команды определен ряд других стандартных идовинов, которые имеют приставку **AFX_ID.** Эти идентификаторы включают стандартные идентификаторы окон (префикс **AFX_IDW_),** идентификаторы строк (префикс **AFX_IDS_)** и несколько других типов.

Иди, начинаюеся с **AFX_ID** префикса, редко используется программистами, но, возможно, потребуется ссылаться на эти иждивенца, когда преобладающие функции фреймворка также относятся к **AFX_ID**s.

Иди не задокументирован ы индивидуально в этой ссылке. Вы можете найти более подробную информацию о них в технических заметок [20](../../mfc/tn020-id-naming-and-numbering-conventions.md), [21,](../../mfc/tn021-command-and-message-routing.md)и [22](../../mfc/tn022-standard-commands-implementation.md).
