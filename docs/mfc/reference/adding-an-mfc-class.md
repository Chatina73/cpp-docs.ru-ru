# <a name="adding-an-mfc-class"></a>Добавление класса MFC

Чтобы добавить классы, полученные из классов библиотеки Microsoft Foundation Class (MFC), в ваш проект, используйте кнопку **Add Class** в [class Wizard.](mfc-class-wizard.md) Укажите имя нового класса, выберите базовый класс и выберите идентификатор диалогового окна, с которым он связан (если таковые имеется). Мастер создает файл заголовка и файл реализации и добавляет их в проект.

> [!NOTE]
> Вы можете добавить классы MFC в приложение ATL COM, если вы изначально [создали приложение при поддержке MFC.](../../atl/reference/mfc-support-in-atl-projects.md) Вы также можете добавить классы MFC в проекты Win32, которые имеют поддержку MFC.

### <a name="to-add-an-mfc-class-to-your-project"></a>Добавление класса MFC в ваш проект

Из класса View, право нажмите имя проекта. Нажмите **Класс Мастер,** а затем нажмите **Добавить класс,** чтобы открыть [добавить MFC класса](mfc-add-class-wizard.md) диалогового окна. Используйте **Class Wizard** для добавления обработчиков сообщений или новых методов.
