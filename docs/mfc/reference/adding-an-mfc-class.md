# <a name="adding-an-mfc-class"></a>Добавление класса MFC

Чтобы добавить классы, производные от классов библиотеки Microsoft Foundation Class (MFC), в проект используйте кнопку **Добавить класс** в [мастере классов](mfc-class-wizard.md). Укажите имя нового класса, выберите базовый класс и выберите идентификатор диалогового окна, с которым оно связано (при наличии). Мастер создаст файл заголовка и файл реализации и добавит их в проект.

> [!NOTE]
> Классы MFC можно добавить в COM-приложение ATL, если изначально [приложение было создано с поддержкой MFC](../../atl/reference/mfc-support-in-atl-projects.md). Классы MFC также можно добавить в проекты Win32, поддерживающие MFC.

### <a name="to-add-an-mfc-class-to-your-project"></a>Добавление класса MFC в проект

В представление классов щелкните правой кнопкой мыши имя проекта. Щелкните **мастер классов** , а затем нажмите кнопку **Добавить класс** , чтобы открыть диалоговое окно [Добавление класса MFC](mfc-add-class-wizard.md) . Используйте **мастер классов** для добавления обработчиков сообщений или новых методов.
