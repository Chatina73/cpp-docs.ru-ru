# <a name="mapping-messages-to-functions"></a>Сопоставление сообщений с функциями

[Мастер класса](mfc-class-wizard.md) позволяет связывать обработчики сообщений (функции членов классов пользовательского интерфейса MFC) с сообщениями, генерируемыми ресурсами приложения. Они используют [карты сообщений MFC](../../mfc/messages-and-commands-in-the-framework.md) для создания привязки.

При использовании Класса View для создания нового класса, полученного из одного из классов фреймворка, он автоматически помещает полный и функциональный класс в заголовки (.h) и нареализацию (.cpp) файлы, которые вы указываете.

> [!NOTE]
> Чтобы добавить новый класс, который не обрабатывает сообщения, создайте класс непосредственно в текстовом редакторе.

### <a name="to-define-or-remove-a-message-handler-using-the-class-wizard"></a>Определить или удалить обработчик сообщений с помощью мастера класса

1. В **классе View**, правой кнопкой мыши класса.

1. В контекстном меню выберите [класс мастера](mfc-class-wizard.md).
