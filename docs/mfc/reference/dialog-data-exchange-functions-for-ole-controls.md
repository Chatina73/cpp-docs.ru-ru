# <a name="dialog-data-exchange-functions-for-ole-controls"></a>Функции обмена данными диалоговых окон для элементов управления OLE

В этом разделе перечислены функции DDX_OC, используемые для обмена данными между свойством элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления и элемент данных диалоговое окно, представлении формы или объекте представления элемента управления.

### <a name="ddxoc-functions"></a>Функции DDX_OC

|||
|-|-|
|[DDX_OCBool](#ddx_ocbool)|Управляет передачей **BOOL** данных между свойством элемента управления OLE и **BOOL** элемент данных.|
|[DDX_OCBoolRO](#ddx_ocboolro)|Управляет передачей **BOOL** данных между свойством только для чтения элемента управления OLE и **BOOL** элемент данных.|
|[DDX_OCColor](#ddx_occolor)|Управляет передачей **OLE_COLOR** данных между свойством элемента управления OLE и **OLE_COLOR** элемент данных.|
|[DDX_OCColorRO](#ddx_occolorro)|Управляет передачей **OLE_COLOR** данных между свойством только для чтения элемента управления OLE и **OLE_COLOR** элемент данных.|
|[DDX_OCFloat](#ddx_ocfloat)|Управляет передачей **float** (или **двойные**) данные между свойством элемента управления OLE и **float** (или **двойные**) данные-член.|
|[DDX_OCFloatRO](#ddx_ocfloatro)|Управляет передачей **float** (или **двойные**) данные между свойством только для чтения элемента управления OLE и **float** (или **двойные**) данных член.|
|[DDX_OCInt](#ddx_ocint)|Управляет передачей **int** (или **long**) данные между свойством элемента управления OLE и **int** (или **long**) данные-член.|
|[DDX_OCIntRO](#ddx_ocintro)|Управляет передачей **int** (или **long**) данные между свойством только для чтения элемента управления OLE и **int** (или **long**) данные-член.|
|[DDX_OCShort](#ddx_ocshort)|Управляет передачей **короткие** данных между свойством элемента управления OLE и **короткие** элемент данных.|
|[DDX_OCShortRO](#ddx_ocshortro)|Управляет передачей **короткие** данных между свойством только для чтения элемента управления OLE и **короткие** элемент данных.|
|[DDX_OCText](#ddx_octext)|Управляет передачей **CString** данных между свойством элемента управления OLE и **CString** элемент данных.|
|[DDX_OCTextRO](#ddx_octextro)|Управляет передачей **CString** данных между свойством только для чтения элемента управления OLE и **CString** элемент данных.|

##  <a name="ddx_ocbool"></a>  DDX_OCBool

`DDX_OCBool` Функция управляет передачей **BOOL** данных между свойством элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления и **BOOL** членом данных в диалоговом окне, представлении формы или Объект представления элемента управления.

```
void AFXAPI DDX_OCBool(
    CDataExchange* pDX,
    int nIDC,
    DISPID dispid,
    BOOL& value);
```

### <a name="parameters"></a>Параметры

*pDX*<br/>
Указатель на объект `CDataExchange` . Структура предоставляет этот объект для формирования контекста обмена данными, включая его направление.

*nIDC*<br/>
Идентификатор элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления.

*Идентификатор DISPID*<br/>
Идентификатор диспетчера свойства элемента управления.

*значение*<br/>
Ссылка на переменную-член диалогового окна, представления формы или объекта представления элемента управления, с которой осуществляется обмен данными.

### <a name="remarks"></a>Примечания

Дополнительные сведения об DDX см [обмен данными окон и проверка](../../mfc/dialog-data-exchange-and-validation.md).

### <a name="requirements"></a>Требования

  **Заголовок:** afxdisp.h

Функция `DDX_OCTextRO` управляет передачей данных `CString` между доступным только для чтения свойством элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления и элементом данных `CString` диалогового окна, представления формы или объекта представления элемента управления.

```
void AFXAPI DDX_OCTextRO(
    CDataExchange* pDX,
    int nIDC,
    DISPID dispid,
    CString& value);
```

### <a name="parameters"></a>Параметры

*pDX*<br/>
Указатель на объект `CDataExchange` . Структура предоставляет этот объект для формирования контекста обмена данными, включая его направление.

*nIDC*<br/>
Идентификатор элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления.

*Идентификатор DISPID*<br/>
Идентификатор диспетчера свойства элемента управления.

*значение*<br/>
Ссылка на переменную-член диалогового окна, представления формы или объекта представления элемента управления, с которой осуществляется обмен данными.
