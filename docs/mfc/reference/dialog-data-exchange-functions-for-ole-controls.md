# <a name="dialog-data-exchange-functions-for-ole-controls"></a>Функции обмена данными диалоговых окон для элементов управления OLE

В этой теме перечислены функции DDX_OC, используемые для обмена данными между свойством элемента управления OLE в диалоговом поле, представлением формы или объектом представления управления, а также членом поля диалогов, представления формы или объекта управления.

### <a name="ddx_oc-functions"></a>функции DDX_OC

|||
|-|-|
|[DDX_OCBool](#ddx_ocbool)|Управляет передачей данных **BOOL** между свойством управления OLE и членом данных **BOOL.**|
|[DDX_OCBoolRO](#ddx_ocboolro)|Управляет передачей данных **BOOL** между свойством только для чтения элемента управления OLE и членом данных **BOOL.**|
|[DDX_OCColor](#ddx_occolor)|Управляет передачей **данных OLE_COLOR** между свойством управления OLE и членом **данных OLE_COLOR.**|
|[DDX_OCColorRO](#ddx_occolorro)|Управляет передачей **OLE_COLOR** данных между только считываемым свойством управления OLE и **OLE_COLOR** участником данных.|
|[DDX_OCFloat](#ddx_ocfloat)|Управляет передачей данных **поплавка** (или **двойного)** между свойством управления OLE и **поплавком** (или **двойным)** членом данных.|
|[DDX_OCFloatRO](#ddx_ocfloatro)|Управляет передачей данных **поплавка** (или **двойного)** между только считываемым свойством управления OLE и **поплавком** (или **двойным)** членом данных.|
|[DDX_OCInt](#ddx_ocint)|Управляет передачей данных **Int** (или **длинных)** между свойством управления OLE и членом данных **int** (или **long).**|
|[DDX_OCIntRO](#ddx_ocintro)|Управляет передачей данных **Int** (или **длинных)** между свойством только для чтения элемента управления OLE и членом данных **Int** (или **long).**|
|[DDX_OCShort](#ddx_ocshort)|Управляет передачей **коротких** данных между свойством управления OLE и **коротким** участником данных.|
|[DDX_OCShortRO](#ddx_ocshortro)|Управляет передачей **коротких** данных между только считываемым свойством управления OLE и **коротким** участником данных.|
|[DDX_OCText](#ddx_octext)|Управляет передачей данных **CString** между свойством управления OLE и членом данных **CString.**|
|[DDX_OCTextRO](#ddx_octextro)|Управляет передачей данных **CString** между свойством только для чтения элемента управления OLE и членом данных **CString.**|

## <a name="ddx_ocbool"></a><a name="ddx_ocbool"></a>DDX_OCBool

Функция `DDX_OCBool` управляет передачей данных **BOOL** между свойством управления OLE в диалоговом поле, видом формы или объектом представления управления и членом данных **BOOL** в диалоговом поле, представлении формы или объекте управления.

```cpp
void AFXAPI DDX_OCBool(
    CDataExchange* pDX,
    int nIDC,
    DISPID dispid,
    BOOL& value);
```

### <a name="parameters"></a>Параметры

*Pdx*<br/>
Указатель на объект `CDataExchange`. Структура предоставляет этот объект для формирования контекста обмена данными, включая его направление.

*nIDC*<br/>
Идентификатор элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления.

*Dispid*<br/>
Идентификатор диспетчера свойства элемента управления.

*value*<br/>
Ссылка на переменную-член диалогового окна, представления формы или объекта представления элемента управления, с которой осуществляется обмен данными.

### <a name="remarks"></a>Remarks

Дополнительные сведения об DDX см [обмен данными окон и проверка](../../mfc/dialog-data-exchange-and-validation.md).

### <a name="requirements"></a>Требования

  **Заголовок:** afxdisp.h

Указатель на объект `CDataExchange`. Структура предоставляет этот объект для формирования контекста обмена данными, включая его направление.

*nIDC*<br/>
Идентификатор элемента управления OLE в диалоговом окне, представлении формы или объекте представления элемента управления.

*Dispid*<br/>
Идентификатор диспетчера свойства элемента управления.

*value*<br/>
Ссылка на переменную-член диалогового окна, представления формы или объекта представления элемента управления, с которой осуществляется обмен данными.
