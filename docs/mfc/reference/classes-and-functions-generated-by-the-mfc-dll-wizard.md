# <a name="classes-and-functions-generated-by-the-mfc-dll-wizard"></a>Классы и функции, создаваемые мастером MFC DLL

Код, создаваемый мастером MFC DLL, зависит от типа создаваемой библиотеки DLL и выбранных параметров. Мастер DLL MFC создает один и тот же код для обеих форм обычных библиотек DLL MFC.

|Разновидность библиотеки DLL|Параметр|Классы|Функции|
|-----------------|------------|-------------|---------------|
|[Расширение](../../build/extension-dlls-overview.md)|Нет|Нет|`DllMain`|
|[Обычный](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Нет|Класс приложения, производный от `CWinApp`|Нет|
|[Обычный](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Автоматизация|Класс приложения, производный от `CWinApp`|`DllGetClassObject` `DllCanUnloadNow` `DllRegisterServer`|
|[Расширение](../../build/extension-dlls-overview.md)|Сокеты окон|Нет|`DllMain`|
|[Обычный](../../build/regular-dlls-dynamically-linked-to-mfc.md)|Сокеты окон|Класс приложения, производный от `CWinApp`|`InitInstance` содержит вызов `AfxSocketInit`|
