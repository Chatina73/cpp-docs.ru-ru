# <a name="declaring-a-variable-based-on-your-new-control-class"></a>Объявление переменной на основании нового класса элемента управления

После создания класса управления MFC можно объявить переменную на его основе. Чтобы создать контекст для новой переменной, необходимо открыть редактор диалогов оговора и отредкить диалоговую будку, в которой требуется использовать многоразовый элемент управления. Кроме того, в диалоговом поле уже должен быть связанный с ним класс. Для получения информации об использовании редактора диалогов [см.](../../windows/dialog-editor.md)

### <a name="to-declare-a-variable-based-on-your-reusable-class"></a>Объявление переменной на основе вашего многоразового класса

1. При редактировании диалогового окна перетащите элемент управления того же типа, что и базовый класс нового элемента управления, из панели инструментов Управления на поле диалога.

1. Поместите указатель мыши над упавным элементом управления.

1. При нажатии на клавишу CTRL, дважды щелкните элемент управления.

   Отображается [диалоговое](../../ide/add-member-variable-wizard.md) окно Add Member Variable.

1. В поле **доступа** выберите правильный доступ для элемента управления.

1. Нажмите на переменную флажок **Control.**

1. В поле **переменного имени** введите имя.

1. Под **категорией,** нажмите **Контроль**.

1. В списке **идентификатора управления** выберите элемент управления, который вы добавили. Список **переменных типов** должен отображать правильный переменный тип, а поле **типа управления** должно отображать правильный тип управления.

1. В поле **Комментарий** добавьте любой комментарий, который вы хотите отойти в коде.

1. Нажмите кнопку **ОК**.
