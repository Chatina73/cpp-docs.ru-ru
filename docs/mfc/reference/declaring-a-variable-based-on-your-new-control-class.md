# <a name="declaring-a-variable-based-on-your-new-control-class"></a>Объявление переменной на основании нового класса элемента управления

После создания класса элемента управления MFC можно объявить переменную на ее основе. Чтобы предоставить контекст для новой переменной, необходимо открыть редактор диалоговых окон и изменить диалоговое окно, в котором будет использоваться элемент управления для повторного использования. Кроме того, диалоговое окно уже должно иметь связанный с ним класс. Сведения об использовании редактора диалоговых окон см. в разделе [Редактор диалоговых окон](../../windows/dialog-editor.md).

### <a name="to-declare-a-variable-based-on-your-reusable-class"></a>Объявление переменной на основе класса, используемого для повторного использования

1. При редактировании диалогового окна перетащите элемент управления того же типа, что и базовый класс нового элемента управления, с панели инструментов элементы управления на диалоговое окно.

1. Поместите указатель мыши на удаленный элемент управления.

1. Удерживая нажатой клавишу CTRL, дважды щелкните элемент управления.

   Откроется диалоговое окно [Добавление переменной члена](../../ide/adding-a-member-variable-visual-cpp.md#add-member-variable-wizard) .

1. В поле **доступ** выберите правильный доступ для элемента управления.

1. Установите флажок **Управляющая переменная** .

1. В поле **имя переменной** введите имя.

1. В разделе **Категория** выберите **элемент Управление**.

1. В списке **идентификатор элемента управления** выберите добавленный элемент управления. В списке **тип переменной** должен отображаться правильный тип переменной, а в поле **тип элемента управления** должен отображаться правильный тип элемента управления.

1. В поле **Комментарий** добавьте любой комментарий, который должен отображаться в коде.

1. Нажмите кнопку **ОК**.
