# <a name="chtmleditctrlbase-class"></a>Класс CHtmlEditCtrlBase

Представляет компонент редактирования HTML.

## <a name="syntax"></a>Синтаксис

```
template <class T> class CHtmlEditCtrlBase
```

##  <a name="addtoglyphtable"></a>  CHtmlEditCtrlBase::AddToGlyphTable

Добавляет запись в таблицу глифа, который задает изображения, отображаемые для конкретных тегов в режиме конструктора.

```
HRESULT AddToGlyphTable(
    LPCTSTR szTag,
    LPCTSTR szImgUrl,
    unsigned short nTagType,
    unsigned short nAlignment,
    unsigned short nPosInfo,
    unsigned short nDirection,
    unsigned int nImgWidth,
    unsigned int nImgHeight) const;
```

### <a name="parameters"></a>Параметры

*szTag*<br/>
Имя тега (например, «P» или «таблица»).

*szImgUrl*<br/>
URL-адрес изображения.

*nTagType*<br/>
Тип тега: 0 означает, что образ предназначен для только открывающего тега. 1 означает, что образ предназначен для только закрывающий тег. 2 означает, что в образе для открывающих и закрывающих тегов. Единый теги, например br и комментарий должно быть добавлено с тип тега, равным 0.

*nAlignment*<br/>
Выравнивание (только прямоугольную элементов): Этот параметр определяет, что изображения для элемента с атрибут выравнивания. Слева = 0, center = 1, right = 2 и не определено = 3. Слева, справа или center атрибуты должны задаваться явным образом в элементе.

*nPosInfo*<br/>
Данные размещения. Определяет, какие каскадные таблицы стилей (CSS) позиционирования значение применяется глифа, где статический позиционирования = 0, абсолютное позиционирование = 1, относительное позиционирование = 2 и все = 3. Это поле позволяет указать один глиф для тега, если он не позиционируется и другой глиф для отображения опорную точку, при помещении тега.

*nDirection*<br/>
Направление. Этот параметр указывает образ для тега, на основе порядка чтения для текущего языка. значение 0 указывает, слева направо, значение 1 указывает, справа налево, 2 означает сверху вниз, 3 указывает снизу вверх и 4 указывает все. Обычно это поле равным 4.

*nImgWidth*<br/>
Ширина изображения в пикселях.

*nImgHeight*<br/>
Высота изображения в пикселях.

### <a name="return-value"></a>Возвращаемое значение

Возвращает S_OK в случае успеха или ошибки HRESULT в случае сбоя.

### <a name="remarks"></a>Примечания

Дополнительные сведения о параметрах см. в разделе «Формат строки таблицы глиф» в [с помощью редактирования глифы](/previous-versions/windows/internet-explorer/ie-developer/platform-apis/aa969614\(v=vs.85\)).

Этот метод отправляет [идентификатор команды IDM_ADDTOGLYPHTABLE](/previous-versions/aa769891\(v=vs.85\)) в элемент управления WebBrowser.
