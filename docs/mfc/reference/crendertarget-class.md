## <a name="crendertargetcreatecompatiblerendertarget"></a><a name="createcompatiblerendertarget"></a>CRenderTarget::СоздатьCompatibleRenderTarget

Создает новую цель визуализации биткарты для использования во время промежуточного внеземного чертежа, совместимого с текущей целью рендеринга.

```
BOOL CreateCompatibleRenderTarget(
    CBitmapRenderTarget& bitmapTarget,
    CD2DSizeF sizeDesired = CD2DSizeF(0., 0.),
    CD2DSizeU sizePixelDesired = CD2DSizeU(0, 0),
    D2D1_PIXEL_FORMAT* desiredFormat = NULL,
    D2D1_COMPATIBLE_RENDER_TARGET_OPTIONS options = D2D1_COMPATIBLE_RENDER_TARGET_OPTIONS_NONE);
```

### <a name="parameters"></a>Параметры

*bitmapTarget*<br/>
При возврате этого метода содержится адрес указателя на новую цель визуализации битовой карты. Этот параметр передается без инициализации.

*размерНежелательный*<br/>
Необходимый размер новой цели рендеринга в независях от устройства пикселей, если он должен отличаться от первоначальной цели рендеринга, или NULL. Дополнительные сведения см. в разделе «Примечания».

*размерPixelDesired*<br/>
Необходимый размер новой цели рендеринга в пикселях, если он должен отличаться от первоначальной цели рендеринга, или NULL. Дополнительные сведения см. в разделе «Примечания».

*желаемыйФормат*<br/>
Необходимый формат пикселя и альфа-режим новой цели рендеринга, или NULL. Если формат пикселя установлен на DXGI_FORMAT_UNKNOWN или если этот параметр недействителен, новая цель рендера использует тот же формат пикселя, что и исходная цель рендеринга. Если альфа-режим находится D2D1_ALPHA_MODE_UNKNOWN или этот параметр null, альфа-режим новой цели внебрачно-юмерии по умолчанию D2D1_ALPHA_MODE_PREMULTIPLIED. Для получения информации о поддерживаемых форматах пикселей см.

*Параметры*<br/>
Значение, означавшее, должна ли новая цель рендеринга быть совместимой с GDI.

### <a name="return-value"></a>Возвращаемое значение

Если метод удается, он возвращает TRUE. В противном случае, он возвращает FALSE.

## <a name="crendertargetdrawbitmap"></a><a name="drawbitmap"></a>CRenderTarget::DrawBitmap

Рисует отформатированный текст, описанный указанным объектом IDWriteTextLayout.

```
void DrawBitmap(
    CD2DBitmap* pBitmap,
    const CD2DRectF& rectDest,
    float fOpacity = 1.0,
    D2D1_BITMAP_INTERPOLATION_MODE interpolationMode = D2D1_BITMAP_INTERPOLATION_MODE_LINEAR,
    const CD2DRectF* pRectSrc = NULL);
```

### <a name="parameters"></a>Параметры

*pBitmap*<br/>
Бит-карта для визуализации.

*rectDest*<br/>
Размер и положение в независях от устройства пикселей в пространстве координат цели визы, области, к которой обращается бит-карта. Если прямоугольник не упорядочен, ничего не нарисовано, но цель рендера не входит в состояние ошибки.

*fOpacity*<br/>
Значение между 0.0f и 1.0f, включительно, которое определяет значение непрозрачности для того чтобы примениться к bitmap; это значение умножается на альфа-значения содержимого биткарты.

*интерполяцияMode*<br/>
Режим интерполяции для использования, если битная карта масштабируется или поворачивается операцией чертежа.

*pRectSrc*<br/>
Размер и положение в независях от устройства пикселей в пространстве координат биткарты, области в пределах битной карты для рисования.

*pBrush*<br/>
