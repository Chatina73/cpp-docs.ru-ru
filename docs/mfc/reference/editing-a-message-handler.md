# <a name="editing-a-message-handler"></a>Редактирование обработчика сообщений

После определения обработчика сообщений можно переходить к определению функции-члена, чтобы добавить или изменить код.

Чтобы перейти к определению функции-члена из [редактора диалоговых окон](../../windows/dialog-editor.md), дважды щелкните элемент управления, для которого уже определен обработчик. Будет выполнен переход к файлу, в котором определен обработчик сообщений выбранного элемента управления.

Чтобы перейти к определению функции-члена из [представление классов](/visualstudio/ide/viewing-the-structure-of-code), дважды щелкните имя функции в представление классов.
