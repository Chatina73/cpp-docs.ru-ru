# <a name="adding-an-mfc-odbc-consumer"></a>Добавление потребителя ODBC MFC

Потребитель ODBC MFC состоит из класса набора записей ODBC и привязок данных, необходимых для доступа к источнику данных.

### <a name="to-add-an-mfc-odbc-consumer"></a>Добавление потребителя ODBC MFC

1. В разделе Представление классов щелкните проект правой кнопкой мыши. В контекстном меню выберите команду **Добавить**, а затем — **Добавить класс**.

1. В папке Visual C++ дважды щелкните значок **потребителя ODBC MFC** или выберите его и нажмите кнопку **Открыть**.

   Откроется мастер потребителей MFC ODBC.

1. Определите параметры, как описано в [мастере потребителей ODBC MFC](../../mfc/reference/mfc-odbc-consumer-wizard.md).

1. Нажмите кнопку **Готово** , чтобы закрыть мастер, который вставит в проект созданный код потребителя ODBC.
