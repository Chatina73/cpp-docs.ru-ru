# <a name="using-image-lists-with-header-controls"></a>Использование списков изображений с элементами управления "Заголовок"

Элементы заголовка могут отображать изображение в элементе заголовка. Этот образ, хранящийся в связанном списке изображений, имеет размер 16 x 16 пикселей и имеет те же характеристики, что и изображения значков, используемые в элементе управления "представление списка". Чтобы реализовать такое поведение успешно, необходимо сначала создать и инициализировать список изображений, связать список с элементом управления "заголовок", а затем изменить атрибуты элемента заголовка, который будет отображать изображение.

Следующая процедура иллюстрирует сведения, используя указатель на элемент управления заголовка ( `m_pHdrCtrl` ) и указатель на список изображений ( `m_pHdrImages` ).

Например, в стандартном элементе управления "список" может быть список изображений (16 x 16 пикселей), используемый как небольшое представление элемента управления представления списка, так и элементы заголовка элемента управления "представление списка".
