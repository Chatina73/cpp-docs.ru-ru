# <a name="settings-for-the-toolbar-control"></a>Параметры панели инструментов

Кнопки на панели инструментов могут отображать точечные рисунки, строки или и то, и другое. По умолчанию размер изображения устанавливается равным 16 на 15 пикселей. Все кнопки имеют одинаковую ширину, по умолчанию 24 – 22 пикселя. Высота панели инструментов определяется высотой кнопок, а ширина панели инструментов равна ширине клиентской области родительского окна, также по умолчанию.

Панель инструментов может иметь встроенные функции настройки, включая диалоговое окно настройки, которое позволяет пользователю вставлять, удалять или переупорядочивать кнопки на панели инструментов. Приложение определяет, доступны ли функции настройки пользователю, и определяет область, в которой пользователь может настраивать панель инструментов. Дополнительные сведения о настройке панели инструментов см. в разделе Class [CToolBarCtrl](../mfc/reference/ctoolbarctrl-class.md) в *справочнике по MFC*.
