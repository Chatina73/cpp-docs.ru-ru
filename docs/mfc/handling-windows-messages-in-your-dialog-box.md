# <a name="handling-windows-messages-in-your-dialog-box"></a>Обработка сообщений Windows для диалогового окна

Диалоговые окна — это окна, поэтому они могут работать с сообщениями Windows, если вы предоставляете соответствующие функции обработчика. При создании класса диалогового окна с помощью [мастера классов](reference/mfc-class-wizard.md)мастер добавляет пустую схему сообщения в класс. Используйте мастер для отображения сообщений или команд Windows, которые должен обработать ваш класс.

Дополнительные сведения см. [в разделе Сопоставление сообщений Windows с классом диалогового окна](mapping-windows-messages-to-your-class.md) .
