# <a name="ready-to-use-list-classes"></a>Готовые классы списков

Следующие классы являются готовыми к использованию классами списков.

[CObList](../mfc/reference/coblist-class.md)<br/>
Хранит указатели на объекты класса `CObject` или на объекты классов, производных от `CObject` , в связанном списке.

[CPtrList](../mfc/reference/cptrlist-class.md)<br/>
Сохраняет указатели на **`void`** (универсальные указатели) в связанном списке.

[CStringList](../mfc/reference/cstringlist-class.md)<br/>
Сохраняет `CString` объекты в связанном списке.
