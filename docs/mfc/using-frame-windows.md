# <a name="using-frame-windows"></a>Использование окон фрейма

Платформа MFC создает окна фрейма документа, а также их представления и документы — в рамках реализации команд New и Open в меню файл. Поскольку платформа выполняет большую часть окна кадров, вы воспроизводите только небольшую роль в создании, использовании и уничтожении этих окон. Однако вы можете явно создавать собственные окна фрейма и дочерние окна для особых целей.
