# <a name="orchestrating-other-window-actions"></a>Оркестрация других действий окна

Окна фрейма организует полумодальные состояния, например контекстную справку и предварительный просмотр печати. Описание роли окна фрейма в режиме предварительного просмотра, см. в разделе [печать и предварительный](../mfc/printing-and-print-preview.md).
