# <a name="orchestrating-other-window-actions"></a>Оркестрация других действий окна

Окно фрейма управляет состояниями семимодал, такими как контекстно-зависимая справка и предварительный просмотр печати. Описание роли фрейма в окне предварительного просмотра см. в разделе [Печать и предварительный просмотр печати](printing-and-print-preview.md).
