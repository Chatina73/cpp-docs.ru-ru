# <a name="ole-db-classes"></a>Классы OLE DB

Поддержка OLE DB в MFC состоит в данный момент класса [COLEDBRecordView](../mfc/reference/coledbrecordview-class.md). `COleDBRecordView` Отображает записи базы данных в элементах управления, через представление формы, который напрямую подключен к [CRowset](../data/oledb/crowset-class.md) объекта. Дополнительные сведения о шаблонах потребителей OLE DB, см. в разделе [списка из шаблонов потребителей OLE DB](../data/oledb/ole-db-consumer-templates-reference.md).
