# <a name="ole-db-classes"></a>Классы OLE DB

Поддержка OLE DB в MFC в настоящее время состоит из класса [коледбрекордвиев](reference/coledbrecordview-class.md). `COleDBRecordView` Отображает записи базы данных в элементах управления с помощью представления формы, непосредственно подключенного к объекту [CRowset](../data/oledb/crowset-class.md) . Дополнительные сведения о шаблонах потребителей OLE DB см. в разделе [список шаблонов потребителей OLE DB](../data/oledb/ole-db-consumer-templates-reference.md).
