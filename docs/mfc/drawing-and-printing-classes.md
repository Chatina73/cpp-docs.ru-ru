# <a name="drawing-and-printing-classes"></a>Классы рисования и печати

В Windows все графические выходные данные рисуются в виртуальной области рисования, называемой контекстом устройства (DC). MFC предоставляет классы для инкапсуляции различных типов контроллеров домена, а также инкапсуляции для средств рисования Windows, таких как точечные рисунки, кисти, палитры и перья.
