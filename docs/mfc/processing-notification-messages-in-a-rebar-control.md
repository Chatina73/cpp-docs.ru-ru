# <a name="processing-notification-messages-in-a-rebar-control"></a>Обработка уведомляющих сообщений в элементах управления главной панели

В родительском классе элемента управления главной панели создайте `OnChildNotify` функцию обработчика с оператором Switch для всех сообщений уведомлений управления главной панелью (`CReBarCtrl`), которые необходимо обменять. Уведомления отправляются в родительское окно, когда пользователь перетаскивает объекты по элементу управления главной панели, изменяет макет полос главной панели, удаляет полосы из элемента управления главной панели и т. д.

Объект элемента управления "Главная панель" может отправлять следующие сообщения уведомления:

- RBN_AUTOSIZE, отправленный элементом управления главной панели (созданным с помощью стиля RBS_AUTOSIZE), когда Главная панель автоматически изменяет размер.

- RBN_BEGINDRAG, отправленный элементом управления главной панели, когда пользователь начинает перетаскивать полосу.

- RBN_CHILDSIZE, отправленный элементом управления главной панели при изменении размера дочернего окна полосы.

- RBN_DELETEDBAND, отправленный элементом управления главной панели после удаления полосы.

- RBN_DELETINGBAND, отправленный элементом управления главной панели при удалении диапазона.

- RBN_ENDDRAG, отправленный элементом управления главной панели, когда пользователь перестает перетаскивать полосу.

- RBN_GETOBJECT, отправленный элементом управления главной панели (созданным с помощью стиля RBS_REGISTERDROP) при перетаскивании объекта на полосу в элементе управления.

- RBN_HEIGHTCHANGE, отправленный элементом управления главной панели при изменении его высоты.

- RBN_LAYOUTCHANGED отправляется элементом управления "Главная панель", когда пользователь изменяет макет полос элемента управления.

Дополнительные сведения об этих уведомлениях см. в разделе [Справочник по элементу управления "Главная панель](/windows/win32/controls/rebar-control-reference) " в Windows SDK.
