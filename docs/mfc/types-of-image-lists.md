# <a name="types-of-image-lists"></a>Типы списков изображений

Существует два типа списков изображений ([CImageList](../mfc/reference/cimagelist-class.md)): nonmasked и маской. «Список nonmasked образа» состоит из цветовой битовой карты, содержащий один или несколько образов. «Список скрытого изображения» состоит из двух рисунков одинакового размера. Во-первых, цветовой битовой карты с изображениями, а второй — монохромную битовую карту, содержащий масок — один для каждого изображения в первом растрового изображения.

Одной из перегрузок `Create` флаг, указывающий, скрыт ли список изображений принимает функцию-член. (Другие перегрузки Создание списков изображений с маской).

При рисовании nonmasked изображения, просто он копируется в целевой контекст устройства; то есть он нарисован поверх цвет фона контекста устройства. При рисовании скрытого изображения, биты изображения, объединяются с биты маски, обычно создания прозрачные области в битовой карте, где показан цвет фона целевой контекст устройства через. При рисовании скрытого изображения, можно указать несколько стилей рисования. Например можно указать, что изображение быть сглажен для указания выбранного объекта.
