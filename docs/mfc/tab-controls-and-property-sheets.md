# <a name="tab-controls-and-property-sheets"></a>Элементы управления "Вкладка" и вкладки свойств

Вкладки свойств — это многостраничные диалоговые окна или "диалоговые окна с вкладками", которые могут отображать до 24 ресурсов шаблона диалогового окна для пользователя. Примеры вкладок свойств см. в диалоговом окне Свойства экрана Windows или в следующем образце приложения MFC:

[CMNCTRL1: демонстрируются общие классы элементов управления, часть 1](../overview/visual-cpp-samples.md)

[CMNCTRL2: демонстрируются общие классы элементов управления, часть 2](../overview/visual-cpp-samples.md)

Вкладки свойств можно легко реализовать с помощью класса MFC [CPropertySheet](../mfc/reference/cpropertysheet-class.md).
