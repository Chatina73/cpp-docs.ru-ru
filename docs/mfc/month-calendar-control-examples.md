# <a name="month-calendar-control-examples"></a>Примеры элемента управления "Календарь на месяц"

Элемент управления, находящийся на отдельной вкладке в образце, демонстрирует основные функциональные возможности и позволяет пользователю динамически изменять определенные атрибуты.
