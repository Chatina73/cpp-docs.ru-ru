# <a name="month-calendar-control-examples"></a>Примеры элемента управления "Календарь на месяц"

Пример приложения [CMNCTRL1](../overview/visual-cpp-samples.md) демонстрирует различные атрибуты `CMonthCalCtrl` класса. Элемент управления, находящийся на отдельной вкладке в образце, демонстрирует основные функциональные возможности и позволяет пользователю динамически изменять определенные атрибуты.
