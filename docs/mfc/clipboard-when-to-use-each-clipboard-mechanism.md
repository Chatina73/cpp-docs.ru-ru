# <a name="clipboard-when-to-use-each-clipboard-mechanism"></a>Буфер обмена. Использование механизма буфера обмена

Придерживайтесь следующих рекомендаций в использование буфера обмена.

- Использование механизма буфера обмена OLE для включения новых возможностей в будущем. Хотя стандартный API буфера обмена сохраняются, механизма OLE — это будущее передачи данных.

- Используйте механизма буфера обмена OLE, если вы создаете приложение, OLE, или нужно, чтобы функции OLE, такие как перетаскивание.

- Использование механизма буфера обмена OLE, если вы предоставляете OLE-форматы.
