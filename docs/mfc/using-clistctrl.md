# <a name="using-clistctrl"></a>Использование CListCtrl

Используйте элемент управления списком для отображения любого расположения иконок с метками, как в File Explorer, или столбцовых списков текста, с иконками или без нее. Для описания четырех возможных "взглядов" (не путать с представлениями MFC) вы можете иметь в элементе управления списком - представление значка, малый вид значка, представление списка и представление отчета - см. Просмотры в обзоре класса [CListCtrl.](../mfc/reference/clistctrl-class.md)

В некоторых представлениях пользователи могут перетаскивать значки в разные позиции или отображать значки. Например, см. правое стекло в Файл Енисеие, которое использует элемент управления списком в окне nondialog. Вы можете поэкспериментировать с доступными представлениями в меню Explorer's View.

Для получения соответствующей информации [см.](/windows/win32/Controls/list-view-controls-overview)

> [!NOTE]
> SDK Windows называет элементы управления списками «управления представлениями списка». Это использование "вид" не относится к mFC типа мстяжа худощавого представления, в частности. `CListView` Для получения дополнительной информации [см.](../mfc/list-control-and-list-view.md)
