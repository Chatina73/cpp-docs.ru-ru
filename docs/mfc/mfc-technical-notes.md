# <a name="mfc-technical-notes"></a>Технические примечания MFC

Техническое примечание — это документ, написанный для программистов программистами.

Каждое техническое примечание описывает проблему или функцию, которая выходит за рамки документации по MFC. Эти технические заметки отражать запросы информации от пользователей, а также специализированные сведения, которые разработчики MFC предложит опытным пользователям.

Просмотреть технические примечания можно двумя способами:

- [Технические примечания по номеру](technical-notes-by-number.md)

- [Технические примечания по категориям](technical-notes-by-category.md)
