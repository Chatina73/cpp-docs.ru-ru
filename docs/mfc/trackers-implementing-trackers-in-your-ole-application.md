# <a name="trackers-implementing-trackers-in-your-ole-application"></a>Средства отслеживания. Реализация средств отслеживания в приложении OLE

Средства отслеживания предоставляют графический интерфейс, чтобы пользователи могли взаимодействовать с элементами клиента OLE. С помощью различных tracker стилей OLE клиентские элементы, которые могут отображаться со штриховой границей, изменение размера дескрипторы и множество других визуальных эффектов. - [Эластичные соединения и средства отслеживания](../mfc/rubber-banding-and-trackers.md).
