# <a name="trackers-implementing-trackers-in-your-ole-application"></a>Средства отслеживания. Реализация средств отслеживания в приложении OLE

Средства мониторинга предоставляют графический интерфейс, позволяющий пользователям взаимодействовать с элементами OLE-клиентов. Используя различные стили Tracker, элементы OLE-клиента могут отображаться с помощью штриховых границ, маркеров изменения размера или различных других визуальных эффектов. В этой статье рассматриваются следующие вопросы:

- [Как реализовать отслеживание в коде](../mfc/how-to-implement-tracking-in-your-code.md).

- [Резиновые полосы и средства мониторинга](../mfc/rubber-banding-and-trackers.md).

В статье также рассматривается использование стилей с датчиками. Кроме того, он делает несколько ссылок на пример [OCLIENT](../overview/visual-cpp-samples.md)в MFC OLE.
