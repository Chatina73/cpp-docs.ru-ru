# <a name="using-slider-controls"></a>Использование ползунков

Типичное использование элемента управления Slider соответствует шаблону ниже:

- Элемент управления создан. Если элемент управления указан в шаблоне диалогового окна, при создании диалогового окна Создание выполняется автоматически. (В классе диалогового окна должен быть член [CSliderCtrl](../mfc/reference/csliderctrl-class.md) , соответствующий элементу управления Slider.) Кроме того, можно использовать функцию [создания](../mfc/reference/csliderctrl-class.md#create) члена, чтобы создать элемент управления как дочернее окно любого окна.

- Вызовите различные функции члена множества, чтобы задать значения для элемента управления. Изменения, которые можно внести, включают задание минимальных и максимальных позиций для ползунка, рисование делений, задание диапазона выбора и изменение положения ползунка. Для элементов управления в диалоговом окне хорошее время это делается в функции [онинитдиалог](../mfc/reference/cdialog-class.md#oninitdialog) диалогового окна.

- По мере того как пользователь взаимодействует с элементом управления, он будет отсылать различные сообщения с уведомлениями. Можно извлечь значение ползунка из элемента управления, вызвав функцию члена [жетпос](../mfc/reference/csliderctrl-class.md#getpos) .

- После завершения работы с элементом управления необходимо убедиться, что он правильно уничтожен. Если элемент управления "ползунок" находится в диалоговом окне, он и `CSliderCtrl` объект будут уничтожены автоматически. В противном случае необходимо убедиться, что элемент управления и `CSliderCtrl` объект должным образом уничтожены.
