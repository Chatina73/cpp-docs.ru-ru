# <a name="containers-for-activex-controls"></a>Контейнеры для элементов управления ActiveX

Элементы управления ActiveX, разработанные в Visual C++, можно использовать в других приложениях, если они поддерживают включение элементов управления ActiveX. Некоторые приложения Майкрософт, начиная с перечисленных версий, поддерживают включение элементов управления ActiveX.

>[!IMPORTANT]
> ActiveX — это устаревшая технология, которую не следует использовать для новой разработки. Дополнительные сведения о современных технологиях, которые заменяют ActiveX, см. в разделе [элементы управления ActiveX](activex-controls.md).

> [!NOTE]
> Следующий список не является полным списком приложений, которые поддерживают элементы управления ActiveX, но представляют набор, который чаще всего используется при тестировании:

Приложения, поддерживающие элемент управления ActiveX, включают:

- Internet Explorer 3. x и более поздние версии

- Visual Basic 4. x и выше

- Visual C++ 4. x и выше

- Доступ 95 и выше

- Excel 97 и более поздние версии

- Word 97 и более поздние версии

- Доступ 97 и выше

- FrontPage 97 и более поздние версии

- PowerPoint 97 и более поздние версии

- Visual InterDev 97 и более поздние версии

Ниже перечислены приложения сторонних производителей, поддерживающие включение элементов управления ActiveX:

- PowerBuilder

- Delphi

- Построитель C++

- Подключаемый модуль Нкомпасс для Netscape Navigator
