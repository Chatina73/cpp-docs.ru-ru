# <a name="containers-for-activex-controls"></a>Контейнеры для элементов управления ActiveX

Вы можете использовать элементы управления ActiveX, разработанные в Visual C, в других приложениях, если они поддерживают сдерживание управления ActiveX. Ряд приложений корпорации Майкрософт, начиная с перечисленных версий, поддерживают сдерживание управления ActiveX.

>[!IMPORTANT]
> ActiveX является устаревшей технологией, которая не должна использоваться для новых разработок. Для получения дополнительной информации о современных технологиях, которые заменяли ActiveX, [см.](activex-controls.md)

> [!NOTE]
> Следующий список не является полным списком приложений, которые поддерживают элементы управления ActiveX, но представляет набор, используемый чаще всего в тестировании:

Приложения, поддерживающие сдерживание управления ActiveX, включают:

- Internet Explorer 3.x и больше

- Визуальный базовый 4.x и больше

- Визуальный C q 4.x и больше

- Доступ 95 и больше

- Excel 97 и больше

- Слово 97 и больше

- Доступ 97 и больше

- FrontPage 97 и больше

- PowerPoint 97 и больше

- Визуальный InterDev 97 и больше

Ниже приведены не-Microsoft приложений, которые поддерживают сдерживание управления ActiveX:

- PowerBuilder

- Delphi

- Строитель СЗЗ

- Подключаемый модуль NCompass для Netscape Navigator
