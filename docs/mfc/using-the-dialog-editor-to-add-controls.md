# <a name="using-the-dialog-editor-to-add-controls"></a>Использование редактора диалоговых окон для добавления элементов управления

При создании ресурса диалогового шаблона с помощью [редактора диалоговых окон](../windows/dialog-editor.md)перетащите элементы управления из палитры элементов управления и поместите их в диалоговое окно. Это добавляет спецификации для этого типа элемента управления к ресурсу диалогового шаблона. При создании объекта диалогового окна и вызове `Create` его `DoModal` функции или члена платформа создает элемент управления Windows и помещает его в диалоговое окно на экране.

Вместо этого вы можете [создавать элементы управления вручную,](../mfc/adding-controls-by-hand.md) если хотите. Это больше работы.
