# <a name="window-dialog-and-control-classes"></a>Классы окон, диалоговых окон и элементов управления

Класс `CWnd` и его производные классы инкапсулируют `HWND`, маркер окна Windows. `CWnd` может использоваться сама по себе или как основа для наследования новых классов. Производные классы, предоставляемые библиотекой классов, представляют различные виды окон.

[CWnd](../mfc/reference/cwnd-class.md)<br/>
Базовый класс для всех окон. Можно использовать один из классов, производных от `CWnd`, или создавать собственные классы непосредственно из него.
