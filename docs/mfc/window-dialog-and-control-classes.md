# <a name="window-dialog-and-control-classes"></a>Классы окон, диалоговых окон и элементов управления

Класс `CWnd` и его производные классы инкапсулируют обработчик `HWND` в окно Windows. `CWnd` может использоваться самостоятельно или в качестве основы для наследования новых классов. Производные классы, предоставляемые библиотекой классов, представляют различные виды окон.

[CWnd](../mfc/reference/cwnd-class.md)<br/>
Базовый класс для всех окон. Вы можете использовать один из классов, производных от класса `CWnd` , или получить от него собственные классы.
