# <a name="using-image-lists-in-a-toolbar-control"></a>Использование списков изображений в элементе управления панели инструментов

По умолчанию изображения, используемые кнопками в элементе управления ToolBar, хранятся в виде одного точечного рисунка. Однако можно также хранить изображения кнопок в наборе списков изображений. Объект элемента управления ToolBar может использовать до трех отдельных списков изображений:

- Включенный список изображений содержит изображения для кнопок панели инструментов, которые в данный момент включены.

- Список отключенных изображений содержит изображения для кнопок панели инструментов, которые в настоящее время отключены.

- Выделенный список изображений содержит изображения для кнопок панели инструментов, выделенных в настоящий момент. Этот список изображений используется только в том случае, если панель инструментов использует стиль TBSTYLE_FLAT.

Эти списки изображений используются элементом управления ToolBar при связывании их с `CToolBarCtrl` объектом. Это можно сделать, выполнив вызовы [CToolBarCtrl:: сетимажелист](../mfc/reference/ctoolbarctrl-class.md#setimagelist), [сетдисабледимажелист](../mfc/reference/ctoolbarctrl-class.md#setdisabledimagelist)и [сесотимажелист](../mfc/reference/ctoolbarctrl-class.md#sethotimagelist).

По умолчанию MFC использует `CToolBar` класс для реализации панелей инструментов приложения MFC. Однако `GetToolBarCtrl` функцию члена можно использовать для получения внедренного `CToolBarCtrl` объекта. Затем можно вызывать `CToolBarCtrl` функции членов с помощью возвращенного объекта.
