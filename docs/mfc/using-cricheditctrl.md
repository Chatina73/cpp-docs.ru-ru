# <a name="using-cricheditctrl"></a>Использование CRichEditCtrl

Элемент управления Rich Edit — это окно, в котором пользователь может вводить и редактировать текст. Тексту можно присвоить форматирование символов и абзацев, а также включить внедренные объекты OLE. Элемент управления Rich Edit представлен в MFC классом [CRichEditCtrl](../mfc/reference/cricheditctrl-class.md) .
