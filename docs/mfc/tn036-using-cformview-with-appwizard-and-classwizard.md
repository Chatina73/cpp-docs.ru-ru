# <a name="tn036-using-cformview-with-appwizard-and-classwizard"></a>TN036. Использование CFormView с AppWizard и ClassWizard

В этом техническом примечании описано, как изменить созданное помощью мастера приложение, чтобы оно использовало `CFormView` вместо значения по умолчанию в `CView` качестве основного класса представления. Это поддерживается непосредственно в данной версии Visual C++.
