# <a name="list-items-and-image-lists"></a>Элементы списков и списки изображений

"Элемент" в элементе управления списком[(CListCtrl)](../mfc/reference/clistctrl-class.md)состоит из значка, метки и, возможно, другой информации (в "подпунктах").

Иконки для элементов управления списками содержатся в списках изображений. Один список изображений содержит полноразмерные значки, используемые в представлении значка. Второй, дополнительный список изображений содержит меньшие версии тех же иконок для использования в других представлениях элемента управления. Третий дополнительный список содержит "государственные" изображения, такие как флажки, для отображения перед маленькими иконками в определенных представлениях. Четвертый дополнительный список содержит изображения, отображаемые в отдельных элементах заголовка элементы управления список.

> [!NOTE]
> Если элемент управления представления списка создан с LVS_SHAREIMAGELISTS стилем, вы несете ответственность за уничтожение списков изображений, когда они больше не используются. Укажите этот стиль, если вы назначите одни и те же списки изображений нескольким элементам управления представлениями; в противном случае более одного элемента управления может попытаться уничтожить один и тот же список изображений.

Для получения дополнительной информации о элементах списка смотрите [списки изображений](/windows/win32/Controls/using-list-view-controls) и [элементов и элементов](/windows/win32/Controls/using-list-view-controls) Windows ВСУ. Также смотрите класс [CImageList](../mfc/reference/cimagelist-class.md) в *MFC Reference* and [Using CImageList](../mfc/using-cimagelist.md) в этом семейство статей.
