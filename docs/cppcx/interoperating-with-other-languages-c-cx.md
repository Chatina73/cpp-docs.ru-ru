# <a name="interoperating-with-other-languages-ccx"></a>Взаимодействие с другими языками (C++/CX)

В этой части документации описано, как использовать C++/CX для создания среда выполнения Windows компонентов, которые могут использоваться другими языками программирования и библиотеками. Можно также создавать компоненты, которые могут использоваться не всеми языками. В этом разделе описываются различные аспекты, которые следует C++учитывать при взаимодействии приложения/CX с компонентами, написанными с помощью JavaScript, .NET Framework управляемого языка или библиотеки шаблонов C++ среда выполнения Windows.
