# <a name="platformmetadataflagsattribute-attribute"></a>Platform::Metadata::FlagsAttribute - атрибут

Указывает, что перечисление может обрабатываться как битовое поле (т. е. набор флагов).

## <a name="syntax"></a>Синтаксис

```cpp
public ref class Flags abstract : Attribute
```
