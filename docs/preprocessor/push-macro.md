# <a name="push_macro-no-locpragma"></a>`push_macro` pragma

Сохраняет значение макроса *имени макроса* в верхней части стека для этого макроса.

## <a name="syntax"></a>Синтаксис

> **`#pragma push_macro("`**_имя макроса_**`")`**

## <a name="remarks"></a>Примечания

Можно получить значение *имени макроса* с помощью `pop_macro` .

Пример [ `pop_macro` pragma ](../preprocessor/pop-macro.md) см. в разделе.
