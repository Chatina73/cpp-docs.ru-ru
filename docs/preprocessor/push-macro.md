# <a name="push_macro-pragma"></a>Прагма push_macro

Сохраняет значение макроса *имени макроса* в верхней части стека для этого макроса.

## <a name="syntax"></a>Синтаксис

> **#pragma push_macro ("** _Macro-Name_ **")**

## <a name="remarks"></a>Примечания

Можно получить значение *имени макроса* с помощью `pop_macro`.

Пример см. в разделе [pop_macro](../preprocessor/pop-macro.md) .
