# <a name="no_search_namespace-import-attribute"></a>no_search_namespace, атрибут импорта

**C++Зависящ**

Функция имеет те же функциональные возможности, что и атрибут [no_namespace](../preprocessor/no-namespace.md) , но используется в библиотеках типов, `#import` где используется директива с атрибутом [auto_search](../preprocessor/auto-search.md) .

## <a name="syntax"></a>Синтаксис

> **#import** *Библиотека типов* **no_search_namespace**
