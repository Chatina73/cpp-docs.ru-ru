# <a name="auto_search-import-attribute"></a>auto_search атрибут импорта

**Блок, относящийся только к языку C++**

Указывает, что при ссылке на библиотеку типов с помощью `#import` и сама по себе ссылается на другую библиотеку типов, компилятор может сделать неявным `#import` для другой библиотеки типов.

## <a name="syntax"></a>Синтаксис

> **auto_search** типов **#import** *-Library*

## <a name="remarks"></a>Комментарии

**Завершение блока, относящегося только к языку C++**
