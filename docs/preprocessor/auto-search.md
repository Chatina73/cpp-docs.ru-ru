# <a name="auto_search-import-attribute"></a>auto_search, атрибут импорта

**C++Зависящ**

Указывает, что при ссылке на библиотеку типов с помощью `#import` и сама по себе ссылается на другую библиотеку типов, компилятор может сделать `#import` неявным для другой библиотеки типов.

## <a name="syntax"></a>Синтаксис

> **#import** *Библиотека типов* **auto_search**
