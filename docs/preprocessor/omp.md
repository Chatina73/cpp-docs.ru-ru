# <a name="omp-pragma"></a>Прагма omp

Принимает одну или несколько директив OpenMP вместе с любыми необязательными директивными предложениями.

## <a name="syntax"></a>Синтаксис

> **#pragma omp** *директива*

Дополнительные сведения см. в разделе [директивы OpenMP](../parallel/openmp/reference/openmp-directives.md).
