# <a name="_debug"></a>_DEBUG

Компилятор определяет `_DEBUG` при указании параметра/MTd или/MDd. Эти параметры указывают версии отладки библиотеки времени выполнения C.

Дополнительные сведения см. в статье [Методы отладки CRT](/visualstudio/debugger/crt-debugging-techniques).
