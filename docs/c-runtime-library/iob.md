# <a name="iob"></a>_iob

Массив структур управления stdio.

## <a name="syntax"></a>Синтаксис

```
FILE _iob[_IOB_ENTRIES];
```

## <a name="remarks"></a>Примечания

`IOB_ENTRIES` в stdio.h определяется как 20.
