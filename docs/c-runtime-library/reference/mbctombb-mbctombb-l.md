# <a name="_mbctombb-_mbctombb_l"></a>_mbctombb, _mbctombb_l

Преобразует двухбайтовый многобайтовый символ в соответствующий однобайтовый многобайтовый символ.

> [!IMPORTANT]
> Этот API нельзя использовать в приложениях, выполняемых в среде выполнения Windows. Дополнительные сведения: [Функции CRT, которые не поддерживаются в приложениях универсальной платформы Windows](../../cppcx/crt-functions-not-supported-in-universal-windows-platform-apps.md).

## <a name="syntax"></a>Синтаксис

```C
unsigned int _mbctombb(
   unsigned int c
);
unsigned int _mbctombb_l(
   unsigned int c,
   _locale_t locale
);
```

### <a name="parameters"></a>Параметры

*C*<br/>
Многобайтовый символ для преобразования.

*Языкового стандарта*<br/>
Используемый языковой стандарт.

## <a name="return-value"></a>Возвращаемое значение

В случае успеха, **_mbctombb** и **_mbctombb_l** возвращает один байт характер, который соответствует *c*; в противном случае он возвращает *c*.

## <a name="remarks"></a>Remarks

**Функции _mbctombb** и **_mbctombb_l** преобразовывать данный мультибайтный символ в соответствующий мультибайтный символ с одним байтом. Персонажи должны соответствовать однобайным символам в диапазоне 0x20 - 0x7E или 0xA1 - 0xDF, которые должны быть преобразованы.

Выходное значение зависит от настройки категории **LC_CTYPE** языкового стандарта; дополнительные сведения см. в разделе [setlocale](setlocale-wsetlocale.md). Версия этой функции без **_l** суффикса использует текущий локали для этого поведения, зависящем от локали; версия с **_l** суффикса идентична, за исключением того, что она использует параметр локализации, передаваемый вместо этого. Для получения дополнительной информации см. [Locale](../../c-runtime-library/locale.md).

В предыдущих **версиях, _mbctombb** назывался **zentohan**. Вместо этого используйте **_mbctombb.**

По умолчанию глобальное состояние этой функции приспозировано к приложению. Чтобы изменить это, [см. Глобальное состояние в CRT](../global-state.md).
