# <a name="mbrtoc16-mbrtoc32"></a>mbrtoc16, mbrtoc32

Переводит первый мультибайтовый символ UTF-8 в строке в эквивалентный символ UTF-16 или UTF-32.

## <a name="syntax"></a>Синтаксис

```C
size_t mbrtoc16(
   char16_t* destination,
   const char* source,
   size_t max_bytes,
   mbstate_t* state
);

size_t mbrtoc32(
   char32_t* destination,
   const char* source,
   size_t max_bytes,
   mbstate_t* state
);
```

### <a name="parameters"></a>Параметры

*Назначения*\
Указатель на **char16_t** или **char32_t** эквивалент многобайного символа UTF-8 для преобразования. Если функция не действительна, функция не хранит значение.

*Источник*\
Указатель на многобайтую строку символов UTF-8 для преобразования.

*max_bytes*\
Максимальное количество байтов в *источнике* для изучения для преобразования персонажа. Этот аргумент должен быть значением между одним и числом байтов, включая любой нулевой терминатор, оставшийся в *источнике*.

*Государства*\
Указатель на **mbstate_t** объект состояния преобразования, используемый для интерпретации многобайной строки UTF-8 для одного или нескольких символов вывода.

## <a name="return-value"></a>Возвращаемое значение

При успехе возвращается значение первого из этих условий, которое применяется, учитывая текущее *значение состояния:*

|Значение|Условие|
|-----------|---------------|
|0|Следующее *max_bytes* или меньшее количество символов, преобразованных из *источника,* соответствует символу нулевой ширины, то есть значению, хранящимся, если *назначение* не является нулевым.<br /><br /> *состояние* содержит исходное состояние смены.|
|От 1 до *max_bytes*, включительно|Возвратное значение — это количество байтов *источника,* которые заполняют допустимый многобайтовый символ. Преобразованный широкий символ сохраняется, если *пункт назначения* не является недействительным.|
|–3|Следующий широкий символ, полученный в результате предыдущего вызова функции, хранится в *пункте назначения,* если *пункт назначения* не является недействительным. Никакие байты от *источника* не потребляются этим вызовом к функции.<br /><br /> Когда *источник* указывает на мультибайтовый символ UTF-8, который должен представлять более одного широкого символа (например, суррогатная пара), то *значение состояния* обновляется так, что следующий вызов функции выписывает дополнительный символ.|
|-2|Следующие *байты max_bytes* представляют собой неполный, но потенциально действительный мультибайтный символ UTF-8. Значение не хранится в *пункте назначения.* Этот результат может произойти, если *max_bytes* равен нулю.|
|-1|Произошла ошибка кодирования. Следующие *max_bytes* или меньше байтов не способствуют полному и действию многобайтному символу UTF-8. Значение не хранится в *пункте назначения.*<br /><br /> **EILSE** хранится в **errno,** а *состояние* значения состояния преобразования не уточняется.|

## <a name="remarks"></a>Remarks

Функция **mbrtoc16** читает до *max_bytes* байтов от *источника,* чтобы найти первый полный, действительный utF-8 мультибайт ный символ, а затем хранит эквивалентный utF-16 характер в *пунктназначения*. Если персонажу требуется более одного вывода UTF-16, например, суррогатная пара, то *значение состояния* устанавливается для хранения следующего символа UTF-16 в *пункте назначения* при следующем вызове **mbrtoc16.** Функция **mbrtoc32** идентична, но выход хранится как символ UTF-32.

Если *источник* недействителен, эти функции возвращают эквивалент вызова, сделанного с использованием аргументов **NULL** для *назначения*,(пустая, `""` нулевые строки) для *источника,* и 1 для *max_bytes.* Пройденые значения *назначения* и *max_bytes* игнорируются.

Если *источник* не является недействительным, функция начинается в начале строки и проверяет до *max_bytes* байтов, чтобы определить количество байтов, необходимых для завершения следующего мультибайтного символа UTF-8, включая любые последовательности смены. Если исследуемые байты содержат действительный и полный мультибайтный символ UTF-8, функция преобразует символ в эквивалентный 16-битный или 32-битный широкий символ или символы. Если *назначение* не является недействительным, функция хранит первый (и, возможно, только) результат характера в пункт назначения. При необходимости дополнительных выходных символов значение устанавливается в *состоянии,* так что последующие вызовы на функцию выведут дополнительные символы и вернут значение -3. Если больше не требуется символы вывода, то *состояние* устанавливается в исходное состояние смены.

Для преобразования мультибайтных символов, не относясь от UTF-8, используйте символы [mbrtowc,](mbrtowc.md) [mbtowc или _mbtowc_l](mbtowc-mbtowc-l.md) функции.

По умолчанию глобальное состояние этой функции приспозировано к приложению. Чтобы изменить это, [см. Глобальное состояние в CRT](../global-state.md).
