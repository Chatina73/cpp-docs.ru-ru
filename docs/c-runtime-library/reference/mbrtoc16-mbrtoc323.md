# <a name="mbrtoc16-mbrtoc32"></a>mbrtoc16, mbrtoc32

Преобразует первый многобайтовый символ в строке обычных символов в эквивалентный символ в кодировке UTF-16 или UTF-32.

## <a name="syntax"></a>Синтаксис

```C
size_t mbrtoc16(
   char16_t* destination,
   const char* source,
   size_t max_bytes,
   mbstate_t* state
);

size_t mbrtoc32(
   char32_t* destination,
   const char* source,
   size_t max_bytes,
   mbstate_t* state
);

```

### <a name="parameters"></a>Параметры

*Назначение*<br/>
Указатель на **char16_t** или **char32_t** эквивалентный Многобайтовый символ для преобразования. Если значение равно null, функция не сохраняет значение.

*source*<br/>
Указатель на строку многобайтовых символов, которую нужно преобразовать.

*max_bytes*<br/>
Максимальное число байтов в *источника* следует просмотреть на наличие символа для преобразования. Это должно быть значение от 1 до число байтов, включая символ завершения null, оставшихся в *источника*.

*state*<br/>
Указатель на **mbstate_t** объект состояния преобразования, используемые для интерпретации многобайтовой строки в один или несколько выходных символов.

## <a name="return-value"></a>Возвращаемое значение

В случае успешного выполнения возвращает значение первого из этих условий, к которым применяется, с учетом текущего *состояние* значение:

|Значение|Условие|
|-----------|---------------|
|0|Следующий *max_bytes* или меньше символов, преобразованных из *источника* соответствуют расширенному символу null, который является значением, хранящимся Если *назначения* не равно null.<br /><br /> *состояние* содержит начальное состояние сдвига.|
|Между 1 и *max_bytes*включительно|Возвращаемое значение больше байтов *источника* , составляют допустимый Многобайтовый символ. Преобразованный расширенный символ хранится в том случае, если *назначения* не равно null.|
|-3|Следующий расширенный символ, полученный в результате предыдущего вызова функции был сохранен в *назначения* Если *назначения* не равно null. Ни одного байта из *источника* используются этим вызовом функции.<br /><br /> При *источника* указывает на Многобайтовый символ, который требуется несколько расширенных символов для представления (например, суррогатная пара), а затем *состояние* значение обновляется, чтобы при следующем вызове функции записывает  out дополнительный символ.|
|-2|Следующий *max_bytes* байтов представляют неполный, но потенциально допустимый Многобайтовый символ. Значение не хранится в *назначения*. Это может произойти, если *max_bytes* равно нулю.|
|-1|Произошла ошибка кодирования. Следующий *max_bytes* или меньше байт не составляют полный и допустимый Многобайтовый символ. Значение не хранится в *назначения*.<br /><br /> **EILSEQ** хранится в **errno** и состояние преобразования *состояние* не определен.|

## <a name="remarks"></a>Примечания

**Mbrtoc16** функция считывает до *max_bytes* байтов из *источника* найти первый полный допустимый Многобайтовый символ и затем сохраняет эквивалентный UTF-16 символ в *назначения*. Исходные байты интерпретируются в соответствии с многобайтовым языковым стандартом текущего потока. Если многобайтового символа требуется несколько символов UTF-16 выходных данных, например суррогатной пары, то *состояние* имеет значение хранения следующий символ UTF-16 в *назначения* при следующем вызове для **mbrtoc16**. **Mbrtoc32** функция идентична, но выходные данные сохраняются как символ UTF-32.

Если *источника* имеет значение null, эти функции возвращают результат эквивалентный вызову установлено с использованием аргументов **NULL** для *назначения*, **«»** для *источника*и 1 для *max_bytes*. Значения, передаваемые в *назначения* и *max_bytes* игнорируются.

Если *источника* — не null, функция начинается с начала строки и проверяет не более *max_bytes* байт для определения числа байтов, необходимых для составления следующего многобайтового символа, включая любые последовательности сдвигов. Если проверяемые байты содержат допустимый и полный многобайтовый символ, функция преобразует его в эквивалентный 16- или 32-битовый расширенный символ (либо несколько символов). Если *назначения* не равно NULL, функция сохраняет первый (и, возможно, единственный) результат символ в конечном объекте. Если требуются дополнительные выходные символы, значение задается в *состояние*, чтобы последующие вызовы функции выводились дополнительные символы и возвращают значение -3. Если символов больше не требуется, *состояние* присваивается начальное состояние сдвига.
