# <a name="localeconv"></a>localeconv

Получает подробные сведения о параметрах языкового стандарта.

## <a name="syntax"></a>Синтаксис

```C
struct lconv *localeconv( void );
```

## <a name="return-value"></a>Возвращаемое значение

**localeconv** возвращает указатель на заполненный объект структуры типа [lconv.](../../c-runtime-library/standard-types.md) Значения, содержащиеся в объекте, копируются из параметров локализации в локальном хранилище потоков и могут быть перезаписаны последующими вызовами **в localeconv.** Изменения, внесенные в значения в этом объекте, не изменяют настройки локализации. Призывы к [установке](setlocale-wsetlocale.md) с *значениями категории* **LC_ALL,** **LC_MONETARY**или **LC_NUMERIC** перезаписывают содержимое структуры.

## <a name="remarks"></a>Remarks

Функция **localeconv** получает подробную информацию о количественном форматировании для текущего локализации. Эти сведения хранятся в структуре типа **lconv**. Структура **lconv**, определенная в LOCALE.H, содержит следующие члены:

|Поле|Значение|
|-|-|
decimal_point,<br/>_W_decimal_point|Указатель на десятичную точку символа для неденежных количеств.
Thousands_sep<br/>_W_thousands_sep|Указатель на символ, который отделяет группы цифр слева от десятичной точки для неденежных количеств.
группирование|Указатель на **символ**размером с ряд, который содержит размер каждой группы цифр в неденежных количествах.
int_curr_symbol,<br/>_W_int_curr_symbol|Указатель на символ международной валюты для текущего локализации. Первые три символа определяют международный алфавитный символ валюты, как определено в стандарте *ISO 4217. Коды для представления валют и фондов*. Четвертый символ (непосредственно перед нуль-символом) разделяет международный символ валюты и денежное значение.
currency_symbol,<br/>_W_currency_symbol|Указатель на символ местной валюты для текущего локализации.
mon_decimal_point,<br/>_W_mon_decimal_point|Указатель на десятичную точку символа для денежных количеств.
mon_thousands_sep,<br/>_W_mon_thousands_sep|Указатель на сепаратор для групп цифр слева от десятичной места в денежных количествах.
mon_grouping|Указатель на **символ**размером с ряд, который содержит размер каждой группы цифр в денежных количествах.
positive_sign,<br/>_W_positive_sign|Строка, обозначающая знак для неотрицательных денежных значений.
negative_sign,<br/>_W_negative_sign|Строка, обозначающая знак для отрицательных денежных значений.
int_frac_digits|Число цифр справа от десятичной запятой в международных форматированных денежных значениях.
frac_digits|Число цифр справа от десятичной запятой в форматированных денежных значениях.
p_cs_precedes|Задайте значение 1, если символ валюты предшествует неотрицательному форматированному денежному значению. Задайте значение 0, если символ стоит после значения.
p_sep_by_space|Задайте значение 1, если символ валюты отделяется пробелом от неотрицательного форматированного денежного значения. Задайте значение 0, если символ не отделяется пробелом.
n_cs_precedes|Задайте значение 1, если символ валюты предшествует отрицательному форматированному денежному значению. Задайте значение 0, если символ следует за значением.
n_sep_by_space|Задайте значение 1, если символ валюты отделяется пробелом от отрицательного форматированного денежного значения. Задайте значение 0, если символ не отделяется пробелом.
p_sign_posn|Позиция положительного знака в форматированных неотрицательных денежных значениях.
n_sign_posn|Позиция положительного знака в форматированных отрицательных денежных значениях.

За исключением указанного, члены структуры `char *` `wchar_t *` **lconv,** которые имеют и версии, являются указателями на строки. Любой из них, который равен **""** (или **L"** для **wchar_t)** <strong>\*</strong>либо нулевой длины или не поддерживается в текущем месте. Обратите внимание, что **decimal_point** и **_W_decimal_point** всегда поддерживаются и ненулевой длины.

**Члены структуры** являются небольшими неотрицательными числами, а не символами. Любые из них, равные **CHAR_MAX**, не поддерживаются в текущем языковом стандарте.

Значения **группировки** и **mon_grouping** интерпретируются в соответствии со следующими правилами:

- **CHAR_MAX** - Не выполняйте никакой дальнейшей группировки.

- 0 - Используйте предыдущий элемент для каждой из оставшихся цифр.

- *n* - Количество цифр, вкоторыехающих текущую группу. Проверяется следующий элемент для определения размера следующей группы разрядов перед текущей группой.

Значения **int_curr_symbol** интерпретируются в соответствии со следующими правилами:

- Первые три символа определяют международный алфавитный символ валюты, как определено в стандарте *ISO 4217. Коды для представления валют и фондов*.

- Четвертый символ (непосредственно перед нуль-символом) отделяет международный символ валюты от денежного значения.

Значения **p_cs_precedes** и **n_cs_precedes** интерпретируются в соответствии со следующими правилами (правило **n_cs_precedes** указано в круглых скобках):

- 0 - Символ валюты следует значению для ненегативной (негативной) отформатируемой денежной стоимости.

- 1 - Символ валюты предшествует значению для ненегативной (негативной) отформатируемой денежной стоимости.

Значения **p_sep_by_space** и **n_sep_by_space** интерпретируются в соответствии со следующими правилами (правило **n_sep_by_space** указано в круглых скобках):

- 0 - Символ валюты отделен от значения пространством для ненегативной (негативной) отформатируемой денежной стоимости.

- 1 - Нет разделения пространства между символом валюты и значением для ненегативной (негативной) отформатируемой денежной стоимости.

Значения **p_sign_posn** и **n_sign_posn** интерпретируются в соответствии со следующими правилами:

- 0 - Parentheses окружают символ количества и валюты.

- 1 - Строка знака предшествует количеству и символу валюты.

- 2 - Строка знака следует за символом количества и валюты.

- 3 - Строка знака немедленно предшествует символу валюты.

- 4 - Строка знака немедленно следует за символом валюты.

По умолчанию глобальное состояние этой функции приспозировано к приложению. Чтобы изменить это, [см. Глобальное состояние в CRT](../global-state.md).
