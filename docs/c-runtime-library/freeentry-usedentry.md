# <a name="freeentry-usedentry"></a>_FREEENTRY, _USEDENTRY

## <a name="syntax"></a>Синтаксис

```
#include <malloc.h>
```

## <a name="remarks"></a>Примечания

Эти константы представляют значения, присвоенные подпрограммами `_heapwalk` элементу **_useflag** структуры **_HEAPINFO**. Они показывают состояние записи кучи.
