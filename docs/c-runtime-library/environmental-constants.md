# <a name="environmental-constants"></a>Константы среды

## <a name="syntax"></a>Синтаксис

```
#include <stdlib.h>
```

## <a name="remarks"></a>Remarks

Эта константа определяет длину строк среды.

|Константа|Значение|
|--------------|-------------|
|`_MAX_ENV`|Максимальный размер строки среды.|
