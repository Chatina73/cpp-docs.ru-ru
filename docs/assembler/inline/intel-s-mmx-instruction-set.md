# <a name="intel39s-mmx-instruction-set"></a>Intel&#39;набор инструкций MMX s

**Блок, относящийся только к системам Microsoft**

Microsoft C++ компилятор позволяет использовать набор во встроенном ассемблере инструкций MMX (мультимедийное расширение) корпорации Intel. Инструкции MMX также поддерживаются дизассемблированным кодом отладчика. Компилятор выдает предупреждение, если функция содержит инструкции MMX, но не содержит инструкцию EMMS для очистки мультимедийного состояния. Дополнительные сведения см. на веб-сайте Intel.

**Завершение блока, относящегося только к системам Майкрософт**
