# <a name="intel39s-mmx-instruction-set"></a>Набор&#39;инструкций Intel s MMX

**Блок, относящийся только к системам Microsoft**

Компилятор Майкрософт C++ позволяет использовать набор инструкций Intel MMX (расширение мультимедиа) во встроенном ассемблере. Инструкции MMX также поддерживаются дизассемблированным кодом отладчика. Компилятор выдает предупреждение, если функция содержит инструкции MMX, но не содержит инструкцию EMMS для очистки мультимедийного состояния. Дополнительные сведения см. на веб-сайте Intel.
