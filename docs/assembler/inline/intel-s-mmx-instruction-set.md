# <a name="intel39s-mmx-instruction-set"></a>Набор инструкций MMX Intel&#39;s

**Блок, относящийся только к системам Microsoft**

Компилятор Microsoft C++ позволяет использовать набор инструкций Intel MMX (расширение мультимедиа) во встроенном ассемблере. Инструкции MMX также поддерживаются дизассемблированным кодом отладчика. Компилятор выдает предупреждение, если функция содержит инструкции MMX, но не содержит инструкцию EMMS для очистки мультимедийного состояния. Дополнительные сведения см. на веб-сайте Intel.
