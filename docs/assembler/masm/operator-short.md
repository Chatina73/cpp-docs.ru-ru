# <a name="operator-short"></a>оператор SHORT

Задает тип *метка* коротких. Все ссылки на *метка* должен быть коротким (в диапазоне от -128 до + 127 байт из инструкции перехода к *метка*).

## <a name="syntax"></a>Синтаксис

> КОРОТКИЕ метки
