# <a name="operator-short"></a>оператор SHORT

Задает короткий тип *метки* . Все переходы к *метке* должны быть короткими (в пределах от-128 до + 127 байт от инструкции перехода к *метке*).

## <a name="syntax"></a>Синтаксис

> **Короткая** *Метка*
