# <a name="data"></a>.DATA

 (только 32-разрядный MASM.) При использовании с [. MODEL](dot-model.md)запускает ближайший сегмент данных для инициализированных данных (имя сегмента _data).

## <a name="syntax"></a>Синтаксис

> **.\ данных**
> ⟦ *сегментитем* ⟧...
