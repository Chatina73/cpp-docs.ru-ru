# <a name="operator-amp"></a>Оператор &amp;

Побитовое **AND**. Используется только в пределах [. Если](../../assembler/masm/dot-if.md), [. ХОТЯ](../../assembler/masm/dot-while.md), или [. ПОВТОРИТЕ](../../assembler/masm/dot-repeat.md) блокирует и вычисляется во время выполнения, а не во время сборки.

## <a name="syntax"></a>Синтаксис

> expression1 и expression2
