# <a name="union"></a>UNION

Объявляет объединение одного или нескольких типов данных. *Объявления полей* должны быть допустимыми определениями данных. Опустить метку " [конец](ends-masm.md) *имени* " для вложенных определений **объединения** .

## <a name="syntax"></a>Синтаксис

> *Name* **Union** ⟦*alignment*⟧ ⟦__,__ **неуникальный**⟧ \
> *объявления полей*\
> ⟦*имя*⟧ **заканчивается**
