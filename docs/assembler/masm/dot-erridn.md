# <a name="perioderridn-perioderridni"></a>&period;ERRIDN, &period;ERRIDNI

Приводит к ошибке, если текст элементов идентичны.

## <a name="syntax"></a>Синтаксис

> **. ERRIDN**\[**я**] *textitem1* __,__ *textitem2* \[ __,__ *сообщение*]

## <a name="remarks"></a>Примечания

Если  **&period;ERRIDNI** — учитывая, сравнение выполняется без учета регистра.
