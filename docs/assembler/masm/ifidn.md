# <a name="ifidn-ifidni"></a>IFIDN IFIDNI

Предоставляет сборку, если текстовых элементов идентичны.

## <a name="syntax"></a>Синтаксис

> **IFIDN**\[**я**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Примечания

Если **IFIDNI** — учитывая, сравнение выполняется без учета регистра. См. в разделе [IF](../../assembler/masm/if-masm.md) сложный синтаксис.
