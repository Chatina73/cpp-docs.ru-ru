# <a name="operator-lt"></a>Оператор LT

Возвращает значение true (-1), если *expression1* меньше *Expression2*, или возвращает false (0), если нет.

## <a name="syntax"></a>Синтаксис

> *Выражение1* , **lt** , *выражение2*
