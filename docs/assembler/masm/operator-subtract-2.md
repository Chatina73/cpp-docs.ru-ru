# <a name="operator--"></a>Оператор -

Возвращает значение *expression1* минус *выражение2*. Второй оператор меняет знак *выражения* на противоположный.

## <a name="syntax"></a>Синтаксис

> *expression1* **-** *expression2*
>
> __-__*выражение*
