# <a name="data"></a>\@данных

Имя группы данных по умолчанию. Возвращает значение ДГРАУП для всех моделей, кроме **плоского**. Вычисляется как **плоский** в модели **плоской** памяти (текстовый макрос).

## <a name="syntax"></a>Синтаксис

> **\@данных**
