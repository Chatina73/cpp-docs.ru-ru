# <a name="operator-eq"></a>Оператор EQ

Возвращает значение true (-1), если *expression1* равно *Expression2*, или возвращает false (0), если нет.

## <a name="syntax"></a>Синтаксис

> *Выражение1* **EQ** *выражение2*
