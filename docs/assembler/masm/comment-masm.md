# <a name="comment"></a>COMMENT

Обрабатывает весь *текст* между или в той же строке, что и разделители, в качестве комментария.

## <a name="syntax"></a>Синтаксис

> *Разделитель* комментариев ⟦*Text*⟧ \
> ⟦*Text*⟧ \
> ⟦*Text*⟧ *Разделитель* ⟦*Text*⟧
