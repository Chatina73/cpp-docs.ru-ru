# <a name="ml-nonfatal-error-a2010"></a>Некритичная ошибка ML A2010

**Недопустимое выражение типа**

Операнд для [этого](../../assembler/masm/operator-this.md) или типа [ptr](../../assembler/masm/operator-ptr.md) не является допустимым выражением типа.
