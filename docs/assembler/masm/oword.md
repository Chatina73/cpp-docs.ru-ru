# <a name="oword"></a>OWORD

Используется в качестве описателя типа, если требуется 16-байтовый тип данных.

## <a name="syntax"></a>Синтаксис

> ⟦*имя*⟧ **оворд**  *инициализатор* ⟦ __,__ *инициализатор* ... ⟧
