# <a name="ml-nonfatal-error-a2034"></a>Некритичная ошибка ML A2034

**должен находиться в блоке сегмента**

Один из следующих элементов обнаружен вне блока сегмента:

- Инструкция

- Определение метки

- [Этот](../../assembler/masm/operator-this.md) оператор

- Оператор [$](../../assembler/masm/dollar.md)

- Определение процедуры

- Директива [aligned](../../assembler/masm/align-masm.md)

- Директива [org](../../assembler/masm/org.md)
