# <a name="ml-nonfatal-error-a2078"></a>Некритичная ошибка ML A2078

**Инструкция не допускает ДАЛЕКОй непрямой адресации**

Условный переход или цикл не могут принимать операнды в памяти. Ему необходимо предоставить относительный адрес или метку.
