# <a name=""></a>\@\@.

Определяет метку кода, которую можно распознать только между *Label1* и *ярлык2*, где *Label1* — это начало кода или Предыдущая метка `@@:`, а *ярлык2* — либо конец кода, либо Следующая `@@:` метка. См. [\@B](at-b.md) и [\@F](at-f.md).
