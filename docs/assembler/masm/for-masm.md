# <a name="for-masm"></a>FOR (MASM)

Помечает блок, который повторяется один раз для каждого *аргумент*, с текущим *аргумент* замена *параметр* на каждом проходе.

## <a name="syntax"></a>Синтаксис

> ДЛЯ *параметр* [[: REQ |: =*по умолчанию*]], \< *аргумент* [[, *аргумент*]]... ><br/>
> *Инструкции*<br/>
> ENDM

## <a name="remarks"></a>Примечания

Совпадение с кодом [IRP](../../assembler/masm/irp.md).
