# <a name="operator-"></a>Оператор ;;

Обрабатывает *текст* виде комментария в макросе, проявляющаяся только в определении макроса. Список не содержит *текст* где макрос расширяется.

## <a name="syntax"></a>Синтаксис

> ;; *текста*
