# <a name="ml-nonfatal-error-a2189"></a>Некритичная ошибка ML A2189

**Недопустимое сочетание с выравниванием сегмента**

Выравнивание, заданное директивой **выравнивания** или, `EVEN` было больше, чем текущее выравнивание сегмента, как указано в директиве **сегмента** .
