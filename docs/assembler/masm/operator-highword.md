# <a name="operator-highword"></a>Оператор HIGHWORD

Возвращает старшие 16 битов нижнего 32 бита *выражения*. Выражения MASM — это 64-разрядные значения.

## <a name="syntax"></a>Синтаксис

>  *Выражение* HIGHWORD
