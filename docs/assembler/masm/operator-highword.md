# <a name="operator-highword"></a>Оператор HIGHWORD

Возвращает старшие 16 разрядов младшие 32 разряда *выражение*. Выражения MASM являются 64-разрядными значениями.

## <a name="syntax"></a>Синтаксис

> Выражение HIGHWORD
