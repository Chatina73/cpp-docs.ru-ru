# <a name="while-32-bit-masm"></a>. WHILE (32-разрядный MASM)

Создает код, выполняющий блок *инструкций* , пока *условие* остается истинным. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.**  *Условие* while\
> *инструкции*\
> **.ENDW**
