# <a name="486p-32-bit-masm"></a>.486P (32-разрядный компилятор MASM)

Включает сборку всех инструкций (включая привилегированные) для процессора 80486. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.486P**
