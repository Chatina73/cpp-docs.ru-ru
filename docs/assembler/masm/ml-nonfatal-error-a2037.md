# <a name="ml-nonfatal-error-a2037"></a>Некритичная ошибка ML A2037

**оператор не допускается в определении структуры**

Определение структуры содержит недопустимую инструкцию.

Структура не может содержать инструкции, метки, процедуры, директивы потока управления, [. Запуск](dot-startup.md)или [. Завершите работу](dot-exit.md).
