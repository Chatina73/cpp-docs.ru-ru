# <a name="typedef"></a>TYPEDEF

Определяет новый тип или имя **, именуемое** *Name*, которое эквивалентно *типу* или *протодефинитион*.

## <a name="syntax"></a>Синтаксис

> *Name* **TYPEDEF** *Type* имя  |   *протодефинитион*
