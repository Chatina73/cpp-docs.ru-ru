# <a name="ml-fatal-error-a1005"></a>Неустранимая ошибка ML A1005

**ограничение ассемблера: таблица с именем параметра макроса заполнена**

Для макроса определено слишком много параметров, локальных переменных или меток макросов. В таблице имен макросов больше нет места.

Определите меньшее или меньшее число имен или удалите ненужные макросы.
