# <a name="ml-nonfatal-error-a2004"></a>Некритичная ошибка ML A2004

**конфликт типов символов: идентификатор**

Директива [екстерндеф](externdef.md) или [Label](label-masm.md) использовалась для переменной, символа, структуры данных или метки, которая была определена в том же модуле, но с другим типом.
