# <a name="ml-nonfatal-error-a2019"></a>Некритичная ошибка ML A2019

**операнд должен быть типом записи или полем**

Операнд, следующий за оператором [Width](../../assembler/masm/operator-width.md) или [Mask](../../assembler/masm/operator-mask.md) , является недопустимым.

Оператор [Width](../../assembler/masm/operator-width.md) принимает операнд, который является именем поля или записи. Оператор [Mask](../../assembler/masm/operator-mask.md) принимает операнд, который является именем поля или типа записи.
