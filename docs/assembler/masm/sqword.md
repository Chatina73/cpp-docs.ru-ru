# <a name="sqword"></a>SQWORD

Выделяет и при необходимости инициализирует 8 байт со знаком из хранилища для каждого `initializer`. Также можно использовать в качестве спецификатора типа в любом является допустимым типом.

## <a name="syntax"></a>Синтаксис

> Инициализатор SQWORD имя [[, инициализатор]]...
