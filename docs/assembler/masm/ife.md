# <a name="ife"></a>IFE

Предоставляет сборку, если *выражение* имеет значение false (0).

## <a name="syntax"></a>Синтаксис

> IFE выражение

## <a name="remarks"></a>Примечания

См. в разделе [IF](../../assembler/masm/if-masm.md) сложный синтаксис.
