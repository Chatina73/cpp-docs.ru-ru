# <a name="mmx-32-bit-masm"></a>. MMX (32-разрядный MASM)

Включает сборку MMX или одиночную инструкцию, несколько данных (SIMD). (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.MMX**
