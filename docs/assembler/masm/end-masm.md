# <a name="end"></a>END

Помечает конец модуля и при необходимости устанавливает точку входа программы в *procId*.

## <a name="syntax"></a>Синтаксис

> **End** ⟦*procId*⟧

## <a name="remarks"></a>Комментарии

Аргумент *procId* допустим только в 32-разрядном MASM.
