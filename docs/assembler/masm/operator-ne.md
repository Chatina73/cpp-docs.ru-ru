# <a name="operator-ne"></a>Оператор NE

Возвращает значение true (-1), если *expression1* не равно *Expression2*, или возвращает false (0), если это происходит.

## <a name="syntax"></a>Синтаксис

> *expression1* **Ne** *выражение2*
