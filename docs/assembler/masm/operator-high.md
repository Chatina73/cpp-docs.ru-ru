# <a name="operator-high"></a>Оператор HIGH

Возвращает старшие 8 бит младших 16 разрядов *выражения*. Выражения MASM — это 64-разрядные значения.

## <a name="syntax"></a>Синтаксис

> **Высокое** *выражение*
