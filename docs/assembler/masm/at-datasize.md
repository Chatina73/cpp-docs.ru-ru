# <a name="datasize"></a>\@DataSize

0 для **мелких**, **малых**, **средних** и **плоских** моделей, 1 для **Compact** и **больших** моделей и 2 для **огромной** модели (числовой равенство).

## <a name="syntax"></a>Синтаксис

> **\@DataSize**
