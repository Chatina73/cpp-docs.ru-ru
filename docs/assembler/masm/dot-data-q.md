# <a name="data"></a>.DATA?

(только 32-разрядный MASM.) При использовании с [. MODEL](dot-model.md)запускает ближайший сегмент данных для неинициализированных данных (имя сегмента _BSS).

## <a name="syntax"></a>Синтаксис

> **. Data?**\
> ⟦ *сегментитем* ⟧...
