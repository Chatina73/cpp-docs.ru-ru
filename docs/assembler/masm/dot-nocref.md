# <a name="nocref"></a>.NOCREF

Подавляет список символов в файле символов таблицы и браузера.

## <a name="syntax"></a>Синтаксис

> . NOCREF [[имя [[, имя]]...]]

## <a name="remarks"></a>Примечания

Если указаны имена, заданные имена, подавляются. Совпадение с кодом [. XCREF](../../assembler/masm/dot-xcref.md).
