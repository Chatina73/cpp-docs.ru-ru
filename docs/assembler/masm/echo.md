# <a name="echo"></a>ECHO

Отображает *сообщение* для стандартного выходного устройства (по умолчанию это экран).

## <a name="syntax"></a>Синтаксис

> **Эхо** - *сообщение*

## <a name="remarks"></a>Примечания

То же, что и [% out](../../assembler/masm/percent-out.md).
