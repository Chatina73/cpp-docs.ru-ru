# <a name="echo"></a>ECHO

Отображает *сообщение* для стандартного выходного устройства (по умолчанию это экран).

## <a name="syntax"></a>Синтаксис

> **Эхо** - *сообщение*

## <a name="remarks"></a>Комментарии

То же, что и [% out](percent-out.md).
