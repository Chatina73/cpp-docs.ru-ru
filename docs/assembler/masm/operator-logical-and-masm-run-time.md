# <a name="operator-ampamp-masm-run-time"></a>Оператор &amp; &amp; (время выполнения MASM)

Логические **и**. Используется только в [. Если](dot-if.md), [. WHILE](dot-while.md)или [. ПОВТОРЯЮЩИеся](dot-repeat.md) блоки и оцениваются во время выполнения, а не во время сборки.

## <a name="syntax"></a>Синтаксис

> *expression1* **&&** *expression2*
