# <a name="operator-ampamp-masm-run-time"></a>&amp; &amp;оператора (время выполнения MASM)

Логические **и**. Используется только в [. Если](../../assembler/masm/dot-if.md), [. WHILE](../../assembler/masm/dot-while.md)или [. ПОВТОРЯЮЩИеся](../../assembler/masm/dot-repeat.md) блоки и оцениваются во время выполнения, а не во время сборки.

## <a name="syntax"></a>Синтаксис

> *expression1* **&&** *expression2*
