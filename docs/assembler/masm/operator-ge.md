# <a name="operator-ge"></a>Оператор GE

Возвращает значение true (-1), если *expression1* больше или равно *Expression2*, или возвращает false (0), если нет.

## <a name="syntax"></a>Синтаксис

> *Выражение1* **GE** , *выражение2*
