# <a name="ml-nonfatal-error-a2219"></a>Некритичная ошибка ML A2219

> Неправильное выравнивание для смещения в коде очистки

## <a name="remarks"></a>Комментарии

Операнд для [ &period; аллокстакк](dot-allocstack.md) и [ &period; саверег](dot-savereg.md) должен быть кратным 8.  Операнд для [ &period; SAVEXMM128](dot-savexmm128.md) и [ &period; сетфраме](dot-setframe.md) должен быть кратен 16.
