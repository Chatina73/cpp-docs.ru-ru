# <a name="textequ"></a>TEXTEQU

Назначает *textitem* для *имя*. *Textitem* может быть строковый литерал, константа предшествует `%`, или строка, возвращаемая функцией макрос.

## <a name="syntax"></a>Синтаксис

> Имя TEXTEQU [[textitem]]
