# <a name="catstr"></a>\@катстр

Функция макроса, которая объединяет одну или несколько строк. Возвращает строку.

## <a name="syntax"></a>Синтаксис

> **\@ Катстр (** *строка1* ⟦__,__ *строка2*... ⟧ **)**
