# <a name="ml-fatal-error-a1000"></a>Неустранимая ошибка ML A1000

**не удается открыть файл: имя файла**

Ассемблеру не удалось открыть исходный, включаемый или выходной файл.

Может возникнуть одна из следующих причин.

- Файл не существует.

- Файл используется другим процессом.

- Недопустимое имя файла.

- Файл, который доступен только для чтения с выходным именем файла, уже существует.

- Текущий диск заполнен.

- Текущий каталог является корневым и заполнен.

- Не удается записать устройство в.

- Диск не готов.
