# <a name="equ"></a>EQU

Первая директива назначает числовое значение *выражение* для *имя*.

## <a name="syntax"></a>Синтаксис

> *имя* равно *выражение*

> *имя* равно \< *текста*>

## <a name="remarks"></a>Примечания

*Имя* не может быть переопределен более поздней версии.

Второй директивы назначает указанное *текст* для *имя*. *Имя* можно назначить другой *текст* позже. См. в разделе [TEXTEQU](../../assembler/masm/textequ.md).
