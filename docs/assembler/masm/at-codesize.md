# <a name="codesize"></a>\@Кодесизе

0 для **мелких**, **малых**, **компактных**и **плоских** моделей и 1 для **средних**, **крупных**и **огромных** моделей (числовой равенство).

## <a name="syntax"></a>Синтаксис

> **\@Кодесизе**
