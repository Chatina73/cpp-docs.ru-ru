# <a name="df"></a>DF

Выделяет и при необходимости инициализирует 6 байтов хранилища для каждого *инициализатора*. **DF** является синонимом [фворд](fword.md).

## <a name="syntax"></a>Синтаксис

> ⟦*имя*⟧ **DF** *инициализатор* ⟦__,__ *инициализатор* ... ⟧

## <a name="remarks"></a>Комментарии

Также можно использовать в качестве спецификатора типа в любом месте, где тип является допустимым.
