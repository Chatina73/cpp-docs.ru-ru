# <a name="errdef"></a>.ERRDEF

Выдает ошибку, если *Name* — ранее определенная метка, переменная или символ.

## <a name="syntax"></a>Синтаксис

> **. Еррдеф** *имя* ⟦__,__ *сообщение*⟧
