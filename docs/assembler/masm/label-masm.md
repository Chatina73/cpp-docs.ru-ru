# <a name="label"></a>LABEL

Создает новую метку путем присвоения *имени* текущего значения счетчика Location и заданной *куалифиедтипе* .

## <a name="syntax"></a>Синтаксис

> *имя* **метки** *куалифиедтипе*
>
> *имя* **Метка** ⟦**NEAR**  |    |  ⟧ **ptr** ⟦*куалифиедтипе*⟧
