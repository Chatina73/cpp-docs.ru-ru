# <a name="group-32-bit-masm"></a>Группа (32-разрядный компилятор MASM)

Добавьте указанные *сегменты* в группу с *именем.* (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> *сегмент* группы имен ⟦ __,__ *сегмент* ... ⟧
