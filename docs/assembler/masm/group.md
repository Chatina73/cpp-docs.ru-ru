# <a name="group"></a>GROUP

(только 32-разрядный MASM.) Добавьте указанные *сегменты* в группу с *именем.*

## <a name="syntax"></a>Синтаксис

>   *сегмент* группы имен ⟦__,__ *сегмент* ... ⟧
