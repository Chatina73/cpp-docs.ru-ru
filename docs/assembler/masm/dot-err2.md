# <a name="err2"></a>.ERR2

[. Блок ERR](dot-err.md) , вычисленный при каждой сборке Pass, если **параметр: SETIF2** имеет **значение true.**

## <a name="syntax"></a>Синтаксис

> **. ERR2** ⟦*сообщение*⟧
