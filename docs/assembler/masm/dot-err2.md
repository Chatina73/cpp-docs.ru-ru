# <a name="err2"></a>.ERR2

[. ERR](../../assembler/masm/dot-err.md) проверяется при каждом проходе сборки, если блок **параметр: SETIF2** является **значение TRUE.**

## <a name="syntax"></a>Синтаксис

> . ERR2 [[сообщение]]
