# <a name="struct"></a>STRUCT

Объявляет тип структуры с указанными *объявлениями полей*. Каждое поле должно быть допустимым определением данных. То же, что и [струк](struc.md).

## <a name="syntax"></a>Синтаксис

> *имя* **структуры** ⟦*Выравнивание*⟧ ⟦ __,__ **неуникальное**⟧ *объявление поля*\
> *имя* **заканчивается**

## <a name="remarks"></a>Примечания

Аргумент *Name* должен быть одинаковым в открывающей и закрывающей инструкциях.
