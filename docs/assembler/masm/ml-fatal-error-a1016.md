# <a name="ml-fatal-error-a1016"></a>Неустранимая ошибка ML A1016

**Ошибка внутреннего ассемблера**

Драйвер MASM, называемый ML.exe, создается Системная ошибка.

Соберите сведения об обстоятельствах появления ошибки и обратитесь в корпорацию Майкрософт. Службы технической поддержки доступна на [ http://support.microsoft.com/ ](http://support.microsoft.com/).
