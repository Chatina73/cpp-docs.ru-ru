# <a name="ml-fatal-error-a1016"></a>Неустранимая ошибка ML A1016

**Внутренняя ошибка ассемблера**

Драйвер MASM, называемый ML.exe, создал системную ошибку.

Соберите сведения об обстоятельствах появления ошибки и обратитесь в корпорацию Майкрософт. Служба поддержки продуктов доступна по адресу [https://support.microsoft.com/](https://support.microsoft.com/) .
