# <a name="ml-error-messages"></a>Сообщения об ошибках ML

Сообщения об ошибках, создаваемые компонентами MASM, делятся на три категории:

- **Неустранимые ошибки.** Они указывают на серьезную проблему, препятствующую завершению нормальной работы программы.

- **Некритические ошибки.** Программа может завершить процесс. Если это так, то его результат, скорее всего, не будет одним из нужных.

- **Предупреждения.** Эти сообщения указывают на условия, которые могут помешать получению нужных результатов.

Все сообщения об ошибках имеют следующий вид:

> *Служебная программа*: *имя файла* (*строка*): {*Error_type*} (*код*): *Message_text*

Здесь:

\ *служебной программы*
Программа, которая отправила сообщение об ошибке.

*Имя файла*\
Файл, содержащий условие формирования ошибок.

*Строка*\
Приблизительная строка, в которой существует условие ошибки.

*Error_type*\
Неустранимая ошибка, ошибка или предупреждение.

\ *кода*
Уникальный 5-или 6-значный код ошибки.

*Message_text*\
Краткое и общее описание условия ошибки.
