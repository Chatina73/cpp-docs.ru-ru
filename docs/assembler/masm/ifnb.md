# <a name="ifnb"></a>IFNB

Предоставляет сборку, если *textitem* не является пустым.

## <a name="syntax"></a>Синтаксис

> IFNB textitem

## <a name="remarks"></a>Примечания

См. в разделе [IF](../../assembler/masm/if-masm.md) сложный синтаксис.
