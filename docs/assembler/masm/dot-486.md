# <a name="486-32-bit-masm"></a>.486 (32-разрядный компилятор MASM)

Включает сборку непривилегированных инструкций для процессора 80486. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.486**
