# <a name="operator-dup"></a>Оператор DUP

Указывает *количество* объявлений *инитиалвалуе*.

## <a name="syntax"></a>Синтаксис

> *Count* **DUP** __(__*инитиалвалуе* ⟦__,__ *инитиалвалуе* ... ⟧__)__
