# <a name="errnz"></a>**.ERRNZ**

Выдает ошибку, если *выражение* имеет значение true (отличное от нуля).

## <a name="syntax"></a>Синтаксис

> **. Еррнз** *выражение* ⟦ __,__ *сообщение*⟧
