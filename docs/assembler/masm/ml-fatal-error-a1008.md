# <a name="ml-fatal-error-a1008"></a>Неустранимая ошибка ML A1008

**Непарная вложенность макроса**

Либо макрос не был завершен до конца файла, либо при обнаружении завершающей директивы [ендм](endm.md) вне блока макроса.

Одной из причин этой ошибки является пропуск точки перед [. Повторите](dot-repeat.md) или [. WHILE](dot-while.md).
