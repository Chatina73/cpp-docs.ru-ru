# <a name="operator-gt"></a>Оператор GT

Возвращает значение true (-1), если *expression1* больше *Expression2*, или возвращает false (0), если нет.

## <a name="syntax"></a>Синтаксис

> *Выражение1* **gt** *выражение2*
