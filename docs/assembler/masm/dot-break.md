# <a name="break-32-bit-masm"></a>. BREAK (32-разрядный MASM)

Создает код для завершения [. WHILE](dot-while.md) или [. Повтор](dot-repeat.md) блока, если *условие* истинно. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **. Прервать** ⟦ **.**  *Условие*if ⟧
