# <a name="operator-ptr"></a>Оператор PTR

Первый оператор заставляет *выражение* обрабатываться как имеющий указанный *тип*. Второй оператор задает указатель на *тип*.

## <a name="syntax"></a>Синтаксис

> *выражение* *типа* **ptr**
>
> \[*расстояние*]  *Тип* PTR
