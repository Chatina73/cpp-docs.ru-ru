# <a name="dd"></a>DD

Выделяет и при необходимости инициализирует двойное слово (4 байта) хранилища для каждого `initializer`. `DD` является синонимом [DWORD](../../assembler/masm/dword.md).

## <a name="syntax"></a>Синтаксис

> [[имя]] Инициализатор дд [[, инициализатор]]...

## <a name="remarks"></a>Примечания

Можно также использовать как спецификатор типа в любом является допустимым типом.
