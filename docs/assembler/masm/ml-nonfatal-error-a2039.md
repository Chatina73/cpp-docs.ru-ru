# <a name="ml-nonfatal-error-a2039"></a>Некритичная ошибка ML A2039

**слишком длинная строка**

В строке исходного файла превышено ограничение в 512 символов.

Если несколько физических строк объединяются с символом продолжения строки (\), то результирующая логическая строка по-прежнему ограничена 512 символами.
