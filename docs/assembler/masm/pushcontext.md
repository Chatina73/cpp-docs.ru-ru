# <a name="pushcontext"></a>PUSHCONTEXT

Сохраняет часть или весь текущий *контекст*: регистр сегмента предполагает, значение основания, флаги перечисления и cref, а также значения процессора или сопроцессора. *Контекст* **может принимать значение** (только 32-разрядный MASM), **основание системы счисления**, **список**, **ЦП** (только 32-разрядный компилятор MASM) или **все**.

## <a name="syntax"></a>Синтаксис

> *Контекст* пушконтекст
