# <a name="continue-32-bit-masm"></a>. ПРОДОЛЖИТЬ (32-разрядный компилятор MASM)

Создает код для перехода к началу [. WHILE](dot-while.md) или [. Повтор](dot-repeat.md) блока, если *условие* истинно. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **. ПРОДОЛЖИТе** ⟦**.** *Условие* if ⟧
