# <a name="ifdif-ifdifi"></a>IFDIF IFDIFI

Предоставляет сборку, если текстовые элементы отличаются.

## <a name="syntax"></a>Синтаксис

> **IFDIF**\[**я**] *textitem1* __,__ *textitem2*

## <a name="remarks"></a>Примечания

Если **IFDIFI** — учитывая, сравнение выполняется без учета регистра. См. в разделе [IF](../../assembler/masm/if-masm.md) сложный синтаксис.
