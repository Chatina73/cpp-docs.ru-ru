# <a name="real10"></a>REAL10

Выделяет и при необходимости инициализирует 10-байтовое число с плавающей запятой для каждого *инициализатор*.

## <a name="syntax"></a>Синтаксис

> Инициализатор REAL10 имя [[, инициализатор]]...
