# <a name="386-32-bit-masm"></a>.386 (32-разрядный компилятор MASM)

Включает сборку непривилегированных инструкций для процессора 80386; Отключает сборку инструкций, появившихся в последующих процессорах. (только 32-разрядный MASM.)

## <a name="syntax"></a>Синтаксис

> **.386**

## <a name="remarks"></a>Комментарии

Также включает инструкции 80387.
