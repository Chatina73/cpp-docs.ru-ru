# <a name="operator-le"></a>Оператор LE

Возвращает значение true (-1), если *expression1* меньше или равно *Expression2*, или возвращает false (0), если нет.

## <a name="syntax"></a>Синтаксис

> *Выражение1* **Le** *выражение2*
