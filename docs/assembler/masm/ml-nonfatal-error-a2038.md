# <a name="ml-nonfatal-error-a2038"></a>Некритичная ошибка ML A2038

**Отсутствует операнд для оператора макроопределения**

Ассемблер нашел конец списка параметров макроса сразу после [!](operator-logical-not-masm-run-time.md) [%](operator-percent.md)оператор OR.
