# <a name="elseif"></a>ELSEIF

Объединяет [else](else-masm.md) и [If](if-masm.md) в одну инструкцию.

## <a name="syntax"></a>Синтаксис

> **ELSEIF** *constantExpression*\
> *инструкции*\
> ⟦**Else**\
> *else-Операторы*⟧ \
> **ENDIF**
