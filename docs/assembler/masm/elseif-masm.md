# <a name="elseif"></a>ELSEIF

Объединяет [else](else-masm.md) и [If](if-masm.md) в одну инструкцию.

## <a name="syntax"></a>Синтаксис

> \ **ELSEIF** *constantExpression*
> *инструкции*\
> ⟦**ELSE**\
> *else-Операторы*⟧ \
> **ENDIF**

## <a name="remarks"></a>Заметки

Дополнительные [сведения см. в разделе](if-masm.md) .
