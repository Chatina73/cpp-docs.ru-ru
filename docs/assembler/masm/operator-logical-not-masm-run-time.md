# <a name="operator--masm-run-time"></a>operator ! (Время выполнения MASM)

Логическое отрицание. Используется только в [. Если](dot-if.md), [. WHILE](dot-while.md)или [. ПОВТОРЯЮЩИеся](dot-repeat.md) блоки и оцениваются во время выполнения, а не во время сборки.

## <a name="syntax"></a>Синтаксис

> **!** *expression*
