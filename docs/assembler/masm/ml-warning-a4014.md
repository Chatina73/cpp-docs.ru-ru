# <a name="ml-warning-a4014"></a>Предупреждение ML A4014

инструкции и инициализированные данные не поддерживаются в сегментах BSS

Предпринята попытка определить инициализированные данные в разделе BSS.  Раздел BSS определяется как класс, имя которого — BSS.  Сюда входит упрощенный сегмент `.data?`.
