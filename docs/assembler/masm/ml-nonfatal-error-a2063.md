# <a name="ml-nonfatal-error-a2063"></a>Некритичная ошибка ML A2063

**может выравняться только степени 2: выражение**

Выражение, указанное с помощью директивы [aligned](align-masm.md) , недопустимо.

Выражение **выравнивания** должно быть степенью 2 в диапазоне от 2 до 256 и должна быть меньше или равна выравниванию текущего сегмента, структуры или объединения.
