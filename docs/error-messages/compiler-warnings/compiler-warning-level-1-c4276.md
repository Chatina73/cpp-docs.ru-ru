# <a name="compiler-warning-level-1-c4276"></a>Предупреждение компилятора (уровень 1) C4276

"функция": прототип не предоставлен; предполагается отсутствие параметров

Когда вы принимаете адрес функции, используя соглашение о вызовах [__stdcall](../../cpp/stdcall.md) , необходимо предоставить прототип, чтобы компилятор мог создать декорированное имя функции. Поскольку у *функции* нет прототипа, компилятор при создании декорированного имени предполагает, что функция не имеет параметров.
