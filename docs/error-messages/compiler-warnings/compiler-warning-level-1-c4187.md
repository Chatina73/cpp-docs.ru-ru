# <a name="compiler-warning-level-1-c4187"></a>Предупреждение компилятора (уровень 1) C4187

\#атрибуты импорта "атрибут1" и "атрибут2" несовместимы; Оба пропущены

Оператор [#import](../../preprocessor/hash-import-directive-cpp.md) задает атрибуты `no_implementation` и `implementation_only` . Оба атрибута игнорируются.
