# <a name="compiler-warning-level-1-c4834"></a>Предупреждение компилятора (уровень 1) порог предупреждения c4834

> Отмена возвращаемого значения функции с атрибутом "noreturn"

## <a name="remarks"></a>Замечания

Начиная с стандарта C++ 17, `[[nodiscard]]` атрибут указывает, что возвращаемое значение функции не предназначено для удаления. Если вызывающий объект отклоняет возвращаемое значение, компилятор создает предупреждение порог предупреждения c4834.

Чтобы устранить это предупреждение, определите, почему код не использует возвращаемое значение. Использование функции может не соответствовать ее намерениям. Вы можете обойти это предупреждение, используя приведение к **`void`** .

Это предупреждение было введено в Visual Studio 2017 версии 15,3 как предупреждение уровня 3. Оно было изменено на предупреждение уровня 1 в Visual Studio 2017 версии 15,7. Код, скомпилированный без предупреждений в версиях компилятора перед тем, как Visual Studio 2017 версии 15,3, теперь может создавать порог предупреждения c4834. Сведения о том, как отключить предупреждения, появившиеся в определенной версии компилятора или более поздней версии, см. в разделе [предупреждения компилятора по версиям компилятора](compiler-warnings-by-compiler-version.md).

### <a name="to-turn-off-the-warning-without-code-changes"></a>Отключение предупреждения без изменения кода

Предупреждение для определенной строки кода можно отключить с помощью директивы pragma [warning](../../preprocessor/warning.md) , `#pragma warning(suppress : 4034)` . Вы также можете отключить предупреждение в файле с помощью директивы pragma warning, `#pragma warning(disable : 4034)` . Предупреждение можно отключить глобально в сборках командной строки с помощью **`/wd4034`** параметра командной строки.

Чтобы отключить предупреждение для всего проекта в интегрированной среде разработки Visual Studio, сделайте следующее:

1. Откройте диалоговое окно **страницы свойств** для проекта. Сведения об использовании диалогового окна страницы свойств см. в разделе [страницы свойств](../../build/reference/property-pages-visual-cpp.md).
1. Выберите страницу " **Свойства конфигурации**  >  **C/C++**  >  **Advanced** ".
1. Измените свойство **Отключить определенные предупреждения** , чтобы добавить *`4034`* . Нажмите кнопку **ОК** , чтобы применить изменения.

## <a name="example"></a>Пример

В этом примере создается порог предупреждения c4834 и демонстрируется способ исправления:

```cpp
// C4834.cpp
// compile using: cl /EHsc /std:c++17
#include <iostream>

[[nodiscard]]
int square_of(int i) { return i * i; }

int main()
{
    square_of(42); // warning C4834: discarding return value of function with 'nodiscard' attribute
    // to fix, make use of the return value, as shown here:
    // std::cout << "square_of(42) = " << square_of(42) << "\n";
    return 0;
}
```
