# <a name="compiler-warning-level-1-c4506"></a>Предупреждение компилятора (уровень 1) C4506

нет определения для встроенной функции "функция"

Данная функция объявлена и помечена для встраивания, но не определена.

Компилятор не выполнил встраивание функции.

Убедитесь, что внешние функции, которые должны быть встроены, объявляются с помощью ключевого слова `extern`.
