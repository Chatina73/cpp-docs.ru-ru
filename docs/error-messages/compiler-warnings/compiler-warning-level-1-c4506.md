# <a name="compiler-warning-level-1-c4506"></a>Предупреждение компилятора (уровень 1) C4506

нет определения для подставляемой функции «function»

Заданная функция был объявлен и помечено для встраиванию, но не был определен.

Компилятор был встроенную функцию.

Убедитесь, что внешние функции для встраивания, объявляются с `extern` ключевое слово.
