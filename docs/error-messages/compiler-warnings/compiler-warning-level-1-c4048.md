# <a name="compiler-warning-level-1-c4048"></a>Предупреждение компилятора (уровень 1) C4048

разные индексы объявленного массива: "идентификатор1" и "идентификатор2"

Выражение включает указатели на массивы разного размера. Указатели используются без преобразования.

Это предупреждение можно исправить при явном приведении массивов к тому же или к эквивалентному типу.
