# <a name="compiler-warning-level-1-c4340"></a>Предупреждение компилятора (уровень 1) C4340

"значение": значение упаковано из положительного числа в отрицательное

**`enum`** Значение больше самого крупного **`enum`** положительного значения, упакованного в отрицательное значение.
