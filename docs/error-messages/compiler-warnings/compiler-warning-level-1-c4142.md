# <a name="compiler-warning-level-1-c4142"></a>Предупреждение компилятора (уровень 1) C4142

мягкое переопределение типа

Тип переопределяется способом, который не влияет на созданный код.

Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

- Функция-член производного класса имеет другой тип возвращаемого значения от соответствующей функции-члена базового класса.

- Тип, определенный с помощью команды `typedef`, переопределяется с использованием другого синтаксиса.

Следующий пример приводит к возникновению ошибки C4142:

```c
// C4142.c
// compile with: /W1
float X2;
X2 = 2.0 + 1.0;   // C4142

int main() {
   float X2;
   X2 = 2.0 + 1.0;   // OK
}
```
