# <a name="compiler-warning-level-1-c4375"></a>Предупреждение компилятора (уровень 1) C4375

метод "Method2", не являющийся общим, не переопределяет "Method1"

Тип, реализующий другой тип, определенный метод переопределения, но переопределение не было открытым. Поэтому метод не переопределяет метод базового типа.
