# <a name="compiler-warning-level-1-c4411"></a>Предупреждение компилятора (уровень 1) C4411

«Идентификатор»: символ соответствует регистру смещения

Идентификатор является локальным символом, который соответствует регистру смещения и поэтому может использоваться в операнде с другим символом.
