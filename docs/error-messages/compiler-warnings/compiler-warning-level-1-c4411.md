# <a name="compiler-warning-level-1-c4411"></a>Предупреждение компилятора (уровень 1) C4411

"идентификатор": символ разрешается в регистр смещения

Идентификатор является локальным символом, который разрешается в регистр смещения и поэтому может использоваться в операнде с другим символом.
