# <a name="compiler-warning-level-1-c4015"></a>Компилятор предупреждение (уровень 1) C4015

«Идентификатор»: тип битового поля должны быть целочисленными

Битовое поле не объявлен как целочисленный тип. Компилятор предполагает, что базовый тип битового поля без знака. Битовые поля должны объявляться как типы целого числа без знака.
