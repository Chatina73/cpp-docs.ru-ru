# <a name="compiler-warning-level-1-c4811"></a>Предупреждение компилятора (уровень 1) C4811

value of pragma conform(forScope, show) == value

Это предупреждение выдается при использовании параметра **show** прагмы [conform](../../preprocessor/conform.md) . *value* — это текущее значение conform.
