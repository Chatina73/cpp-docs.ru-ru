# <a name="compiler-warning-level-1-c4379"></a>Предупреждение компилятора (уровень 1) C4379

Версия среды CLR не поддерживается этим компилятором. Использование этой версии может привести к непредвиденным результатам.

На компьютере установлена предыдущая версия среды CLR, но не текущая версия. Чтобы разрешить C4379, установите версию среды CLR, которая поставляется вместе с компилятором.
