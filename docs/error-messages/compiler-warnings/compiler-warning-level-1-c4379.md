# <a name="compiler-warning-level-1-c4379"></a>Предупреждение компилятора (уровень 1) C4379

Версии версии среда CLR не поддерживается на этом компьютере. С помощью этой версии может привести к непредвиденным результатам.

У вас есть предыдущую версию общеязыковой среды выполнения на компьютере, но не текущей версии. Чтобы устранить C4379, установите версию среды CLR, которая поставляется с вашим компилятором.
