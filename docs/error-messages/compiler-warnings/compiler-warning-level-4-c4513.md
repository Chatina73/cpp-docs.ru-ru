# <a name="compiler-warning-level-4-c4513"></a>Предупреждение компилятора (уровень 4) C4513

"класс": не удалось создать деструктор

Компилятору не удается создать деструктор по умолчанию для данного класса; деструктор не создан. Деструктор находится в базовом классе, который недоступен для производного класса. Если базовый класс имеет закрытый деструктор, сделайте его открытым или защищенным.
