# <a name="compiler-warning-level-3-c4306"></a>Предупреждение компилятора (уровень 3) C4306

> "*идентификатор*": преобразование "*тип1*" в "*тип2*" большего размера

Идентификатор является приведением типа к большему указателю. Незаполненные старшие биты нового типа будут заполнены нулем.

Это предупреждение может указывать на нежелательное преобразование. Полученный указатель может быть недопустимым.
