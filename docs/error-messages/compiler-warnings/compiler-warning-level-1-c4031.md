# <a name="compiler-warning-level-1-c4031"></a>Предупреждение компилятора (уровень 1) C4031

второй список формальных параметров длиннее первого

Функция повторно объявляется с другими формальными параметрами. Компилятор использует формальные параметры в первом объявлении.
