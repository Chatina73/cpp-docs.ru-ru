# <a name="compiler-warning-level-1-c4049"></a>Предупреждение компилятора (уровень 1) C4049

ограничение компилятора: завершение эмиссии номера строки

Файл содержит более 16 777 215 (2<sup>24</sup>-1) исходных строк. Компилятор останавливает нумерацию в 16 777 215.

Для кода после строки 16 777 215:

- Образ не будет содержать отладочную информацию для номеров строк.

- Некоторые диагностические данные могут быть зарегистрированы с неверными номерами строк.

- Списки ASM (/FAs) могут содержать неправильные номера строк.
