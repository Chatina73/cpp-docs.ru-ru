# <a name="compiler-warning-level-4-c4132"></a>Предупреждение компилятора (уровень 4) C4132

"объект": требуется инициализация константного объекта

Константа не инициализирована. Значение символа с **`const`** атрибутом не может быть изменено после его определения, ему должно быть присвоено значение инициализации.
