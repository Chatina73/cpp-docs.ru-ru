# <a name="compiler-warning-level-1-c4116"></a>Компилятор предупреждение (уровень 1) C4116

Определение именованного типа в круглых скобках

Структуры, объединения или перечисляемого типа без имени определяется в выражении в скобках. Определение типа не имеет смысла.

В вызове функции C определение имеет глобальную область. В вызове функции C++ определение имеет той же области, что вызываемой функции.
