# <a name="compiler-warning-level-4-c4710"></a>Предупреждение компилятора (уровень 4) C4710

«функция»: функция не является встроенной

Заданная функция была выбрана для подстановки, но компилятор не выполняет встраивание.

Встраивание кода выполняется по своему усмотрению компилятора. **Встроенного** ключевое слово, например **зарегистрировать** слова используются как подсказку для компилятора. Компилятор использует эвристику для определения, если его следует подстановку определенной функции для ускорения в коде, скорость компиляции, или подстановку определенной функции для уменьшения код при компиляции для пространства. Компилятор будет только встроенные функции малого минимального объема.

В некоторых случаях компилятор будет не подставляет определенную функцию по техническим причинам. См. в разделе [C4714](../../error-messages/compiler-warnings/compiler-warning-level-4-c4714.md) список причин, компилятор может использовать функцию как встроенную.

Это предупреждение отключено по умолчанию. Подробнее: [Выключенные по умолчанию предупреждения компилятора](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
