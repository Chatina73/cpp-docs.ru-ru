# <a name="compiler-warning-level-3-c4511"></a>Предупреждение компилятора (уровень 3) C4511

«класс»: не удалось создать конструктор копии

Компилятору не удалось создать конструктора копии по умолчанию для класса; базовый класс может иметь закрытый конструктор копии.
