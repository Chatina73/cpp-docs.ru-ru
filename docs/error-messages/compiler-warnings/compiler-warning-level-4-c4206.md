# <a name="compiler-warning-level-4-c4206"></a>Предупреждение компилятора (уровень 4) C4206

**использовано нестандартное расширение: преобразования пуст**

Файл был пустым после предварительной обработки.

Это расширение может помешать кода в другие компиляторы. Он формирует ошибку совместимости ANSI ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) и применяется только к исходному коду C.
