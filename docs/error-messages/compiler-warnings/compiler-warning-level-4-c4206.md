# <a name="compiler-warning-level-4-c4206"></a>Предупреждение компилятора (уровень 4) C4206

**использовано нестандартное расширение: пустая единица преобразования**

Файл был пуст после предварительной обработки.

Это расширение может препятствовать переносу кода в другие компиляторы. Она создает ошибку в режиме совместимости с ANSI ([/Za](../../build/reference/za-ze-disable-language-extensions.md)) и применима только к исходному коду на языке C.
