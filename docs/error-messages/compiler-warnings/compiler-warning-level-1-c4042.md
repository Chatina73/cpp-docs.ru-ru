# <a name="compiler-warning-level-1-c4042"></a>Предупреждение компилятора (уровень 1) C4042

"идентификатор": имеет недопустимый класс хранения

Указанный класс хранения не может использоваться с этим идентификатором в этом контексте. Вместо этого компилятор использует класс хранения по умолчанию:

- `extern`, если *идентификатор* является функцией.

- **Auto**, если *идентификатор* является формальным параметром или локальной переменной.

- Отсутствует класс хранения, если *идентификатор* является глобальной переменной.

Это предупреждение может быть вызвано указанием класса хранилища, отличного от **Register** , в объявлении параметра.

Следующий пример приводит к возникновению ошибки C4042

```cpp
// C4042.cpp
// compile with: /W1 /LD
int func2( __declspec( thread ) int tls_i )    // C4042
// try the following line instead
// int func2( int tls_i )
{
   return tls_i;
}
```
