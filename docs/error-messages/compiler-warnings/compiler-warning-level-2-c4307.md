# <a name="compiler-warning-level-2-c4307"></a>Предупреждение компилятора (уровень 2) C4307

"оператор": переполнение целочисленной константы

Оператор используется в выражении, результатом которого является целочисленная константа, которая переполняет пространство, выделенное для него. Для константы может потребоваться использовать больший тип. **Целое число со знаком** содержит меньшее значение, чем `unsigned int`, так как для представления знака **целочисленный тип со знаком** использует один бит.

Следующий пример приводит к возникновению ошибки C4307:

```cpp
// C4307.cpp
// compile with: /W2
int i = 2000000000 + 2000000000;   // C4307
int j = (unsigned)2000000000 + 2000000000;   // OK

int main()
{
}
```
