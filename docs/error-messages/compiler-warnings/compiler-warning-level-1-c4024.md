# <a name="compiler-warning-level-1-c4024"></a>Предупреждение компилятора (уровень 1) C4024

"функция": разные типы для формального и фактического параметра "Number"

Соответствующие формальные и фактические параметры имеют разные типы. Компилятор передает фактический параметр без изменений. Принимающая функция преобразует тип параметра в ожидаемый тип.
