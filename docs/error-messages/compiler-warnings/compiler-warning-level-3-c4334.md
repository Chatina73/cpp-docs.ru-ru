# <a name="compiler-warning-level-3-c4334"></a>Предупреждение компилятора (уровень 3) C4334

"оператор": результат 32-разрядного сдвига неявным образом преобразуется в 64 бит (предполагалось 64-разрядный сдвиг)

Результат 32-разрядного сдвига неявно преобразован в 64-бит, и компилятор подозревает, что предполагался переход на более 64-разрядный сдвиг.  Чтобы устранить это предупреждение, используйте 64-разрядный сдвиг или явно приведите результат сдвига к 64-bit.

## <a name="example"></a>Пример

Следующий пример приводит к возникновению ошибки C4334.

```cpp
// C4334.cpp
// compile with: /W3 /c
void SetBit(unsigned __int64 *p, int i) {
   *p |= (1 << i);   // C4334
   *p |= (1i64 << i);   // OK
}
```
