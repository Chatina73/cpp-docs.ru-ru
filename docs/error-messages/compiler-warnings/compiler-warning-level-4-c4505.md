# <a name="compiler-warning-level-4-c4505"></a>Предупреждение компилятора (уровень 4) C4505

«функция»: неиспользованная локальная функция была удалена

Указанная функция является локальной и не используется в теле модуля. Таким образом функция является неиспользуемым кодом.

Компилятор не удалось создать код для этой неиспользуемой функции.
