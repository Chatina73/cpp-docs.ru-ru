# <a name="compiler-warning-level-4-c4931"></a>Предупреждение компилятора (уровень 4) C4931

> Предполагается, что библиотека типов была построена для указателей на *число*битов

Не указана явная информация с атрибутом **ptrsize** директивы [#import](../../preprocessor/hash-import-directive-cpp.md) ; Компилятор завершил, что размер указателя библиотеки типов — *Number*.

Это предупреждение отключено по умолчанию. Подробнее: [Выключенные по умолчанию предупреждения компилятора](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
