# <a name="compiler-warning-level-1-c4098"></a>Предупреждение компилятора (уровень 1) C4098

"функция": функция void, возвращающая значение

Функция, объявленная с типом возвращаемого значения [void](../../cpp/void-cpp.md) **`return`** , имеет оператор, возвращающий значение. Компилятор предполагает, что функция возвращает значение типа **`int`** .
