# <a name="compiler-warning-level-1-c4502"></a>Предупреждение компилятора (уровень 1) C4502

"спецификация компоновки" требует использования ключевого слова "extern" и должна предшествовать всем другим спецификаторам

Компоновка была указана без **`extern`** ключевого слова. Компоновка не относится к типам, не являющимся внешними.

Компилятор принял **`extern`** ключевое слово.
