# <a name="compiler-warning-level-2-c4007"></a>Предупреждение компилятора (уровень 2) C4007

"функция": должен быть "атрибут"

Обязательный атрибут для функции не указан явно. Например, функция **Main** должна иметь **`__cdecl`** атрибут. Компилятор вызывает принудительное выполнение атрибута.
