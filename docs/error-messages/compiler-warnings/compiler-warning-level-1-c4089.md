# <a name="compiler-warning-level-1-c4089"></a>Предупреждение компилятора (уровень 1) C4089

"функция": различные типы в фактическом параметре "номер", формальный параметр "номер"

Соответствующие формальные и фактические параметры имеют разные типы. Фактический параметр передается без изменений. Функция приводит фактический параметр к типу, указанному в определении функции.
