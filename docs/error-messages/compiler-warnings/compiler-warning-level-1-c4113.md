# <a name="compiler-warning-level-1-c4113"></a>Предупреждение компилятора (уровень 1) C4113

"идентификатор1" отличается в списках параметров от "идентификатор2"

Указатель на функцию назначен другому указателю на функцию, но списки формальных параметров функций не согласуются. Присваивание компилируется без изменения.
