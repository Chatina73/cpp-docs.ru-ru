# <a name="compiler-warning-level-1-c4113"></a>Компилятор предупреждение (уровень 1) C4113

«идентификатор1» отличается в списках параметров от «идентификатор2»

Указатель функции будет назначен другой указатель на функцию, но не согласны списки формальных параметров функции. Присвоение компилируется без изменений.
