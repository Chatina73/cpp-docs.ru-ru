# <a name="compiler-warning-level-1-c4403"></a>Предупреждение компилятора (уровень 1) C4403

Недопустимый оператор PTR

Оператор PTR используется ненадлежащим образом во встроенном коде ассемблера.

Компилятор проигнорировал оператор PTR.
