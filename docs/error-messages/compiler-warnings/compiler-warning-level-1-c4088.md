# <a name="compiler-warning-level-1-c4088"></a>Предупреждение компилятора (уровень 1) C4088

"функция": несоответствие указателя в фактическом параметре "номер", формальный параметр "номер"

Соответствующие формальные и фактические параметры имеют другой уровень косвенного обращения. Фактический параметр передается без изменений. Вызываемая функция интерпретирует свое значение как указатель.
