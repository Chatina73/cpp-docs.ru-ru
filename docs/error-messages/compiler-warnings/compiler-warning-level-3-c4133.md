# <a name="compiler-warning-level-3-c4133"></a>Предупреждение компилятора (уровень 3) C4133

"тип": несовместимые типы — из "тип1" в "тип2"

Это предупреждение может быть вызвано попыткой вычесть два указателя различных типов.

Чтобы избежать этого предупреждения, укажите соответствующее приведение типов.
