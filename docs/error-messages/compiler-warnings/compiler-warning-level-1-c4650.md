# <a name="compiler-warning-level-1-c4650"></a>Предупреждение компилятора (уровень 1) C4650

отладочная информация не находится в предкомпилированном заголовке; будут доступны только глобальные символы из заголовка

Файл предкомпилированного заголовка не был скомпилирован с символьной отладочной информацией Майкрософт.

При связывании результирующий исполняемый файл или библиотека динамической компоновки не будут включать отладочную информацию для локальных символов, содержащихся в предкомпилированном заголовке.

Это предупреждение можно избежать путем перекомпиляции файла предкомпилированного заголовка с помощью параметра командной строки [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md) .
