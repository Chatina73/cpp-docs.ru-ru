# <a name="compiler-warning-level-4-c4714"></a>Предупреждение компилятора (уровень 4) C4714

Функция "функция", помеченная как __forceinline, не является встроенной

Данная функция была выбрана для встроенного расширения, но компилятор не выполнил встраивание.

Хотя `__forceinline` является более сильным указанием компилятора, чем `__inline`, встраивание по-прежнему выполняется по усмотрению компилятора, но ни один из эвристических методов не используется для определения преимуществ встраивания этой функции.

В некоторых случаях компилятор не будет встроеет определенную функцию по механическим причинам. Например, компилятор не будет встроен:

- Функция, если она будет смешивать и SEH, и C++ EH.

- Некоторые функции с копированием созданных объектов, переданных по значению, если параметр-GX/EHs/ЕХА имеет значение ON.

- Функции, возвращающие недоступный для заполнения объект по значению, если параметр-GX/EHs/ЕХА имеет значение ON.

- Функции со встроенной сборкой при компиляции без-Og/Ox/O1/O2.

- Функции с переменным списком аргументов.

- Функция с оператором **try** (C++ обработка исключений).

Следующий пример приводит к возникновению ошибки C4714:

```cpp
// C4714.cpp
// compile with: /Ob1 /GX /W4
__forceinline void func1()
{
   try
   {
   }
   catch (...)
   {
   }
}

void func2()
{
   func1();   // C4714
}

int main()
{
}
```
