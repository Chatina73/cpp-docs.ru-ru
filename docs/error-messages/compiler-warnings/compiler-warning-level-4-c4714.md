# <a name="compiler-warning-level-4-c4714"></a>Предупреждение компилятора (уровень 4) C4714

функция «функция», помеченная как __forceinline, не встроена

Заданная функция была выбрана для подстановки, но компилятор не выполняет встраивание.

Несмотря на то что `__forceinline` является сильным указанием компилятору, чем `__inline`, встраивание выполняется на усмотрение компилятора, но эвристики для определения преимуществ встраивания этой функции.

В некоторых случаях компилятор будет не подставляет определенную функцию по техническим причинам. Например компилятор не будет подставлять:

- Функция, если это приведет к смешивание SEH и C++ EH.

- Некоторые функции с копиями созданных объектов, передаваемых по значению, если включен - GX/EHs/EHa.

- Функции, возвращающие объект нераскручиваемых по значению, если включен - GX/EHs/EHa.

- Функций во встроенной сборке при компиляции без - Og/Ox/O1/O2.

- Функции с переменное число аргументов.

- Функция с **попробуйте** инструкции (обработку исключений C++).

Следующий пример приводит к возникновению ошибки C4714:

```
// C4714.cpp
// compile with: /Ob1 /GX /W4
__forceinline void func1()
{
   try
   {
   }
   catch (...)
   {
   }
}

void func2()
{
   func1();   // C4714
}

int main()
{
}
```
