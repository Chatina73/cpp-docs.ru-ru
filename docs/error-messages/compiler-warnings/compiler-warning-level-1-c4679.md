# <a name="compiler-warning-level-1-c4679"></a>Предупреждение компилятора (уровень 1) C4679

> "*член*": не удалось импортировать элемент

Компилятор обнаружил конструкцию, которая не может поддерживаться, и не может быть импортирована из метаданных.

Не пытайтесь использовать конструкцию.
