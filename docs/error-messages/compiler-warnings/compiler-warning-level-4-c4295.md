# <a name="compiler-warning-level-4-c4295"></a>Предупреждение компилятора (уровень 4) C4295

> "*массив*": массив слишком мал для включения завершающего нуль-символа

Массив был инициализирован, но последний символ в массиве не равен null; доступ к массиву в виде строки может привести к непредвиденным результатам.

## <a name="example"></a>Пример

Следующий пример приводит к возникновению ошибки C4295. Чтобы устранить эту проблему, можно объявить размер массива больше, чтобы вместить завершающее значение NULL из строки инициализатора, или же можно использовать список инициализаторов массивов, чтобы сделать цель ясной, что это массив **`char`** , а не строка, завершающаяся нулем.

```C
// C4295.c
// compile with: /W4

int main() {
   char a[3] = "abc";           // C4295
   char b[3] = {'d', 'e', 'f'}; // No warning
   a[0] = b[2];
}
```
