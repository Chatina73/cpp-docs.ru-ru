# <a name="compiler-warning-level-1-c4027"></a>Предупреждение компилятора (уровень 1) C4027

> функция объявлена без списка формальных параметров

В объявлении функции не было формальных параметров, но в определении функции или фактических параметрах в вызове есть формальные параметры.

Компилятор принимает, но предупреждает о старом объявлении прямого объявления имени функции в стиле C без списка параметров. При последующих вызовах этой функции компилятор предполагает, что функция принимает фактические параметры типов, найденных в определении или вызове функции. Рекомендуется изменить объявление функции в соответствии с сигнатурой определения функции.
