# <a name="compiler-warning-level-1-c4344"></a>Предупреждение компилятора (уровень 1) C4344

изменение поведения: использование явных аргументов шаблона приводит к вызову функции

Вызов функции, использующей явные аргументы шаблона, вызывает другую функцию, чем если бы явно не были заданы явные аргументы
