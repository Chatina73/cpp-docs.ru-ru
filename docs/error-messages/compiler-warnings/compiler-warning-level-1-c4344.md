# <a name="compiler-warning-level-1-c4344"></a>Предупреждение компилятора (уровень 1) C4344

Изменение поведения: использование аргументов шаблона приведет к вызову «функции»

Вызов функции с помощью явно заданных аргументов шаблона вызывает другую функцию, чем если бы не был задан явно заданные аргументы
