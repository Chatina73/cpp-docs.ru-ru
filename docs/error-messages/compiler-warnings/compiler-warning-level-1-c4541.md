# <a name="compiler-warning-level-1-c4541"></a>Предупреждение компилятора (уровень 1) C4541

"идентификатор" используется в типе "тип" с/GR-; может возникнуть непредсказуемое поведение

Вы попытались использовать функцию, для которой требуются сведения о типе во время выполнения, без включения сведений о типах времени выполнения. Перекомпилируйте с [/GR](../../build/reference/gr-enable-run-time-type-information.md).
