# <a name="compiler-warning-level-1-c4029"></a>Предупреждение компилятора (уровень 1) C4029

объявленный список формальных параметров отличается от определения

Формальные типы параметров в объявлении функции не согласуются с ними в определении функции. Компилятор использует список параметров из определения.
