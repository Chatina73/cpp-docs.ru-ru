# <a name="compiler-warning-level-1-c4540"></a>Предупреждение компилятора (уровень 1) C4540

dynamic_cast, используемый для преобразования в недоступный или неоднозначный базовый; тест времени выполнения завершится ошибкой ("тип1" в "тип2")

Вы использовали **`dynamic_cast`** для преобразования из одного типа в другой. Компилятор определил, что приведение всегда завершится ошибкой (Возвращает **значение NULL**), так как базовый класс недоступен ( **`private`** например,) или неоднозначен (встречается более одного раза в иерархии классов).

Ниже приведен пример этого предупреждения. Класс **B** является производным от класса **A**. Программа использует **`dynamic_cast`** для приведения из класса **b** (производный класс) к классу **A**, который всегда завершится ошибкой, так как класс **B** является **`private`** недоступным. Изменение доступа **к для** **`public`** позволит устранить это предупреждение.

```cpp
// C4540.cpp
// compile with: /W1

struct A {
   virtual void g() {}
};

struct B : private A {
   virtual void g() {}
};

int main() {
   B b;
   A * ap = dynamic_cast<A*>(&b);   // C4540
}
```
