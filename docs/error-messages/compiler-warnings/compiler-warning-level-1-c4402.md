# <a name="compiler-warning-level-1-c4402"></a>Предупреждение компилятора (уровень 1) C4402

необходимо использовать оператор PTR

Тип используется для операнда без оператора PTR при обращении или преобразовании к типу в встроенный код ассемблера.

Компилятор предполагает, что оператор PTR.
