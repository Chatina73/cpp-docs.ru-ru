# <a name="compiler-warning-level-1-c4264"></a>Предупреждение компилятора (уровень 1) C4264

"virtual_function": нет доступных переопределений для виртуальной функции члена из базового класса "класс"; функция скрыта

C4264 всегда создается после [C4263](../../error-messages/compiler-warnings/compiler-warning-level-4-c4263.md).

Это предупреждение отключено по умолчанию. Подробнее: [Выключенные по умолчанию предупреждения компилятора](../../preprocessor/compiler-warnings-that-are-off-by-default.md) .
