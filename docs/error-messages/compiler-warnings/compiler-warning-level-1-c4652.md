# <a name="compiler-warning-level-1-c4652"></a>Предупреждение компилятора (уровень 1) C4652

параметр компилятора "параметр" несовместим с предкомпилированным заголовком; текущий параметр командной строки переопределяет значение, заданное в предкомпилированном заголовке

Данный параметр командной строки отличается от указанного в момент создания предкомпилированного заголовка (PCH). Использовался параметр, указанный в текущей командной строке.

Это предупреждение можно избежать путем повторного создания предкомпилированного заголовка с заданным параметром командной строки.
