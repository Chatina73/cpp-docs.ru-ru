# <a name="compiler-warning-level-1-c4381"></a>Предупреждение компилятора (уровень 1) C4381

"функция1": метод интерфейса не будет реализован методом "функция2", не являющимся общим

Класс должен реализовывать всю функцию в интерфейсе. Класс может удовлетворить этому условию, если один из его базовых классов реализует функцию. Однако функция должна быть реализована как открытая функция.
