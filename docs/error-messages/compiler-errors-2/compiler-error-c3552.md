# <a name="compiler-error-c3552"></a>Ошибка компилятора C3552

"имя_типа": тип возвращаемого значения с поздним заданием не может содержать "auto"

Если **`auto`** ключевое слово используется в качестве заполнителя для возвращаемого типа функции, необходимо указать тип возвращаемого значения с поздним заданным. Однако нельзя использовать другое **`auto`** ключевое слово для указания типа возвращаемого значения с поздним указанием. Например, представленный ниже фрагмент кода приводит к возникновению ошибки C3552.

`auto myFunction->auto; // C3552`
