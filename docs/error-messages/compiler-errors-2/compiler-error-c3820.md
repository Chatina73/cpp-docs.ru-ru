# <a name="compiler-error-c3820"></a>Ошибка компилятора C3820

> "*имя*": инициализаторы должны быть управляемыми

## <a name="remarks"></a>Remarks

Инициализатор *имя* для управляемого типа должен быть также управляемым. Чтобы исправить эту ошибку, укажите метод управляемого инициализатора.
