# <a name="compiler-error-c3817"></a>Ошибка компилятора C3817

"объявление": свойство может применяться только к функции

**`property`** Ключевое слово может быть применено только к определению функции.

C3817 доступен только при использовании устаревшего параметра компилятора **/clr: oldSyntax**.
