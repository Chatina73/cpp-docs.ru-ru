# <a name="compiler-error-c2830"></a>Ошибка компилятора C2830

только параметры размещения для оператора New могут иметь значения по умолчанию

Стандартные формальные параметры для [оператора New](../../standard-library/new-operators.md#op_new) не могут иметь значения по умолчанию. Только определяемые пользователем параметры размещения могут задавать значения по умолчанию.
