# <a name="compiler-error-c2568"></a>Ошибка компилятора C2568

«идентификатор1»: не удалось разрешить перегрузку функции

Компилятор не может определить, какие перегруженные функции для вызова. Фактические параметры, переданные функции должен быть приведен к соответствует формальных параметров перегруженной функции, но не одно совпадение однозначно лучше, чем все остальные.
