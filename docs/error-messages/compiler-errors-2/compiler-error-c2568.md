# <a name="compiler-error-c2568"></a>Ошибка компилятора C2568

"идентификатор1": не удается разрешить перегрузку функции

Компилятор не может определить, какую перегруженную функцию вызвать. Фактические параметры, передаваемые в функцию, должны быть приведены в соответствие с формальными параметрами одной из перегруженных функций, но ни одно совпадение однозначно не является более эффективным, чем все остальные.
