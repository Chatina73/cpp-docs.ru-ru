# <a name="compiler-error-c3832"></a>Ошибка компилятора C3832

«Библиотека типов»: библиотека типов выглядит так, как если бы оно было создано для 32-разрядных указателей; Измените квалификатор «ptrsize»

Явная информация, указанная с `ptrsize` атрибут [#import](../../preprocessor/hash-import-directive-cpp.md) директива не согласуется с то, что компилятор найдет в библиотеке типов.
