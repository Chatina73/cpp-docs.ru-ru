# <a name="compiler-error-c3832"></a>Ошибка компилятора C3832

"Библиотека типов": Библиотека типов выглядит так, как если бы она была построена для 32-разрядных указателей; Измените квалификатор "ptrsize".

Явная информация, предоставляемая `ptrsize` атрибутом директивы [#import](../../preprocessor/hash-import-directive-cpp.md) , не согласуется с тем, что обнаружено компилятором в библиотеке типов.
