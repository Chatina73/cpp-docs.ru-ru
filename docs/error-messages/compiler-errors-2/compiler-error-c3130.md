# <a name="compiler-error-c3130"></a>Ошибка компилятора C3130

Внутренняя ошибка компилятора: не удалось записать блок внедренного кода в PDB-файл

Эта ошибка возникает, если компилятору не удалось записать внедренный блок кода в PDB-файл. Наиболее распространенной причиной сбоя является нехватка места на диске.
