# <a name="compiler-error-c3610"></a>Ошибка компилятора C3610

"ValueType": тип значения должен быть "упакован", прежде чем можно будет вызвать метод "метод"

По умолчанию тип значения не находится в управляемой куче. Прежде чем можно будет вызывать методы из классов среды выполнения .NET, например `Object` , необходимо переместить тип значения в управляемую кучу.

C3610 доступен только при использовании устаревшего параметра компилятора **/clr: oldSyntax**.
