# <a name="compiler-error-c2855"></a>Ошибка компилятора C2855

параметр командной строки "параметр" несовместим с предкомпилированным заголовком

Данный параметр командной строки отличается от параметра, используемого для создания предкомпилированного заголовка. Повторное создание предкомпилированного заголовка с заданным параметром командной строки.
