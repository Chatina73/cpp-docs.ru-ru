# <a name="compiler-error-c3180"></a>Ошибка компилятора C3180

"имя типа": имя превышает ограничение метаданных в символах "Limit"

Компилятор усекает имя управляемого типа в метаданных. Усечение сделает тип непригодным для использования с `#using` директивой (или эквивалентом на другом языке).

Ограничение имени типа включает любые квалификацию пространства имен.
