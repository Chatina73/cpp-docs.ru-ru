# <a name="compiler-error-c3661"></a>Ошибка компилятора C3661

в списке явных переопределений не найдены методы для переопределения

Явное переопределение указывает одно или несколько имен типов.  Однако в типах, соответствующих сигнатуре переопределяющей функции, отсутствует функция с необходимой сигнатурой.  При попытке переопределить в зависимости от имени типа в указанных типах должны быть одна или несколько виртуальных функций, которые соответствуют сигнатуре переопределяющей функции.

Дополнительные сведения см. в разделе [явные переопределения](../../extensions/explicit-overrides-cpp-component-extensions.md).
