# <a name="compiler-error-c2592"></a>Ошибка компилятора C2592

class: base_class_2 наследуется от base_class_1 и не может быть указан повторно

Можно указать только базовые классы, которые не наследуют от других базовых классов. В данном случае в описании `base_class_1` требуется только `class`, так как `base_class_1` уже наследует `base_class_2`.
