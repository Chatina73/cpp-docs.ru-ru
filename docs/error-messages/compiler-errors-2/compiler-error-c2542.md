# <a name="compiler-error-c2542"></a>Ошибка компилятора C2542

"идентификатор": у объекта класса нет конструктора для инициализации

Отсутствует конструктор со списком параметров, который соответствует инициализации.

Эта ошибка может быть вызвана неправильными параметрами при инициализации.
