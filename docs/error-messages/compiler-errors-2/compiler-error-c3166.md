# <a name="compiler-error-c3166"></a>Ошибка компилятора C3166

> "указатель": невозможно объявить указатель на внутренний __gc указатель в качестве члена "тип"

Компилятор обнаружил недопустимое объявление указателя ( **`__nogc`** указатель на **`__gc`** указатель).

C3166 доступен только при использовании устаревшего параметра компилятора **`/clr:oldSyntax`** .
