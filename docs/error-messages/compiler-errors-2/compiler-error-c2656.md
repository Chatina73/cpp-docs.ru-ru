# <a name="compiler-error-c2656"></a>Ошибка компилятора C2656

"функция": функция не разрешена как битовое поле

Функция объявлена как член битового поля.

Эта ошибка может быть вызвана синтаксической ошибкой в списке инициализаторов конструктора.
