# <a name="compiler-error-c2656"></a>Ошибка компилятора C2656

«функция»: функция не допускается в качестве битового поля

Функция объявлена как член является битовым полем.

Эта ошибка может быть вызвана синтаксические ошибки в списке инициализаторов конструктора.
