# <a name="compiler-error-c3501"></a>Ошибка компилятора C3501

нет библиотеки типов, зарегистрированной для ProgID "ProgID"

Идентификатор класса для данного ProgID не имеет связанной библиотеки типов. Поэтому этот идентификатор ProgID нельзя передать в `#import` оператор.
