# <a name="compiler-error-c3275"></a>Ошибка компилятора C3275

"член перечисления": нельзя использовать этот символ без квалификатора

Если вы используете управляемый код, и не менее двух перечислений содержат идентификатор с одним и тем же именем, необходимо явно квалифицировать ссылки на идентификатор.

C3275 доступен только при использовании устаревшего параметра компилятора **/clr: oldSyntax**.
