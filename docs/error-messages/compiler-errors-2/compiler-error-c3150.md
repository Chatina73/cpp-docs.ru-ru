# <a name="compiler-error-c3150"></a>Ошибка компилятора C3150

«элемент»: «атрибут» может применяться только к классу, интерфейс, массив или указатель

`__gc` может использоваться только для класса, интерфейса или массива.

C3150 доступен только с помощью параметра компилятора устаревшие **/CLR: oldSyntax**.
