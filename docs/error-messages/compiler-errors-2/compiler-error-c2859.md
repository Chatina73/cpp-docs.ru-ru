# <a name="compiler-error-c2859"></a>Ошибка компилятора C2859

Имя файла не тип файла, который использовался при создании предкомпилированного заголовка, предкомпилированного заголовка.

Проект базы данных и файлы предварительно скомпилированных заголовков должны создаваться вместе, чтобы обеспечить согласованность информации. Перестройте проект для предкомпилированного заголовка.

Сведения о предкомпилированных заголовках см. в разделе [/Y (предкомпилированные заголовки)](../../build/reference/y-precompiled-headers.md).
