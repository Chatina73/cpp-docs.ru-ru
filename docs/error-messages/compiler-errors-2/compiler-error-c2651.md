# <a name="compiler-error-c2651"></a>Ошибка компилятора C2651

"тип данных": выражение слева от "operator" должно быть классом, структурой или объединением

Чтобы использовать параметр шаблона, как если бы он был классом, специализация шаблона класса с классом вместо целочисленного типа.
