# <a name="compiler-error-c2517"></a>Ошибка компилятора C2517

"идентификатор": право "::" не определено

Идентификатор справа от оператора разрешения области действия (`::`) должен быть определенным членом класса, структуры или объединения слева. Если имя класса, структуры или объединения не указано, идентификатор справа должен быть объявлен с глобальной областью видимости.
