# <a name="compiler-error-c3173"></a>Ошибка компилятора C3173

Несоответствие версий при слиянии idl

Эта ошибка происходит, когда объектный файл содержит внедренные idl, который был создан в предыдущей версии компилятора. Компилятор кодирует номер версии, что обеспечивает тот же компилятор, используемый для создания idl-содержимое, которое внедрено в OBJ-файлов также тот же компилятор, используемые при слиянии внедренные IDL-файле.

Обновление установки Visual C++, таким образом, все средства из последней версии.
