# <a name="compiler-error-c3173"></a>Ошибка компилятора C3173

несовпадение версий при слиянии idl

Эта ошибка возникает, когда объектный файл содержит внедренный IDL, созданный с помощью предыдущей версии компилятора. Компилятор кодирует номер версии, чтобы тот же компилятор, который использовался для создания содержимого IDL, внедренного в OBJ-файлы, также использовал тот же компилятор, что и для слияния внедренного IDL.

Обновите визуальную C++ установку, чтобы все средства были выпущены в последней версии.
