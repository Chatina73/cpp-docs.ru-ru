# <a name="compiler-error-c3818"></a>Ошибка компилятора C3818

объявление свойства массива "свойство1" не должно перегружать свойство индекса "свойство2"

Перегрузка невозможна для свойств, если один является индексатором, а другой — свойством массива.

C3818 доступен только при использовании устаревшего параметра компилятора **/clr: oldSyntax**.
