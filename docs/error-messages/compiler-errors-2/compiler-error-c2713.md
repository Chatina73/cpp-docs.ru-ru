# <a name="compiler-error-c2713"></a>Ошибка компилятора C2713

разрешена только одна форма обработки исключений для каждой функции

В одной и той же функции нельзя использовать структурированную обработку исключений C++ (`__try`/`__except`) и обработку исключений (`try`/`catch`).
