# <a name="compiler-error-c2670"></a>Ошибка компилятора C2670

"идентификатор": шаблон функции не может преобразовать номер параметра из типа "тип"

Не удалось преобразовать параметр в требуемый тип.

Эта ошибка может быть исправлена при создании явного преобразования.
