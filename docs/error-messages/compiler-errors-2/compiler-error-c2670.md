# <a name="compiler-error-c2670"></a>Ошибка компилятора C2670

«Идентификатор»: шаблон функции не удается преобразовать параметр с номером из типа «тип»

Не удалось преобразовать параметр в требуемый тип.

Чтобы устранить эту ошибку, создать явное преобразование.
