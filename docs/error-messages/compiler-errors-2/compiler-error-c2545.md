# <a name="compiler-error-c2545"></a>Ошибка компилятора C2545

"оператор": не удается найти перегруженный оператор

Оператор нельзя использовать с предоставленными операндами. Необходимо указать перегруженный оператор с требуемыми операндами.

Эта ошибка может быть вызвана неправильным типом операндов.

Эту ошибку можно исправить, если определить оператор преобразования или конструктор, принимающий один параметр.
