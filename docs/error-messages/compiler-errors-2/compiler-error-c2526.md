# <a name="compiler-error-c2526"></a>Ошибка компилятора C2526

"идентификатор1": функция C компоновкой не C++ может возвращать класс "идентификатор2"

Функция, определенная с помощью компоновки C, не может возвращать определяемый пользователем тип.
