# <a name="compiler-error-c2773"></a>Ошибка компилятора C2773

\#Импорт и #using доступны только в компиляторе C++

Компилятор C не распознает `#import` директиву препроцессора. Скомпилируйте исходный код как C++. При необходимости используйте значение [/TP](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) .
