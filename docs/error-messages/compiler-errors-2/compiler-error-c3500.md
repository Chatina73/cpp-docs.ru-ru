# <a name="compiler-error-c3500"></a>Ошибка компилятора C3500

недопустимый ProgID "ProgID"

В инструкции `#import` был указан недопустимый ProgID. Проверьте реестр Windows, чтобы убедиться, что указан допустимый идентификатор ProgID.
