# <a name="compiler-error-c3506"></a>Ошибка компилятора C3506

нет библиотеки типов, зарегистрированной для LIBID «id»

Библиотеки типов не был правильно зарегистрирован. Используйте regsvr32.exe для регистрации библиотеки типов.
