# <a name="compiler-error-c2588"></a>Ошибка компилятора C2588

":: ~ идентификатор": недопустимый глобальный деструктор

Деструктор определен для чего-либо, отличного от класса, структуры или объединения. Это не допускается.

Эта ошибка может быть вызвана отсутствием имени класса, структуры или объединения в левой части оператора разрешения области действия ( `::` ).

Следующий пример приводит к возникновению ошибки C2588:

```cpp
// C2588.cpp
~F();   // C2588
```
