# <a name="compiler-error-c2667"></a>Ошибка компилятора C2667

"функция": ни один из числовых перегрузок не подходит для преобразования

Вызов перегруженной функции неоднозначен и не может быть разрешен.

Преобразование, необходимое для сопоставления фактических параметров в вызове функции с одной из перегруженных функций, должно быть строго лучше, чем преобразования, необходимые для всех остальных перегруженных функций.
