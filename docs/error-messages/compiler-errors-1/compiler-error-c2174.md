# <a name="compiler-error-c2174"></a>Ошибка компилятора C2174

"функция": фактический параметр имеет тип "void": параметр Число1, список параметров, число2

Параметр `number1` передан в список параметров `number2` является параметром `void`. Параметры не могут иметь тип `void`. Используйте вместо этого `void*`.
