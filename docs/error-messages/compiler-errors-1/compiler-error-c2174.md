# <a name="compiler-error-c2174"></a>Ошибка компилятора C2174

"функция": фактический параметр имеет тип "void": параметр Число1, список параметров, число2

Параметр, `number1` переданный в список параметров `number2` , является **`void`** параметром. Параметры не могут иметь тип **`void`** . **`void*`** Вместо этого используйте.
