# <a name="fatal-error-c1075"></a>Неустранимая ошибка C1075

в конце файла обнаружен левый непарный токен

Компилятор ожидает соответствующий *токен* перед достижением конца файла.

Эта ошибка может быть вызвана несовпадающими скобками, фигурными скобками или другими парными символами.
