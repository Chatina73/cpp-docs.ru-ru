# <a name="fatal-error-c1854"></a>Неустранимая ошибка C1854

> Невозможно перезаписать данные, сформированные во время создания предкомпилированного заголовка в объектном файле: "*filename*"

После указания параметра [/Yc (создать предкомпилированный заголовочный файл)](../../build/reference/yc-create-precompiled-header-file.md) для того же файла был указан параметр [/Yu (использовать файл](../../build/reference/yu-use-precompiled-header-file.md) предкомпилированного заголовка).

Чтобы устранить эту проблему, в общем случае установите в проекте только один файл для компиляции с параметром **/Yc** и настройте для всех остальных файлов компиляцию с помощью параметра **/Yu** . Дополнительные сведения об использовании параметра **/Yc** и его настройке в интегрированной среде разработки Visual Studio см. в разделе [/Yc (создание файла предкомпилированного заголовка)](../../build/reference/yc-create-precompiled-header-file.md). Дополнительные сведения об использовании предкомпилированных заголовков см. в разделе [Создание предкомпилированных заголовочных файлов](../../build/creating-precompiled-header-files.md).
