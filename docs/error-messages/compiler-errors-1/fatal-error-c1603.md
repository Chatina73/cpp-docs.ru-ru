# <a name="fatal-error-c1603"></a>Неустранимая ошибка C1603

Цель встроенной ветви ассемблера вне диапазона на "число" байт

Вычисленное расстояние между инструкциями ЖККСЗ или ЖЕККСЗ и указанной целевой меткой превышает 128 байт. Обновите код, чтобы метка была ближе к инструкции.
