# <a name="compiler-error-c2223"></a>Ошибка компилятора C2223

выражение слева от "->identifier" должно указывать на структуру или объединение

Операнд слева от `->` не является указателем на класс, структуру или объединение.

Эта ошибка может быть вызвана левым операндом, который является неопределенной переменной (таким образом, типом **`int`** ).
