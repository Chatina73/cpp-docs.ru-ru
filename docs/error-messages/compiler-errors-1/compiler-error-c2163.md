# <a name="compiler-error-c2163"></a>Ошибка компилятора C2163

"функция": недоступна в качестве подставляемой функции

`intrinsic` `function` Директива или pragma перечисляет функцию, недоступную в встроенной форме. Например, некоторые встроенные функции недоступны при компиляции программы, использующей программирование с помощью/CLR.
