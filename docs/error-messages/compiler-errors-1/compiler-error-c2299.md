# <a name="compiler-error-c2299"></a>Ошибка компилятора C2299

"функция": изменение поведения: явная специализация не может быть конструктором копии или оператором присваивания копирования

Эта ошибка также может быть вызвана работой по согласованности компилятора, выполненной для Visual Studio 2005: предыдущие версии Visual C++ позволяли использовать явные специализации для конструктора копии или оператора присваивания копирования.

Чтобы разрешить C2299, не делайте конструктор копии или оператор присваивания функции шаблона, а вместо функции, не являющейся шаблоном, которая принимает тип класса. Любой код, вызывающий конструктор копий или оператор присваивания путем явного указания аргументов шаблона, должен удалить аргументы шаблона.

Следующий пример приводит к возникновению ошибки C2299:

```cpp
// C2299.cpp
// compile with: /c
class C {
   template <class T>
   C (T t);

   template <> C (const C&);   // C2299
   C (const C&);   // OK
};
```
