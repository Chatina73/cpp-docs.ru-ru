# <a name="fatal-error-c1904"></a>Неустранимая ошибка C1904

недопустимое взаимодействие поставщиков: "файл"

Эта ошибка указывает на сбой поставщика атрибутов.

Обратите внимание на обстоятельства ошибки, Попробуйте изолировать проблему и создать воспроизводимый тестовый случай, а затем свяжитесь [с нами](/visualstudio/ide/talk-to-us).
