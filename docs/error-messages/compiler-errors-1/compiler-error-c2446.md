# <a name="compiler-error-c2446"></a>Ошибка компилятора C2446

"оператор": нет преобразования из "тип1" в "тип2"

Компилятор не может преобразовать `type1` в `type2`. Преобразование может быть неразумным, поскольку оно нарушает C/C++ семантику.
