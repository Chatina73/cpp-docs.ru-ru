# <a name="compiler-error-c2410"></a>Ошибка компилятора C2410

"идентификатор": неоднозначное имя члена в "Context"

Идентификатор является членом более чем одной структуры или объединения в этом контексте.

Используйте структуру или описатель объединения для операнда, вызвавшего ошибку. Описатель структуры или объединения — это идентификатор типа **`struct`** или **`union`** ( **`typedef`** имя или переменная того же типа, что и структура или объединение, на которое указывает ссылка). Для использования операнда спецификатор должен быть левым операндом первого оператора выбора члена (.).
