# <a name="compiler-error-c2439"></a>Ошибка компилятора C2439

"идентификатор": не удалось инициализировать член

Не удается инициализировать класс, структуру или член объединения.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Попытка инициализировать косвенный базовый класс или структуру.

1. Попытка инициализировать наследуемый член класса или структуры. Наследуемый член должен инициализироваться конструктором класса или структуры.
