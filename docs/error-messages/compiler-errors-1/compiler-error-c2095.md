# <a name="compiler-error-c2095"></a>Ошибка компилятора C2095

"функция": фактический параметр имеет тип "void": параметр "номер"

Параметр, переданный функции, имеет тип `void`, что недопустимо. Вместо этого используйте указатель на void (`void *`).

`number` указывает, какой параметр `void`.
