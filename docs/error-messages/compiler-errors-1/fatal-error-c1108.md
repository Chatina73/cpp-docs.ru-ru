# <a name="fatal-error-c1108"></a>Неустранимая ошибка C1108

> не удается найти библиотеку DLL: "*имя DLL*"

Указанная библиотека DLL (*имя DLL*) не найдена в пути.

Чтобы устранить эту ошибку, переустановите C++ визуальный элемент или скопируйте соответствующий DLL-файл из установки на компьютер.
