# <a name="fatal-error-c1853"></a>Неустранимая ошибка C1853

> файл предкомпилированного заголовка "*filename*" создан в предыдущей версии компилятора, или предкомпилированный заголовок является C++ и используется с C (или наоборот)

Возможные причины:

- Предкомпилированный заголовок был скомпилирован с предыдущей версией компилятора. Попробуйте перекомпилировать заголовок с текущим компилятором.

- Предкомпилированный заголовок является C++ и используется в c. Попробуйте перекомпилировать заголовок для использования с c, указав один из параметров компилятора [/TC](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) , или измените суффикс исходного файла на "C". Дополнительные сведения см. в разделе [два варианта предварительной компиляции кода](../../build/creating-precompiled-header-files.md#two-choices-for-precompiling-code).
