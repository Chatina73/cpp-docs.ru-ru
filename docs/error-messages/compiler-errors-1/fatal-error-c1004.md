# <a name="fatal-error-c1004"></a>Неустранимая ошибка C1004

обнаружен непредвиденный конец файла

Компилятор достиг конца исходного файла без разрешения конструкции. В коде может отсутствовать один из следующих элементов:

- Закрывающая фигурная скобка

- Закрывающая круглая скобка

- Маркер закрывающего комментария (*/)

- Точка с запятой

Чтобы устранить эту ошибку, проверьте следующее:

- На диске по умолчанию недостаточно места для временных файлов, что требует примерно вдвое больше места, чем исходный файл.

- `#if`Директива, результатом которой является false, не имеет закрывающей `#endif` директивы.

- Исходный файл не заканчивается символом возврата каретки и перевода строки.

Следующий пример приводит к возникновению ошибки C1004:

```cpp
// C1004.cpp
#if TEST
int main() {}
// C1004
```

Возможное решение:

```cpp
// C1004b.cpp
#if TEST
#endif

int main() {}
```
