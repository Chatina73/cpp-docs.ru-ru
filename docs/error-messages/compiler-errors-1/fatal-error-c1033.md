# <a name="fatal-error-c1033"></a>Неустранимая ошибка C1033

не удается открыть PDB-файл базы данных программы

Эта ошибка может быть вызвана ошибками диска, временной блокировкой, созданной антивирусной программой, предыдущим экземпляром отладчика, который не полностью завершил работу, или параллельной сборкой mspdbsrv.exe процессами, которые пытаются получить доступ к одному и тому же файлу, среди других возможных причин.
