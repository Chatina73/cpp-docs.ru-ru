# <a name="fatal-error-c1382"></a>Неустранимая ошибка C1382

PCH-файл "file" был перестроен с момента создания "obj". Повторите построение этого объекта

При использовании [/LTCG](../../build/reference/ltcg-link-time-code-generation.md)компилятор обнаружил файл PCH, который новее, чем CIL. obj, указывающий на него. Сведения в файле CIL. obj устарели. Перестройте объект.

C1382 также может возникнуть при компиляции с параметром **/Yc**, но также при передаче в компилятор нескольких файлов исходного кода.  Чтобы устранить эту проблему, не используйте параметр **/Yc** при передаче компилятору нескольких файлов исходного кода.  Дополнительные сведения см. в разделе [/Yc (Создание предварительно скомпилированного заголовочного файла)](../../build/reference/yc-create-precompiled-header-file.md).
