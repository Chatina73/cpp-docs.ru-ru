# <a name="fatal-error-c1002"></a>Неустранимая ошибка C1002

не хватает размера кучи на 2-ом проходе компилятора

Компилятор исчерпал пространство динамической памяти во время второго прохода, возможно, из-за программы, имеющей слишком много символов или сложных выражений.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Возможные способы устранения этой ошибки

1. Разделите исходный файл на несколько файлов меньшего размера.

1. Разбейте выражения на небольшие части выражения.

1. Удалите другие программы или драйверы, использующие память.
