# <a name="compiler-error-c2018"></a>Ошибка компилятора C2018

неизвестный знак "шестнадцатеричное_число"

Исходный файл содержит непредвиденный символ ASCII, который определяется с шестнадцатеричным значением. Чтобы устранить эту ошибку, удалите символ.
