# <a name="fatal-error-c1350"></a>Неустранимая ошибка C1350

Ошибка при загрузке библиотеки DLL "DLL": Библиотека DLL не найдена

Библиотека DLL, которая поддерживает предпринятую операцию, не найдена.

Это указывает на проблему с установкой, и следует переустановить библиотеку DLL с компакт-диска продукта.
