# <a name="fatal-error-c1121"></a>Неустранимая ошибка C1121

не удалось выполнить вызов CryptoAPI

Компилятор выполнил вызов CryptoAPI и сбой вызова. Переустановите Visual Studio и при необходимости переустановите операционную систему.
