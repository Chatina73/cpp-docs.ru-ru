# <a name="fatal-error-c1065"></a>Неустранимая ошибка C1065

ограничение компилятора: недостаточно тегов

Исходный файл содержит более 65 523 классов, структур, объединений, пространств имен или перечислений.

Разделите файл на небольшие файлы.
