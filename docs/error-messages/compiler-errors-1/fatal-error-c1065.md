# <a name="fatal-error-c1065"></a>Неустранимая ошибка C1065

ограничение компилятора: закончились теги

Исходный файл содержит более 65523 классов, структур, объединений, пространств имен или перечисления.

Разделим файл на файлы меньшего размера.
