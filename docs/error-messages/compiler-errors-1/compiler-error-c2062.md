# <a name="compiler-error-c2062"></a>Ошибка компилятора C2062

непредвиденный тип "тип"

Компилятору не было предполагалось имя типа.

Следующий пример приводит к возникновению ошибки C2062:

```cpp
// C2062.cpp
// compile with: /c
struct A {  : int l; };   // C2062
struct B { private: int l; };   // OK
```

C2062 также может возникать из-за того, как компилятор обрабатывает неопределенные типы в списке параметров конструктора. Если компилятор встречает неопределенный (с ошибками) тип, предполагается, что конструктор является выражением и выдает C2062. Для устранения ошибки используйте только определенные типы в списке параметров конструктора.
