# <a name="compiler-error-c2411"></a>Ошибка компилятора C2411

"идентификатор": недопустимый элемент структуры или объединения в "Context"

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Идентификатор не является членом видимой структуры или объединения в этом контексте.

1. Идентификатор не является членом структуры или объединения, указанного с помощью оператора выбора члена (.).
