# <a name="fatal-error-c1046"></a>Неустранимая ошибка C1046

ограничение компилятора: недопустимая степень вложения структуры

Структура, объединение или класс Превышено ограничение вложенности, 15 уровней. Перепишите определение для уменьшения уровня вложенности. Разделение на несколько частей структуры, объединения или класса с помощью `typedef` чтобы определить один или несколько вложенных структур.
