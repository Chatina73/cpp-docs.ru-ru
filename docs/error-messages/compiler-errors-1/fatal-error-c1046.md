# <a name="fatal-error-c1046"></a>Неустранимая ошибка C1046

ограничение компилятора: недопустимая степень вложения структуры

Превышено ограничение вложенности структуры, объединения или класса, которое составляет 15 уровней. Перепишите определение, чтобы уменьшить уровень вложенности. Разделите структуру, объединение или класс на две или более части, используя **`typedef`** для определения одной или нескольких вложенных структур.
