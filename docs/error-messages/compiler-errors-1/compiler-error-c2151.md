# <a name="compiler-error-c2151"></a>Ошибка компилятора C2151

более одного атрибута языка

Функция имеет более одного ключевого слова ( **`__cdecl`** , **`__stdcall`** или **`__fastcall`** ), указывающего соглашение о вызовах.
