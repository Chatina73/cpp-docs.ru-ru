# <a name="fatal-error-c1085"></a>Неустранимая ошибка C1085

Не удается записать файл тип_файла: "файл": сообщение

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Диск доступен только для чтения.

1. Диск заполнен.

1. Нарушение правил общего доступа.

1. Если в сообщении говорится "неверный номер файла", этот файл мог быть закрыт на переднем плане во время компиляции в фоновом режиме.
