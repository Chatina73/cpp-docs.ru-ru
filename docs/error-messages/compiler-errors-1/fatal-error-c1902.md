# <a name="fatal-error-c1902"></a>Неустранимая ошибка C1902

несоответствие диспетчера базы данных программы; Проверьте установку.

Файл базы данных программы (. pdb) был создан с использованием более новой версии мспдб*xxx*. dll, чем та, которая обнаружена компилятором в системе. Эта ошибка обычно означает, что mspdbsrv. exe или мспдбкоре. DLL отсутствуют или имеют разные версии, чем мспдб*xxx*. dll. (Заполнитель *xxx* в имени файла мспдб*xxx*. dll изменяется при каждом выпуске продукта. Например, в Visual Studio 2015 имя файла — версия mspdb140. dll.)

Убедитесь, что в системе установлены соответствующие версии mspdbsrv. exe, мспдбкоре. dll и мспдб*xxx*. dll. Убедитесь, что несовпадающие версии не были скопированы в каталог, содержащий инструменты компилятора и компоновщика для целевой платформы. Например, вы могли скопировать файлы, чтобы можно было вызвать компилятор или компоновщик из командной строки, не указывая соответствующим образом переменную среды **path** .
