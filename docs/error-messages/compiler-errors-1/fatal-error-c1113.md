# <a name="fatal-error-c1113"></a>Неустранимая ошибка C1113

\#с ошибкой в файле "File"

В директиву [#using](../../preprocessor/hash-using-directive-cpp.md) можно передать только файл в формате MSIL. Параметр компилятора [/CLR](../../build/reference/clr-common-language-runtime-compilation.md) позволяет создать выходной файл MSIL. Другие языки Visual Studio также создают файлы MSIL.
