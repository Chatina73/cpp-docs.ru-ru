# <a name="compiler-error-c2471"></a>Ошибка компилятора C2471

не удается обновить базу данных программы «файл»

Компилятору не удается записать в файл базы данных.

### <a name="to-fix-the-problem"></a>Устранение проблемы

- Убедитесь, что диск содержит достаточно свободного места.

- Убедитесь, что файл не имеет атрибута "только для чтения".

- Убедитесь в отсутствии нарушения общего доступа.
