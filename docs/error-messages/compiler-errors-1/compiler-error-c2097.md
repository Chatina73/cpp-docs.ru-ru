# <a name="compiler-error-c2097"></a>Ошибка компилятора C2097

недопустимая инициализация

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Инициализация переменной с использованием неконстантного значения.

1. Инициализация короткого адреса с длинным адресом.

1. Инициализация локальной структуры, объединения или массива с неконстантным выражением при компиляции с параметром **/Za**.

1. Инициализация с выражением, содержащим оператор "запятая".

1. Инициализация с выражением, которое не является ни константой, ни символьной.
