# <a name="fatal-error-c1009"></a>Неустранимая ошибка C1009

ограничение компилятора: слишком глубокая вложенность макросов

Компилятор попытался развернуть слишком много макросов одновременно. Компилятор имеет ограничение в 256 уровней вложенных макросов. Разделите вложенные макросы на более простые макросы.
