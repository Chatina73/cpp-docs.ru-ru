# <a name="compiler-error-c2487"></a>Ошибка компилятора C2487

"идентификатор": член класса интерфейса DLL не может быть объявлен с интерфейсом DLL

Можно объявить целый класс или определенные члены класса интерфейса, не являющегося интерфейсом DLL, с интерфейсом DLL. Нельзя объявить класс с интерфейсом DLL, а затем объявить член этого класса с интерфейсом DLL.
