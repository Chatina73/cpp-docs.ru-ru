# <a name="compiler-error-c2155"></a>Ошибка компилятора C2155

"?": недопустимый левый операнд, требуется арифметический тип или указатель

Выражение с левой стороны `?` не может сравниваться с нулем. Необходимо использовать арифметическое выражение OR, которое может сравниваться с нулем.
