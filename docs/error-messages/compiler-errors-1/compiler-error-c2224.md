# <a name="compiler-error-c2224"></a>Ошибка компилятора C2224

слева от ". identifier" должен быть тип структуры или объединения

Операнд слева от точки (.) не является классом, структурой или объединением.

Эта ошибка может быть вызвана левым операндом, который является неопределенной переменной (поэтому тип `int`).
