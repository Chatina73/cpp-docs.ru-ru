# <a name="compiler-error-c2325"></a>Ошибка компилятора C2325

«Тип»: непредвиденный тип справа от «name»

Выполняется вызов деструктора неверного типа.

Следующий пример приводит к возникновению ошибки C2325:

```
// C2325.cpp
// compile with: /c
class A {};
typedef A* pA_t;
void f() {
    A** ppa = new A *;
    ppa->~A*;   // C2325

   pA_t *ppa2 = new pA_t;
   ppa2->~pA_t();   // OK
}
```
