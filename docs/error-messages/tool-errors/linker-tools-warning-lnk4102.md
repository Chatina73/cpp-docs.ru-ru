# <a name="linker-tools-warning-lnk4102"></a>Предупреждение средств компоновщика LNK4102

экспорт деструктора удаления "имя"; образ может работать неправильно

Программа предприняла попытку экспорта деструктора удаления. Результирующее удаление может происходить через границу DLL, так что процесс может освободить память, которой он не владеет. Убедитесь, что заданный символ не указан в DEF-файле и что символ не указан в качестве аргумента параметра **/Import** или **/Export** в командной строке компоновщика.

При перестроении библиотеки времени выполнения C можно проигнорировать это сообщение.
