# <a name="expression-evaluator-error-cxx0032"></a>Ошибка вычислителя выражений CXX0032

деление на 0

Выражение содержит делитель, равный нулю, что недопустимо. Этот делитель может быть нулевым литералом или выражением, результатом вычисления которого является ноль.

Эта ошибка идентична CAN0032.
