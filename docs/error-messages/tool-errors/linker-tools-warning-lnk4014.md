# <a name="linker-tools-warning-lnk4014"></a>Предупреждение средств компоновщика LNK4014

не удается найти объект члена "ObjectName"

LIB не удалось найти `objectname` в библиотеке.

Для параметров **/Remove** и **/Extract** требуется полное имя объекта-члена, который должен быть удален или скопирован в файл. Полное имя включает путь к исходному объектному файлу. Чтобы просмотреть полные имена объектов-членов в библиотеке, используйте DUMPBIN [/арчивемемберс](../../build/reference/archivemembers.md) или LIB [/List](../../build/reference/managing-a-library.md).
