# <a name="linker-tools-error-lnk1166"></a>Ошибка средств компоновщика LNK1166

невозможно скорректировать код по смещению = offset, ва = value

LINK не удалось заполнить код по мере необходимости.

Некоторые инструкции не могут пересекать границы страниц на некоторых процессорах. LINK пытается добавить Pad для исправления этой ситуации. В этом случае LINK не может решить проблему.
