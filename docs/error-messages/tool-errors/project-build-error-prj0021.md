# <a name="project-build-error-prj0021"></a>Ошибка построения проекта PRJ0021

> Инструмент "*инструмент*", свойство "*свойство*" содержит недопустимое имя файла.

Имя файла, указанное в *свойстве* Property для *инструмента* , было недопустимым.

Возможно, вы использовали неизвестный или недопустимый макрос.

Сведения о том, как получить доступ к свойствам проекта, см. в разделе [настройка Visual C++ свойств проекта](../../build/working-with-project-properties.md) .
