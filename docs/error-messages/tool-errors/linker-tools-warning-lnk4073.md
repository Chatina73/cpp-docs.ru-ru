# <a name="linker-tools-warning-lnk4073"></a>Предупреждение средств компоновщика LNK4073

не удается создать карту для файла. ILK; Компоновка выполняется не инкрементно

Недостаточно непрерывного пространства в общей памяти для ссылки на создание файла добавочного состояния (. ILK). Ссылка выполнила неинкрементную сборку.
