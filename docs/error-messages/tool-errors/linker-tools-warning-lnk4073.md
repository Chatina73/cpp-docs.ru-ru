# <a name="linker-tools-warning-lnk4073"></a>Предупреждение средств компоновщика LNK4073

не удается создать сопоставление для ILK-файл; компоновка выполняется не инкрементно

Не было большой недостаточно непрерывного пространства в общей памяти ССЫЛКУ, чтобы создать файл инкрементного состояния (). ССЫЛКА выполняется недобавочная сборка.
