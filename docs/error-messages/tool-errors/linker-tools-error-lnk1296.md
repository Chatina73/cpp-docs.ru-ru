# <a name="linker-tools-error-lnk1296"></a>Ошибка средств компоновщика LNK1296

не удалось загрузить имя файла

Указанная библиотека DLL недоступна.

Возможные расположения для библиотеки DLL: текущий каталог, системный каталог, каталог Windows и каталоги, указанные в переменной среды PATH.
