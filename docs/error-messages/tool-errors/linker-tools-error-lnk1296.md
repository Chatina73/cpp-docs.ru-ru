# <a name="linker-tools-error-lnk1296"></a>Ошибка средств компоновщика LNK1296

не удается загрузить имя файла

Указанная библиотека DLL была недоступна.

Возможные местонахождения для библиотеки DLL находятся в текущем каталоге, системный каталог, в каталог Windows и каталоги, указанные в переменной среды PATH.
