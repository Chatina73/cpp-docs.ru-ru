# <a name="resource-compiler-warning-rw4001"></a>Предупреждение компилятора ресурсов RW4001

. Параметры обработки EXE (/L/M/P/T/K/E/31 или/30)

Указаны параметры обработки EXE, но исполняемый файл для обработки отсутствует. Не используйте эти параметры совместно с RES-файлом.
