# <a name="expression-evaluator-error-cxx0002"></a>Ошибка вычислителя выражений CXX0002

Ошибка при доступе к памяти пользователя

Выражение пытается ссылаться на память, которая не выделена для отлаживаемой программы.

Эта ошибка идентична CAN0002.
