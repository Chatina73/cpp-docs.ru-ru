# <a name="linker-tools-warning-lnk4216"></a>Предупреждение средств компоновщика LNK4216

Экспортированная запись точки входа

Точка входа была экспортирована из библиотеки DLL. Не требуется экспортировать точку входа библиотеки DLL.
