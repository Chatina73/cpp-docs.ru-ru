# <a name="project-build-warning-prj0018"></a>Предупреждение при построении проекта PRJ0018

Не найдены следующие переменные среды:

Переменная среды не определена. Эта ошибка выводит список переменных среды, которые не были определены.

Сведения о макросах в процессе сборки см. в разделе [макросы для пользовательских команд сборки](../../build/reference/common-macros-for-build-commands-and-properties.md) .
