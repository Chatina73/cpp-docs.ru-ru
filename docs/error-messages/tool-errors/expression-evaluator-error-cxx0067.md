# <a name="expression-evaluator-error-cxx0067"></a>Ошибка вычислителя выражений CXX0067

Вычисление функции не поддерживается

Выражение содержит вызов функции. Некоторые вычислители выражений не поддерживают вызовы функций.

Эта ошибка идентична CAN0067.
