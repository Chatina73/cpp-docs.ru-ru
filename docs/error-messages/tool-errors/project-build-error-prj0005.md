# <a name="project-build-error-prj0005"></a>Ошибка построения проекта PRJ0005

Не удалось создать временный файл в каталоге "каталог".

Сбой вызова для создания временного файла. Причины сбоя перечислены ниже.

- Исчерпаны временные имена файлов.

- Каталог Temp доступен только для чтения.

- Отсутствует временный каталог или переменная среды TMP.

- На компьютере не хватает свободного места на диске.
