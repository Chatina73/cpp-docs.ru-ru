# <a name="expression-evaluator-error-cxx0034"></a>Ошибка вычислителя выражений CXX0034

**типы, несовместимые с оператором**

Указанные типы операндов не являются допустимыми для операции.

Например, указатель не может быть умножен на любое значение.

Может потребоваться ввести приведение операндов в тип, совместимый с оператором.
