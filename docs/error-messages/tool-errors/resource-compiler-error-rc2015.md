# <a name="resource-compiler-error-rc2015"></a>Ошибка компилятора ресурсов RC2015

слишком много символов в константе

Символьная константа содержит более двух символов.

Символьные константы ограничены одним символом (стандартными символьными константами) или двумя символами (длинными символьными константами).

Обратите внимание, что escape-последовательность (например, \t для Tab) преобразуется в один символ.
