# <a name="nmake-fatal-error-u1034"></a>Неустранимая ошибка NMAKE U1034

Синтаксическая ошибка: отсутствует разделитель

Отсутствует двоеточие ( **:** ), разделяющее целевые объекты и зависимые элементы.
