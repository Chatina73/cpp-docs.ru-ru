# <a name="resource-compiler-fatal-error-rc1022"></a>Неустранимая ошибка компилятора ресурсов RC1022

требуется "#endif"

`#if`Директива, **#ifdef** или **#ifndef** не была прервана `#endif` директивой.

Убедитесь, что `#if` перед этим оператором действует инструкция, **#ifdef** или **#ifndef** .
