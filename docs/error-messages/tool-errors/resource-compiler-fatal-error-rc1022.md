# <a name="resource-compiler-fatal-error-rc1022"></a>Неустранимая ошибка компилятора ресурсов RC1022

требуется "#endif"

Директива `#if`, **#ifdef**или **#ifndef** не была завершена с помощью директивы `#endif`.

Убедитесь, что перед этим оператором действует `#if`, **#ifdef**или **#ifndef** .
