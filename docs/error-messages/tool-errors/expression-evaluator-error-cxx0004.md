# <a name="expression-evaluator-error-cxx0004"></a>Ошибка вычислителя выражений CXX0004

Синтаксическая ошибка

Неправильный синтаксис выражения.

Повторно введите выражение с правильным синтаксисом.

Эта ошибка идентична ошибке CAN0004.
