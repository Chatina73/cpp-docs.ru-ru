# <a name="resource-compiler-fatal-error-rc1105"></a>Неустранимая ошибка компилятора ресурсов RC1105

Недопустимый переключатель, параметр: слишком много ключей/d

Слишком много символов было определено с помощью параметра "Define Symbols (/d)". Определите некоторые символы в источнике и выполните повторную компиляцию.
