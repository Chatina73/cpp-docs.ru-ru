# <a name="resource-compiler-fatal-error-rc1105"></a>Неустранимая ошибка компилятора ресурсов RC1105

Недопустимый ключ, параметр: слишком много ключей /d

Слишком много символов были определены с помощью определения символов (/ d) параметр. Определите некоторые символы в исходном коде и повторной компиляции.
