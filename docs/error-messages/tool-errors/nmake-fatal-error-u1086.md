# <a name="nmake-fatal-error-u1086"></a>Неустранимая ошибка NMAKE U1086

**правило вывода не может иметь зависимости**

Двоеточие (**:**) в выполняется вывод правила следует указать один из следующих:

- Символ новой строки

- Точка с запятой (**;**), которой может следовать команды

- Знак числа (**#**), который может следовать комментарий
