# <a name="nmake-fatal-error-u1086"></a>Неустранимая ошибка NMAKE U1086

**правило вывода не может иметь зависимости**

За двоеточием ( **:** ) в правиле вывода должен следовать один из следующих элементов:

- Символ перевода строки

- Точка с запятой ( **;** ), за которой может следовать команда

- Знак номера ( **#** ), за которым может следовать комментарий
