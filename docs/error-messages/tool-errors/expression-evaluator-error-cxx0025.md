# <a name="expression-evaluator-error-cxx0025"></a>Ошибка вычислителя выражений CXX0025

оператору требуется структура или объединение

Оператор, принимающий выражение типа `struct` или **объединения** , был применен к выражению, которое не является `struct` или **объединением**.

Компоненты переменных класса, структуры или объединения должны иметь полное имя. Компоненты не могут быть указаны без полной спецификации.

Эта ошибка идентична CAN0025.
