# <a name="linker-tools-error-lnk1188"></a>Ошибка средств компоновщика LNK1188

БАДФИКСУПСЕКТИОН:: недопустимый целевой объект адресной привязки "Symbol"; Возможная Секция нулевой длины

Во время ссылки на VxD целевой объект перемещения не содержит раздел. При использовании LINK386 (более старая версия) запись группы OMF (созданная директивой GROUP Directive) могла использоваться для объединения раздела нулевой длины с другой секцией ненулевой длины. Формат COFF не поддерживает директивы GROUP и разделы нулевой длины. Если ссылка автоматически преобразует этот тип объектов OMF в COFF, может возникнуть Эта ошибка.
