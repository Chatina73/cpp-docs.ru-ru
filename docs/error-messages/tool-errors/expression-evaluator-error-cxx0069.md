# <a name="expression-evaluator-error-cxx0069"></a>Ошибка вычислителя выражений CXX0069

переменная должна кадр стека

Средство оценки выражений не может вычислить переменную, поскольку она отсутствует в кадре стека. Это может быть вызвано переменные, объявленные как часть встроенной функции.
