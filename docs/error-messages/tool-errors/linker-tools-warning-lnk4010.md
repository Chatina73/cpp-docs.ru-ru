# <a name="linker-tools-warning-lnk4010"></a>Предупреждение средств компоновщика LNK4010

Недопустимый номер версии подсистемы; Предполагается версия подсистемы по умолчанию

Можно указать версию для подсистемы образа ([/SUBSYSTEM](../../build/reference/subsystem-specify-subsystem.md)). Каждая подсистема имеет минимальные требования к версии. Если указанная версия ниже минимальной, это предупреждение будет вызвано тем, что компоновщик будет использовать подсистему по умолчанию.
