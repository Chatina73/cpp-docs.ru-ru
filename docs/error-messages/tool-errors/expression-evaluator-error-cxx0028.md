# <a name="expression-evaluator-error-cxx0028"></a>Ошибка вычислителя выражений CXX0028

не является элементом структуры или объединения

Указано выражение вида "struct. member" или "Пструкт->Member", но не \<member> является элементом структуры.

Выражение может быть неправильно заключено в круглые скобки.

Эта ошибка идентична CAN0028.
