# <a name="expression-evaluator-error-cxx0028"></a>Ошибка вычислителя выражений CXX0028

элемент структуры или объединения

Выражение формы «Struct.Member» или «pStruct -> член» был указан, но \<член > не является элементом структуры.

Выражение не может быть заключено в скобки правильно.

Эта ошибка идентична CAN0028.
