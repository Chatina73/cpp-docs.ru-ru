# <a name="project-build-error-prj0035"></a>Ошибка построения проекта PRJ0035

> XML-файл "*файл*" содержит содержимое в Юникоде, которое не может быть преобразовано в кодовую страницу ANSI пользователя.
>
> *Содержимое файла в Юникоде*

*файл* — это XML-файл, созданный в качестве командной строки средства веб-развертывания.

Система проектов обнаружила символы Юникода в некоторых свойствах на странице свойств веб-развертывания, которая не может быть правильно преобразована в ANSI.

Решением этой ошибки является обновление содержимого свойства для использования ANSI или установки кодовой страницы на компьютере и установка ее в качестве системы по умолчанию.
