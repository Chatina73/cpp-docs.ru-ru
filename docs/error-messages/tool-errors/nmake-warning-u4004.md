# <a name="nmake-warning-u4004"></a>Предупреждение программы NMAKE U4004

слишком много правил для целевого объекта "TargetName"

Для данного целевого объекта было указано более одного блока описания с помощью одиночных двоеточий ( **:** ) в качестве разделителей. NMAKE выполнил команды в первом блоке описания и проигнорировала последующие блоки.

Чтобы указать один и тот же целевой объект в нескольких зависимостях, используйте двойные двоеточия (`::`) в качестве разделителя в каждой строке зависимости.
