# <a name="linker-tools-warning-lnk4002"></a>Предупреждение средств компоновщика LNK4002

символ, определенный в объекте

Символ, отображаемый в декорированной форме, был указан в недекорированной форме в `object`, но не удалось найти уникальное совпадение с декорированным символом. Этому предупреждению всегда предшествует предупреждение [LNK4022](../../error-messages/tool-errors/linker-tools-warning-lnk4022.md) , за которым следует Неустранимая ошибка [LNK1152](../../error-messages/tool-errors/linker-tools-error-lnk1152.md).
