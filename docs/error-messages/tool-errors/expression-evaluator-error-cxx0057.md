# <a name="expression-evaluator-error-cxx0057"></a>Ошибка вычислителя выражений CXX0057

нет кода в номере строки

Код для указанного номера строки не создан. Его нельзя использовать в качестве точки останова.

Эта ошибка идентична CAN0057.
