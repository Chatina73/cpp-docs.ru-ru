# <a name="project-build-error-prj0050"></a>Ошибка построения проекта PRJ0050

Не удалось зарегистрировать выход. Убедитесь, что у вас есть необходимые разрешения для изменения реестра.

Системе сборки Visual C++ не удалось зарегистрировать выходные данные сборки (DLL или exe). Для изменения реестра необходимо войти в систему с правами администратора.

При создании библиотеки DLL можно попробовать зарегистрировать DLL-файл вручную с помощью regsvr32.exe, при этом должны отобразиться сведения о том, почему произошел сбой сборки.

Если вы не создаете библиотеку DLL, просмотрите журнал сборки для команды, которая вызывает ошибку.
