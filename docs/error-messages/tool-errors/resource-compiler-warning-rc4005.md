# <a name="resource-compiler-warning-rc4005"></a>Предупреждение компилятора ресурсов RC4005

"идентификатор": Переопределение макроса

Идентификатор определен дважды. Компилятор использовал второе определение макроса.

Это предупреждение может быть вызвано определением макроса в командной строке и в коде с помощью `#define` директивы. Это также может быть вызвано макросами, импортированными из включаемых файлов.

Чтобы устранить это предупреждение, либо удалите одно из определений, либо используйте `#undef` директиву перед вторым определением.
