# <a name="expression-evaluator-error-cxx0005"></a>Ошибка вычислителя выражений CXX0005

оператор не поддерживается

В выражении указан неподдерживаемый оператор C. Напишите эквивалентное выражение, используя поддерживаемые операторы C.

Эта ошибка идентична CAN0005.
