# <a name="linker-tools-error-lnk1282"></a>Ошибка средств компоновщика LNK1282

не удается/REBASE-файл; файл подписан

Была предпринята попытка изменить базовый адрес подписанной сборки с помощью параметра/REBASE [editbin](../../build/reference/editbin-reference.md). Чтобы сделать это, сначала изменить базовый адрес и затем подписать сборку.
