# <a name="expression-evaluator-error-cxx0072"></a>Ошибка вычислителя выражений CXX0072

Ошибка: информация о типе отсутствует или неизвестна

PCH-файл не связан в, или в коде имеется ссылка на тип, который находится в модуле, не скомпилированном с помощью/зи.

Эта ошибка идентична CAN0067.
