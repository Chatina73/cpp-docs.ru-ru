# <a name="resource-compiler-fatal-error-rc1203"></a>Неустранимая ошибка компилятора ресурсов RC1203

указан недопустимый шестнадцатеричный идентификатор языка по умолчанию.

За параметром "указать язык по умолчанию" (/l) был указан недопустимый шестнадцатеричный идентификатор языка. Список допустимых идентификаторов языков см. в разделе [строки языка и страны или региона](../../c-runtime-library/locale-names-languages-and-country-region-strings.md) в *справочнике по библиотеке времени выполнения* .
