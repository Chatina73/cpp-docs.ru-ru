# <a name="math-error-m6110"></a>Математическая ошибка M6110

переполнение стека

Выражение с плавающей запятой вызывает переполнение стека с плавающей запятой.

Но не более семи уровней, а также восемь уровней, обычно поддерживаются сопроцессора 8087/287/387 полностью или частично переполнение стека исключений с плавающей запятой.

Программа завершается с кодом завершения 138.
