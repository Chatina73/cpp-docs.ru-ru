# <a name="math-error-m6110"></a>Математическая ошибка M6110

переполнение стека

Выражение с плавающей запятой вызывает переполнение стека с плавающей точкой.

Переполнение стека. исключения с плавающей запятой перехватываются до семи уровней в дополнение к восьми уровням, обычно поддерживаемому сопроцессором 8087/287/387.

Программа завершается с кодом выхода 138.
