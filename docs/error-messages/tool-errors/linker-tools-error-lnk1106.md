# <a name="linker-tools-error-lnk1106"></a>Ошибка средств компоновщика LNK1106

Недопустимый файл или диск полон: не удается найти расположение

Средству не удалось выполнить чтение или запись в `location` в отображенном в памяти файле.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Диск полон.

   Освободите место и повторите компоновку.

1. Попытка связи по сети.

   Некоторые сети не полностью поддерживают размещенные в памяти файлы, используемые компоновщиком. Попробуйте выполнить связывание на локальном диске.

1. Недопустимый блок на диске.

   Хотя операционная система и оборудование диска должны обнаружить такую ошибку, может потребоваться запустить программу проверки диска.

1. Не хватает памяти в куче.

   Дополнительные сведения см. в разделе [C1060](../../error-messages/compiler-errors-1/fatal-error-c1060.md) .
