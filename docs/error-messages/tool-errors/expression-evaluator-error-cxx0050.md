# <a name="expression-evaluator-error-cxx0050"></a>Ошибка вычислителя выражений CXX0050

в левой части:: должен быть классом, структурой или объединением

Значок слева от оператора разрешения области действия (`::`) не класса, структуры или объединения.

Эта ошибка идентична CAN0050.
