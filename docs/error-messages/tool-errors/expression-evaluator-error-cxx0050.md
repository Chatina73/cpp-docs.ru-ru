# <a name="expression-evaluator-error-cxx0050"></a>Ошибка вычислителя выражений CXX0050

Левая сторона:: должна быть классом, структурой или объединением

Символ в левой части оператора разрешения области действия (`::`) не был классом, структурой или объединением.

Эта ошибка идентична CAN0050.
