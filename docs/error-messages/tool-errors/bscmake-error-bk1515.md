# <a name="bscmake-error-bk1515"></a>Ошибка BSCMAKE BK1515

бскфиле: несовместимая версия: добавочное обновление невозможно

BSC файл был создан с помощью другой версии BSCMAKE. Повторно создайте BSC файл.
