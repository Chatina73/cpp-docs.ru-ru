# <a name="linker-tools-warning-lnk4219"></a>Предупреждение средств компоновщика LNK4219

Переполнение адресной привязки имени адресной привязки. Целевой символ "имя целевого символа" выходит за пределы допустимого диапазона, вставляется преобразователь

Компоновщик вставил преобразователь в ситуацию, когда адрес или смещение не удалось поместить в данную инструкцию, так как целевой символ находится далеко от расположения инструкции.

Возможно, потребуется изменить порядок изображения (например, с помощью параметра [/Order](../../build/reference/order-put-functions-in-order.md) ), чтобы избежать дополнительного уровня косвенного обращения.
