# <a name="command-line-warning-d9040"></a>Предупреждение командной строки D9040

параметр "/Analyze" игнорируется; Предупреждения анализа кода недоступны в этом выпуске компилятора

Параметр командной строки **/Analyze** доступен не во всех выпусках Visual Studio. Чтобы устранить это предупреждение, переключитесь на поддерживаемый выпуск Visual Studio или удалите параметр командной строки.
