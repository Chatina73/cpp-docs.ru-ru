# <a name="linker-tools-error-lnk1164"></a>Ошибка средств компоновщика LNK1164

выравнивание раздела раздела (число) больше значения/ALIGN

Размер выравнивания для данного раздела в объектном файле превышает значение, заданное параметром [/align](../../build/reference/align-section-alignment.md) . Значение **/align** должно быть степенью 2 и должно быть равно или превышать выравнивание раздела, заданное в объектном файле.

Либо перекомпилируйте с меньшим выравниванием раздела, либо увеличьте значение **/align** .
