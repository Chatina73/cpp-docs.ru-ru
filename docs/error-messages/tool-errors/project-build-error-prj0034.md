# <a name="project-build-error-prj0034"></a>Ошибка построения проекта PRJ0034

Свойство "Дополнительные зависимости" для настраиваемого этапа сборки уровня проекта содержит "макрос", который вычисляется как "macro_expansion".

Пользовательский этап сборки в проекте содержал ошибку в дополнительной зависимости, вероятно, из-за проблемы с оценкой макроса. Эта ошибка также может означать, что путь неправильно сформирован, содержит символы или сочетания символов, которые не являются допустимыми в пути к файлу.

Чтобы устранить эту ошибку, исправьте макрос или исправьте спецификацию пути. Вычисленный путь представляет собой абсолютный путь из каталога проекта.
