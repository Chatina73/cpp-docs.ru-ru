# <a name="linker-tools-error-lnk1224"></a>Ошибка средств компоновщика LNK1224

Недопустимый базовый адрес образа

Указан недопустимый базовый адрес для образа. Базовый адрес должен быть равен 64 КБ (последние четыре шестнадцатеричных цифры должны быть равны нулю), а основание образа должно соответствовать 32-разрядному значению со знаком или без знака.
