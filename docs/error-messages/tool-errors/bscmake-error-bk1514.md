# <a name="bscmake-error-bk1514"></a>Ошибка BSCMAKE BK1514

каждого. SBR обрезанные файлы, не найдены в имени файла

Ни один из SBR-файлов, указанных для обновления, не был частью исходного файла сведений о просмотре (BSC). Чтобы найти имена SBR файлов, вызвавших эту ошибку, прочтите [BK4502](../../error-messages/tool-errors/bscmake-warning-bk4502.md) предупреждения, предшествующие этой ошибке.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Указано неправильное имя файла для SBR или BSC.

1. Поврежденный BSC файл требует BSCMAKE для перестроения.
