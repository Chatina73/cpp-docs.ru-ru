# <a name="math-error-m6102"></a>Математическая ошибка M6102

DENORMAL

Операция создала очень маленькое число с плавающей запятой, которое является недопустимым из-за потери значимости. Нормальные исключения с плавающей запятой обычно замаскированы, что приводит к их перехвату и обходе.

Программа завершается с кодом выхода 130.
