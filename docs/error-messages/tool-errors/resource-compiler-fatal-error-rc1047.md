# <a name="resource-compiler-fatal-error-rc1047"></a>Неустранимая ошибка компилятора ресурсов RC1047

"слишком много параметров параметра," строка ""

Данный параметр указан слишком много раз. Заданная строка является аргументом для параметра, вызвавшего ошибку.
