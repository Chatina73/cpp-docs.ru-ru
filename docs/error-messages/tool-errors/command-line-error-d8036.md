# <a name="command-line-error-d8036"></a>Ошибка командной строки D8036

"/Option" не разрешено использовать с несколькими исходными файлами

Эти параметры компилятора нельзя использовать с несколькими исходными файлами:

- Имя листинга файла сборки (/FA)

- Переименовать объектный файл (/FO)

- Создание исходных данных браузера без локальных переменных (/fr)

- Создание исходных данных браузера, включающих локальные переменные (/FR)
