# <a name="project-build-error-prj0024"></a>Ошибка построения проекта PRJ0024

> Путь Юникода "*путь*" не может быть преобразован в кодовую страницу ANSI пользователя.

*path* — это исходная версия строки пути в Юникоде. Эта ошибка может возникать в случаях, когда путь в Юникоде не может быть напрямую преобразован в ANSI для текущей системной кодовой страницы.

Эта ошибка может возникать, если вы работаете с проектом, разработанным в системе, с помощью кодовой страницы, которая не находится на компьютере.

Решением этой ошибки является обновление пути для использования текста ANSI или установка кодовой страницы на компьютере и установка ее в качестве системы по умолчанию.
