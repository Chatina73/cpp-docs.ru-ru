# <a name="resource-compiler-fatal-error-rc1020"></a>Неустранимая ошибка компилятора ресурсов RC1020

непредвиденный "#endif"

Обнаружена `#endif` директива без соответствующей `#if` директивы, **#ifdef** или **#ifndef** .

Убедитесь в наличии соответствующих `#endif` `#if` инструкций for, **#ifdef** и **#ifndef** .
