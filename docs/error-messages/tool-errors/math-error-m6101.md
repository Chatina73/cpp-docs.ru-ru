# <a name="math-error-m6101"></a>Математическая ошибка M6101

недопустимо

Недопустимая операция.

Эта ошибка может быть вызвана тем, что операнд имеет значение NaN (не число) или бесконечность.

Программа завершается с кодом выхода 129.
