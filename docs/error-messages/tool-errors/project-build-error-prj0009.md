# <a name="project-build-error-prj0009"></a>Ошибка построения проекта PRJ0009

Создание журнала не может быть открыт для записи.

**Убедитесь, что файл не открыт другим процессом и не защищен от записи.**

После задания **ведение журнала построения** свойства **Да** и выполнять построение или перестроение, Visual C++ не удалось открыть журнал построения в монопольном режиме.

Проверьте **ведение журнала построения** установка, открыв **параметры** диалоговое окно (на **средства** меню, щелкните **параметры** команда) и затем выбрав **сборки VC ++** в **проекты** папки. Файл сборки называется BuildLog.htm и записывается в промежуточный каталог $(IntDir).

Возможные причины этой ошибки:

- Запускается два процесса из Visual C++ и попытке построить ту же конфигурацию одного проекта в обоих одновременно.

- В процессе, который блокирует файл открывается в файл журнала сборки.

- Пользователь не имеет разрешения на создание файла.

- Текущий пользователь имеет доступ на запись в файл BuildLog.htm перезаписывается.
