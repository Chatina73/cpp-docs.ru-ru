# <a name="linker-tools-warning-lnk4229"></a>Предупреждение средств компоновщика LNK4229

обнаружена недопустимая директива/директиве; игнорируют

Директива, переданная через директиву [pragma,](../../preprocessor/comment-c-cpp.md) не является допустимой. Компоновщик игнорирует `/directive`.
