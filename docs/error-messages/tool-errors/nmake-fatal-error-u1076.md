# <a name="nmake-fatal-error-u1076"></a>Неустранимая ошибка NMAKE U1076

слишком длинное имя

Строка превысила один из следующих ограничений:

- 1024 символов для имени макроса.

- 256 символов для целевого пути.

- 2048 символов для команды.
