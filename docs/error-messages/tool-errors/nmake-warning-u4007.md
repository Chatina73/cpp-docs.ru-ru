# <a name="nmake-warning-u4007"></a>Предупреждение программы NMAKE U4007

слишком длинное имя файла "имяфайла"; усечение до 8,3

Базовое имя данного файла содержит более восьми символов, либо расширение содержит более трех символов. NMAKE усекает имя до восьми-символьной базы и расширения с тремя символами.

Если файловая система поддерживает длинные имена файлов, заключите имя в двойные кавычки ( **"** ).
