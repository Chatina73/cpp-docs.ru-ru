# <a name="resource-compiler-error-rc2135"></a>Ошибка компилятора ресурсов RC2135

файл не найден: имя файла

Не найден файл, указанный в командной строке компилятора ресурсов. Проверьте, не был ли файл перемещен в другой каталог и правильно ли введены имя и путь файла.

Поиск файлов выполняется с помощью переменной среды **include** или параметра Visual C++ **include** .
