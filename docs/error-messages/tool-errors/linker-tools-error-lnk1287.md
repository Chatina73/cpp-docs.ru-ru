# <a name="linker-tools-error-lnk1287"></a>Ошибка средств компоновщика LNK1287

Недопустимая управляемая функция точки входа

Точка входа недопустима для управляемого образа. Возвращаемым типом управляемой функции точки входа может быть только **`void`** или **`int`** . Тип параметра управляемой функции точки входа может быть только **`void`** или `String []` .
