# <a name="expression-evaluator-error-cxx0031"></a>Ошибка вычислителя выражений CXX0031

выражение не разворачивается

В процессе вычисления выражения произошла внутренняя ошибка.

Возможно, вам удастся написать эквивалентное выражение, которое может быть вычислено правильно.

Эта ошибка идентична CAN0031.
