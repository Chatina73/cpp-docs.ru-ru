# <a name="project-build-error-prj0031"></a>Ошибка построения проекта PRJ0031

Свойство "Outputs" (выходные данные) настраиваемого этапа сборки для файла "файл" содержит "макрос", результатом которого является "macro_expansion".

Пользовательский этап сборки в файле имел некорректные выходные данные, вероятно, из-за проблемы с оценкой макроса. Эта ошибка также может означать, что путь неправильно сформирован, содержит символы или сочетания символов, которые не являются допустимыми в пути к файлу.

Чтобы устранить эту ошибку, исправьте макрос или исправьте спецификацию пути. Вычисленный путь представляет собой абсолютный путь из каталога проекта.
