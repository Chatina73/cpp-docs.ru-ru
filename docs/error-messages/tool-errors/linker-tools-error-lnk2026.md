# <a name="linker-tools-error-lnk2026"></a>Ошибка средств компоновщика LNK2026

незащищенный модуль для образа SAFESEH

Указан параметр [/SAFESEH](../../build/reference/safeseh-image-has-safe-exception-handlers.md) , но модуль не был совместим с функцией обработки безопасного исключения. Если вы хотите использовать этот модуль с параметром **/SAFESEH**, вам потребуется перекомпилировать модуль.
