# <a name="linker-tools-error-lnk1314"></a>Ошибка средств компоновщика LNK1314

поврежденная или недопустимая таблица символов COFF (неопределенный статический символ или обозначение метки)

Содержимое раздела данного объекта, созданное компилятором, кажется поврежденным.

Эта ошибка может представлять сбой компилятора.
