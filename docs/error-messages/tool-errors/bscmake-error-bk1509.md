# <a name="bscmake-error-bk1509"></a>Ошибка BSCMAKE BK1509

не хватает памяти в куче

В Программе BSCMAKE недостаточно памяти, включая виртуальную память.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Возможные способы устранения этой ошибки

1. Освободите место на диске.

1. Увеличьте размер файла подкачки.

1. Увеличьте размер файла подкачки Windows.

1. Для уменьшения объема памяти для BSCMAKE необходимо использовать/еи или/ES, чтобы исключить некоторые входные файлы или/ем, чтобы исключить тексты макросов.
