# <a name="bscmake-error-bk1510"></a>Ошибка BSCMAKE BK1510

поврежден. SBR-файла

Заданный SBR-файл не имеет ожидаемого формата. Выполните повторную компиляцию, чтобы создать новый SBR-файл.
