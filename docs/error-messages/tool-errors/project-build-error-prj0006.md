# <a name="project-build-error-prj0006"></a>Ошибка построения проекта PRJ0006

Не удалось открыть временный файл "файл". Убедитесь, что файл существует и что каталог не защищен от записи.

Визуальному C++ элементу не удалось создать временный файл во время процесса сборки. Возможные причины:

- Нет временного каталога.

- Временный каталог только для чтения.

- Недостаточно места на диске.

- Папка $ (IntDir) доступна только для чтения или содержит временные файлы, доступные только для чтения.

Эта ошибка также может возникать после ошибки проекта PRJ0007: не удалось создать выходной каталог "каталог". Ошибка проекта PRJ0007 означает, что не удалось создать каталог $ (IntDir), что означает, что создание временных файлов также завершится ошибкой.

Временные файлы создаются при каждом указании:

- Файл ответов.

- Настраиваемый этап сборки.

- Событие сборки.
