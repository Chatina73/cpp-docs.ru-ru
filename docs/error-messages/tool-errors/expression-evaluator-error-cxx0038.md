# <a name="expression-evaluator-error-cxx0038"></a>Ошибка вычислителя выражений CXX0038

Несоответствие числа и/или тип аргумента функции

Вызов указанной функции не соответствует прототип для функции.

Перепишите вызов с правильным количеством аргументов. Приведите тип каждого аргумента в соответствие с прототипом, при необходимости.

Эта ошибка идентична ошибке CAN0038.
