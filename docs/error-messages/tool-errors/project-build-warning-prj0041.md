# <a name="project-build-warning-prj0041"></a>Предупреждение при построении проекта PRJ0041

Не удается найти отсутствующую зависимость "dependency" для файла "файл". Проект может быть построен, но может по-прежнему отображаться в актуальном состоянии до тех пор, пока этот файл не будет найден.

Файл (например, файл ресурсов или IDL/odl-файл) содержит инструкцию include, которую не удалось разрешить системе проекта.

Так как система проектов не обрабатывает инструкции препроцессора (например, #if), ошибочный оператор может фактически не быть частью сборки.

Чтобы устранить это предупреждение, удалите ненужный код в RC-файлах или добавьте файлы заполнителей для соответствующего имени.
