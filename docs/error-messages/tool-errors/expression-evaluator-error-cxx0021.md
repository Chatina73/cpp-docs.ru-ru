# <a name="expression-evaluator-error-cxx0021"></a>Ошибка вычислителя выражений CXX0021

Структура или объединение, используемые в качестве скаляра

В выражении использовалась структура или объединение, но ни один элемент не был указан.

При управлении структурой или переменной объединения имя переменной может отображаться сама по себе без квалификатора поля. Если в выражении используется структура или объединение, то оно должно уточняться конкретным требуемым элементом.

Укажите элемент, значение которого должно использоваться в выражении.

Эта ошибка идентична CAN0021.
