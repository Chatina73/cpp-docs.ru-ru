# <a name="profile-guided-optimization-warning-pg1039"></a>Предупреждение профильной оптимизации PG1039

Не удалось дождаться очистки времени молчания.

Приложение не позволяло запускать [pgosweep](../../build/pgosweep.md) .  Если возможно, запустите pgosweep, когда приложение находится в состоянии простоя.
