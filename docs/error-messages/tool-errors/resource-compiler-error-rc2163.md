# <a name="resource-compiler-error-rc2163"></a>Ошибка компилятора ресурсов RC2163

требуется тип ускорителя [ASCII или VIRTKEY]

Поле `type` в операторе **ACCELERATORS** должно содержать значение ASCII или VIRTKEY.
