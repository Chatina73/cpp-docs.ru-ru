# <a name="nmake-fatal-error-u1095"></a>Неустранимая ошибка NMAKE U1095

слишком длинная Расширенная Командная строка "CommandLine"

После расширения макроса указанная командная строка превысила предельное значение длины командной строки для операционной системы.

MS-DOS позволяет использовать до 128 символов в командной строке.

Если команда предназначена для программы, которая может принимать входные данные командной строки из файла, измените команду и укажите входные данные из файла на диске или во встроенном файле. Например, СВЯЖИТЕ и LIB принимают входные данные из файла ответов.
