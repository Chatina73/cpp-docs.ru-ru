# <a name="nmake-fatal-error-u1071"></a>Неустранимая ошибка NMAKE U1071

цикл в дереве зависимостей для целевого объекта "TargetName"

В дереве зависимостей для данного целевого объекта существует циклическая зависимость. Заданный целевой объект зависит от одного из зависимых объектов данного целевого объекта. Недопустимые циклические зависимости.
