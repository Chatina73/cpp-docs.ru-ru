# <a name="expression-evaluator-error-cxx0014"></a>Ошибка вычислителя выражений CXX0014

Отсутствует операнд

Оператор был указан без обязательного операнда. Проверьте синтаксис выражения.

Эта ошибка идентична CAN0014.
