# <a name="linker-tools-error-lnk1103"></a>Ошибка средств компоновщика LNK1103

отладочная информация повреждена; Перекомпилируйте модуль

Это ошибка может быть вызвана, так как компиляция была прервана, прежде чем файл допустимый объект был создан.

Перекомпилируйте файл заданного объекта. Если повторная компиляция не устраняет проблему, обратите внимание, обстоятельствах возникновения ошибки, попытайтесь изолировать эту проблему и создать воспроизводимый тестовый случай. Обратитесь к руководству файл справки технической поддержки или раздел технической поддержки в одном из вашего руководства.
