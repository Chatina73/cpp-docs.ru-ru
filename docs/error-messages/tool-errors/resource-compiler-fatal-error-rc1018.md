# <a name="resource-compiler-fatal-error-rc1018"></a>Неустранимая ошибка компилятора ресурсов RC1018

непредвиденный "#elif"

`#elif`Директива не появилась в `#if` конструкции, **#ifdef** или **#ifndef** .

Убедитесь, что `#if` перед этим оператором действует инструкция, **#ifdef** или **#ifndef** .
