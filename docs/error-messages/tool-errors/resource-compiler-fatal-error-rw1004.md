# <a name="resource-compiler-fatal-error-rw1004"></a>Неустранимая ошибка компилятора ресурсов RW1004

Непредвиденный конец файла

Эта ошибка может быть вызвана отсутствием перевода строки и символами возврата каретки в последней строке текстового файла.
