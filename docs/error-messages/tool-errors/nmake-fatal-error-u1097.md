# <a name="nmake-fatal-error-u1097"></a>Неустранимая ошибка NMAKE U1097

для синтаксиса имен компонентов требуется зависимость от имени

Текущая зависимость не имеет явной зависимой или неявной зависимой. Синтаксис filename-Parts, использующий описатель percent ( `%` ), представляет компоненты первой зависимости от текущего целевого объекта.
