# <a name="resource-compiler-fatal-error-rc1009"></a>Неустранимая ошибка компилятора ресурсов RC1009

ограничение компилятора: макросы слишком глубоко вложены в макрос

Файл превышает ограничение на число компиляторов ресурсов для вложения макроса. Измените указанный исходный файл, чтобы уменьшить глубину вложения макросов.
