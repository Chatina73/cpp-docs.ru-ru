# <a name="linker-tools-error-lnk1248"></a>Ошибка средств компоновщика LNK1248

> Размер изображения ("*Размер*") превышает максимально допустимый размер (80000000)

Компоновщик определил, что размер выходного файла будет превышать максимально возможный размер для 32-разрядного образа программы. Вам может потребоваться сделать программу в нескольких библиотеках DLL.
