# <a name="linker-tools-error-lnk1248"></a>Ошибка средств компоновщика LNK1248

> размер изображения ("*размер*") превышает максимально допустимый размер (80000000)

Компоновщик определил, что размер выходного файла будет превышать максимально возможный размер для образа 32-разрядной программы. Вы можете сделать программу с несколькими библиотеками DLL.
