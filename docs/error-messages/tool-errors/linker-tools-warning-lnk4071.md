# <a name="linker-tools-warning-lnk4071"></a>Предупреждение средств компоновщика LNK4071

невозможно выполнить инкрементную компоновку для последующих ссылок

Ссылка обнаружила несколько определений для одного или нескольких символов, но [/Force](../../build/reference/force-force-file-output.md) или **/Force: несколько** были использованы для создания выходного файла независимо от ошибок. Ссылка удалила файл добавочного состояния (ILK).
