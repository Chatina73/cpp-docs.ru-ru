# <a name="linker-tools-error-lnk1332"></a>Ошибка средств компоновщика LNK1332

обнаружены \<count> Среда выполнения Windows типы, импортированные в один модуль и определенные в другом модуле

При создании текущего целевого объекта Компоновщик обнаружил <`count`> типов Среда выполнения Windows, каждый из которых импортируется в один модуль и также определяется в другом модуле.

### <a name="to-correct-this-error"></a>Исправление ошибки

- Исправьте все ошибки LNK2039 в сборке в соответствии с предложением в сообщении об ошибке.
