# <a name="project-build-error-prj0025"></a>Ошибка построения проекта PRJ0025

> Пакетный файл "*файл*" содержит содержимое в Юникоде, которое не может быть преобразовано в кодовую страницу ANSI пользователя.
>
> *Содержимое файла в Юникоде*

Система проектов обнаружила содержимое в Юникоде в настраиваемом правиле сборки или в событии сборки, которое не может быть верно преобразовано в текущую кодовую страницу ANSI пользователя.

Решение этой ошибки заключается в обновлении содержимого правила сборки или события сборки для использования ANSI или установки кодовой страницы на компьютере и настройки ее в качестве системы по умолчанию.

Дополнительные сведения о настраиваемых шагах сборки и событиях сборки см. в разделе [Основные сведения о пользовательских шагах сборки и событиях сборки](../../build/understanding-custom-build-steps-and-build-events.md).
