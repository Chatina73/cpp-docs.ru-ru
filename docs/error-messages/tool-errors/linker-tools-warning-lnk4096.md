# <a name="linker-tools-warning-lnk4096"></a>Предупреждение средств компоновщика LNK4096

Значение/BASE "Number" недопустимо для Windows 95 и Windows 98; образ не может быть запущен

Указан недопустимый базовый адрес. Исполняемые файлы Windows 95 и Windows 98 должны иметь базовый адрес больше 0x400000. Дополнительные сведения об основных адресах см. в разделе параметр компоновщика [/base](../../build/reference/base-base-address.md) .
