# <a name="linker-tools-warning-lnk4096"></a>Предупреждение средств компоновщика LNK4096

/ BASE значение «число» является недопустимым для Windows 95 и Windows 98; образ нельзя запустить

Указанный базовый адрес является недопустимым. Windows 95 и Windows 98 исполняемые файлы должны иметь больше 0x400000 базовый адрес. Дополнительные сведения о базовых адресов, см. в разделе [/BASE](../../build/reference/base-base-address.md) параметр компоновщика.
