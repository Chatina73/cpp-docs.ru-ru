# <a name="project-build-error-prj0027"></a>Ошибка построения проекта PRJ0027

Сообщение журнала в Юникоде "содержимое" содержит содержимое, которое не может быть преобразовано в кодовую страницу ANSI пользователя.

Это предупреждение обычно отображается в сочетании с ошибками при создании файлов пакетной службы и/или ответа.

Решение этой ошибки заключается в обновлении содержимого журнала сборки для использования ANSI или установки кодовой страницы на компьютере и настройки ее в качестве системы по умолчанию.
