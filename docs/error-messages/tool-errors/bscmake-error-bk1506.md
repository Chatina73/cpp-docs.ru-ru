# <a name="bscmake-error-bk1506"></a>Ошибка BSCMAKE BK1506

не удается открыть файл "имя файла" [: причина]

BSCMAKE не может открыть файл.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Файл заблокирован другим процессом. Если `reason` сообщает об **отказе в разрешении**, браузер может использовать файл. Закройте окно обзора и повторите сборку.

1. Полный диск.

1. Аппаратная ошибка.

1. Указанный выходной файл имеет то же имя, что и существующий подкаталог.
