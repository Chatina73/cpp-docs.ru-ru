# <a name="resource-compiler-error-rc2111"></a>Ошибка компилятора ресурсов RC2111

Недопустимый тип элемента управления

Каждого оператора CONTROL в **диалоговое ОКНО** инструкция должна быть одно из следующих: 3STATE AUTO3, АВТОМАТИЧЕСКАЯ ПРОВЕРКА, AUTORADIO, BEDIT, ФЛАЖОК, ПОЛЕ СО СПИСКОМ, ЭЛЕМЕНТ УПРАВЛЕНИЯ, CTEXT, DEFPUSHBUTTON, EDITTEXT, GROUPBOX, HEDIT, ЗНАЧОК, IEDIT, LISTBOX, LTEXT, PUSHBOX, PUSHBUTTON, RADIOBUTTON, RTEXT, ПОЛОСЫ ПРОКРУТКИ, USERBUTTON.

Убедитесь, что эти операторы УПРАВЛЕНИЯ написаны правильно.
