# <a name="resource-compiler-error-rc2111"></a>Ошибка компилятора ресурсов RC2111

Недопустимый тип элемента управления

Каждая инструкция CONTROL в инструкции **DIALOG** должна быть одной из следующих: 3STATE, AUTO3, Автопроверка, Авторадио, БЕДИТ, CheckBox, ComboBox, Control, столбец CTEXT, ДЕФПУШБУТТОН, EditText, GROUPBOX, ХЕДИТ, Icon, ИЕДИТ, ListBox, ЛТЕКСТ, ПУШБОКС, кнопка, RADIOBUTTON, РТЕКСТ, ScrollBar, усербуттон.

Убедитесь, что эти операторы управления написаны правильно.
