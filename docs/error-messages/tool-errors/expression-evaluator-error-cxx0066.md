# <a name="expression-evaluator-error-cxx0066"></a>Ошибка вычислителя выражений CXX0066

статический элемент отсутствует

Не удалось найти или определить статический член класса. Эта ошибка может возникать из члена статического класса, который объявлен, но не определен, или определен только и ссылается в модулях, которые не содержат отладочную информацию.

Эта ошибка идентична CAN0066.
