# <a name="expression-evaluator-error-cxx0020"></a>Ошибка вычислителя выражений CXX0020

недопустимые типы операндов для этой операции

К выражению с недопустимым типом для этого оператора применен оператор.

Например, недопустимо принимать адрес регистра или индекс массива с выражением с плавающей запятой.

Эта ошибка идентична CAN0020.
