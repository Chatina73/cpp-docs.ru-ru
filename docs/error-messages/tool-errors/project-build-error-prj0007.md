# <a name="project-build-error-prj0007"></a>Ошибка построения проекта PRJ0007

Не удалось создать выходной каталог «каталог».

Не удалось создать выходной каталог Visual C++. Возможные причины:

- Каталог, в котором был быть создан в выходной каталог только для чтения.

- Недостаточно места на диске.

- Указан недопустимый путь к каталогу.

- Разрешения пользователя не достаточны для создания каталога.
