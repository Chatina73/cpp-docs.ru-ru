# <a name="resource-compiler-fatal-error-rw1023"></a>Неустранимая ошибка компилятора ресурсов RW1023

Ошибка ввода-вывода при записи файла, диск полон

Размер свободного пространства должен быть не менее вдвое больше размера создаваемого исполняемого файла.
