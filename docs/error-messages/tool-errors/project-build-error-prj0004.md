# <a name="project-build-error-prj0004"></a>Ошибка построения проекта PRJ0004

> Не удалось создать командную строку для инструмента "*Tool*".

Одно или несколько свойств были заданы таким образом, чтобы сделать синтаксис вызова *средства* недопустимым:

- Возможно, вы указали некорректные или неизвестные макросы.

- Возможно, на вашем компьютере недостаточно свободного места на диске.

Можно просмотреть параметры средства, просмотрев [страницу свойств Командная строка](../../build/reference/command-line-property-pages.md).
