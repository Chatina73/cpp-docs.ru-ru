# <a name="project-build-error-prj0004"></a>Ошибка построения проекта PRJ0004

> Не удалось создать командную строку для "*средство*" инструмента.

Одно или несколько свойств были указаны в таким образом, синтаксис вызова *средство* недопустимый:

- Указан неправильный или неизвестный макросы.

- На компьютере недостаточно свободного места на диске.

Вы можете просмотреть параметры для средства, просмотрев [страницу свойств командной строки](../../ide/command-line-property-pages.md).
