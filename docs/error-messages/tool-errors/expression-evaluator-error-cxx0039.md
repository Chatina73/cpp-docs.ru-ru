# <a name="expression-evaluator-error-cxx0039"></a>Ошибка вычислителя выражений CXX0039

символ неоднозначен

Вычислителю выражений C не удается определить, какой экземпляр символа следует использовать в выражении. Символ встречается более одного раза в дереве наследования.

`::`Для явного указания экземпляра, используемого в выражении, необходимо использовать оператор разрешения области ().

Эта ошибка идентична CAN0039.
