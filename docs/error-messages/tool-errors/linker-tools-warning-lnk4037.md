# <a name="linker-tools-warning-lnk4037"></a>Предупреждение средств компоновщика LNK4037

>"*символ*" не существует; игнорируют

*Символ* декорированного имени не может быть упорядочен с помощью параметра [/Order](../../build/reference/order-put-functions-in-order.md) , так как он не найден в программе. Проверьте спецификацию *символа* в файле ответов на заказ. Дополнительные сведения см. в описании параметра-компоновщика [/Order (размещение функций в порядке)](../../build/reference/order-put-functions-in-order.md) .

> [!NOTE]
> LINK не может упорядочивать статические функции, так как имена статических функций не являются общедоступными именами символов. Если указано значение **/Order** , это предупреждение компоновщика создается для каждого символа в файле ответов заказа, который является либо статическим, либо не найденным.
