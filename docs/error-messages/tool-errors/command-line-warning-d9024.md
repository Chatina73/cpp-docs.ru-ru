# <a name="command-line-warning-d9024"></a>Предупреждение командной строки D9024

Неизвестный исходный файл типа «имя_файла», объектного файла

Расширение указанного файла не распознан. Файл считается объектный файл и был передан компоновщику.

Распознаются следующие расширения:

- .c (исходный файл C)

- .cxx (исходный файл C++)

- .cpp (исходный файл C++)

- OBJ (объектный файл)

- lib (файл библиотеки)

- .DEF (файл определения модуля)

- .exp (компоновщика Экспортирует файлы, создаваемые ССЫЛКУ/LIB)
