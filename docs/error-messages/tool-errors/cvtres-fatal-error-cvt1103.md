# <a name="cvtres-fatal-error-cvt1103"></a>Неустранимая ошибка CVTRES CVT1103

не удается прочитать имя файла

При попытке CVTRES прочитать заданный файл произошла неустранимая ошибка.

Эта ошибка может быть вызвана повреждением файла.
