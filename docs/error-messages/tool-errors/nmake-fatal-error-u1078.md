# <a name="nmake-fatal-error-u1078"></a>Неустранимая ошибка NMAKE U1078

переполнение константы в "Expression"

Данное выражение содержит константу, которая выходит за пределы диапазона от-2 147 483 648 до 2 147 483 647. Константа присутствует в одной из следующих ситуаций:

- Выражение, заданное с помощью директивы предварительной обработки

- Уровень ошибки, заданный с помощью модификатора команды тире ( **-** )
