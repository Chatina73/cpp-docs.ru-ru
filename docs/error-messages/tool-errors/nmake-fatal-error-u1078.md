# <a name="nmake-fatal-error-u1078"></a>Неустранимая ошибка NMAKE U1078

переполнение константы в «expression»

Указанное выражение содержит константу, превышает допустимый диапазон — 2 147 483 648 для 2 147 483 647. Константа использовалась в одном из следующих ситуаций:

- Выражение, заданное с помощью директивы препроцессора

- Порог ошибок, заданный с тире (**-**) модификатора команды.
