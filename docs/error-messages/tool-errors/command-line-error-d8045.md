# <a name="command-line-error-d8045"></a>Ошибка командной строки D8045

не удается скомпилировать C-файл «файл» с параметром/CLR

Можно передать только файлы исходного кода C++ для компиляции с параметром **/CLR**.  Используйте **/TP** для компиляции c-файла в качестве CPP-файла; см. в разделе [TP, /TC, /TP (определение типа исходного файла)](../../build/reference/tc-tp-tc-tp-specify-source-file-type.md) Дополнительные сведения.

Дополнительные сведения см. в разделе [/clr (компиляция CLR)](../../build/reference/clr-common-language-runtime-compilation.md).

D8045 также может возникать при компиляции приложения ATL с помощью Visual C++. См. в разделе [как: Миграция в/CLR](../../dotnet/how-to-migrate-to-clr.md) Дополнительные сведения.
