# <a name="resource-compiler-error-rc2116"></a>Ошибка компилятора ресурсов RC2116

Ожидается номер для идентификатора

Ожидается число для `id` поля оператора элемента управления в операторе **DIALOG** . Убедитесь, что у вас есть номер или `#define` оператор для идентификатора элемента управления.
