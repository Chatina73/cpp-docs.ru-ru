# <a name="expression-evaluator-error-cxx0030"></a>Ошибка вычислителя выражений CXX0030

неоцениваемое выражение

Средству оценки выражений отладчика не удалось получить значение выражения как записанное. Одна из возможных причин заключается в том, что выражение относится к памяти, которая находится за пределами адресного пространства программы (например, разыменование указателя NULL). Windows не разрешает доступ к памяти, которая находится за пределами адресного пространства программы.

Для управления порядком вычисления может потребоваться переписать выражение с помощью круглых скобок.

Эта ошибка идентична CAN0030.
