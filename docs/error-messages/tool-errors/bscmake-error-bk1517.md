# <a name="bscmake-error-bk1517"></a>Ошибка BSCMAKE BK1517

исходный файл для сбрфиле, скомпилированный с параметрами/Yc и/Yu

SBR файл ссылается на себя. Вероятно, она была перекомпилирована с параметром/Yu после компиляции с помощью/ИК. Сбросьте параметр компилятора для исходного файла в значение/Yc, а затем выберите **перестроить** , чтобы создать новые SBR файлы. Не перекомпилировать исходный файл с помощью/Ю.
