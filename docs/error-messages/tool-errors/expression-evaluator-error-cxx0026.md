# <a name="expression-evaluator-error-cxx0026"></a>Ошибка вычислителя выражений CXX0026

неправильная строка формата

Строка формата указана неправильно. Проверьте синтаксис выражения.

Эта ошибка идентична CAN0026.

Дополнительные сведения см. [в разделе описатели формата в C++](/visualstudio/debugger/format-specifiers-in-cpp).
