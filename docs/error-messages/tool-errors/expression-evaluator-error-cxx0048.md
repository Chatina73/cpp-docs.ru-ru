# <a name="expression-evaluator-error-cxx0048"></a>Ошибка вычислителя выражений CXX0048

последовательность вызовов не поддерживается

Функция, указанная в выражении, использует последовательность вызова, не поддерживаемую средством оценки выражений C. Эту функцию нельзя вызвать в выражении окно контрольных значений.

Эта ошибка идентична CAN0048.
