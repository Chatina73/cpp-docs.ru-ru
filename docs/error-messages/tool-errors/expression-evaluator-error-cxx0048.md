# <a name="expression-evaluator-error-cxx0048"></a>Ошибка вычислителя выражений CXX0048

последовательность вызова не поддерживается

Функции, указанной в выражении используется последовательность вызова, не поддерживается средством оценки выражений C. Не вызывайте эту функцию в выражения окна контрольных значений.

Эта ошибка идентична ошибке CAN0048.
