# <a name="project-build-warning-prj0042"></a>Предупреждение при построении проекта PRJ0042

> Не задано свойство "Outputs" настраиваемого этапа сборки для файла "*файл*". Пользовательский этап сборки будет пропущен.

Пользовательский этап сборки не был выполнен, так как выходные данные не указаны.

Чтобы устранить эту ошибку, выполните одно из следующих действий.

- Исключите настраиваемый шаг сборки из сборки.

- Добавьте выходные данные.

- Удалите содержимое команды настраиваемого этапа сборки.
