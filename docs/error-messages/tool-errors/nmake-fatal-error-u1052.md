# <a name="nmake-fatal-error-u1052"></a>Неустранимая ошибка NMAKE U1052

> файл "*ИмяФайла*" не найден

NMAKE не удалось найти указанный файл с одним из следующих элементов:

- **/F** , параметр

- **! ВКЛЮЧИТЬ** директиву предварительной обработки

- Описатель Sign ( **\@** ) для файла ответов

Убедитесь, что файл существует и имя файла указано правильно.
