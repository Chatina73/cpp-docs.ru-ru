# <a name="resource-compiler-fatal-error-rc1102"></a>Неустранимая ошибка компилятора ресурсов RC1102

Внутренняя ошибка: слишком много аргументов для RCPP

Передано слишком много аргументов препроцессору компилятора ресурсов. Уменьшить количество символов, определенных с определить символы (/ d), определив их в источнике. Это ошибка также может быть вызвана путем указания слишком многие включают пути поиска файлов, с помощью параметра путь поиска включаемых файлов (/ i).
