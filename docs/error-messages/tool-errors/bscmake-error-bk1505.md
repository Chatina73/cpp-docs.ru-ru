# <a name="bscmake-error-bk1505"></a>Ошибка BSCMAKE BK1505

не удается выполнить чтение из файла "имяфайла" [: причина]

BSCMAKE не может выполнить чтение из файла.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Поврежденный файл.

1. Усечение файла в компиляторе заканчивается свободное место на диске или оно прерывается при создании SBR-файла.
