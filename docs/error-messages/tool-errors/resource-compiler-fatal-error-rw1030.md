# <a name="resource-compiler-fatal-error-rw1030"></a>Неустранимая ошибка компилятора ресурсов RW1030

Ошибка вывода

Эта ошибка может быть вызвана, если компилятору ресурсов не удалось создать конечный файл. Убедитесь, что имеется достаточно места на диске и имеется разрешение на запись в томе.
