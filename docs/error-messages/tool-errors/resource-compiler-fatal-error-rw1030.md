# <a name="resource-compiler-fatal-error-rw1030"></a>Неустранимая ошибка компилятора ресурсов RW1030

Ошибка вывода

Эта ошибка может быть вызвана тем, что компилятору ресурсов не удалось создать конечный файл. Убедитесь, что на диске достаточно места и что у вас есть разрешение на запись в том.
