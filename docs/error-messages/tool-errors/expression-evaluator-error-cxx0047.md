# <a name="expression-evaluator-error-cxx0047"></a>Ошибка вычислителя выражений CXX0047

список аргументов не соответствует функции

Выражение вызвало функцию с фактическим списком параметров, который не соответствует списку формальных параметров любой функции с тем же именем, определенным в программе.

Перегруженные функции могут вызываться только в том случае, если имеется точное совпадение параметров или соответствие, которое не требует создания объекта.

Эта ошибка идентична CAN0047.
