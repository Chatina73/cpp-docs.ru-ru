# <a name="resource-compiler-warning-rw4003"></a>Предупреждение компилятора ресурсов RW4003

Клавиша SHIFT или CONTROL использовалась VIRTKEY

В ресурсе таблицы сочетаний клавиш SHIFT или CONTROL требуется VIRTKEY. Поскольку SHIFT и CONTROL обозначены как флаговые биты в сочетании клавиш VIRTKEY, они не могут существовать независимо от VIRTKEY.
