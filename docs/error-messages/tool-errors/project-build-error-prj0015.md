# <a name="project-build-error-prj0015"></a>Ошибка построения проекта PRJ0015

В системе отсутствует устройство со значением NULL. Не удалось запустить сборку.

Эта ошибка может быть вызвана нехваткой системных ресурсов. Закройте некоторые приложения или перезагрузите компьютер.

Это также может быть вызвано отсутствием необходимых прав доступа к устройству со значением NULL на компьютере.
