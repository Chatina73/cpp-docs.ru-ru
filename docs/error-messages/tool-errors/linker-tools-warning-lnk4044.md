# <a name="linker-tools-warning-lnk4044"></a>Предупреждение средств компоновщика LNK4044

Нераспознанный параметр "параметр"; игнорируют

Данный параметр не является допустимым для этого инструмента. Средство игнорирует этот параметр.

Чтобы просмотреть список допустимых параметров, запустите средство без аргументов или параметров.
