# <a name="nmake-fatal-error-u1000"></a>Неустранимая ошибка NMAKE U1000

> Синтаксическая ошибка: ")" отсутствует в вызове макроса

Левая круглая скобка **((**) без соответствующей правой скобки, **)** в вызове макроса. Правильная форма — **$ (**<em>Name</em>**)**; **$** <em>n</em> разрешено использовать для односимвольных имен.
