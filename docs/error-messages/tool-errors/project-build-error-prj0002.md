# <a name="project-build-error-prj0002"></a>Ошибка построения проекта PRJ0002

> результат ошибки, возвращенный из "*командной строки*".

Командная *командной строки*, который был сформирован из введенных пользователем данных в **страницы свойств** диалоговое окно, возвращается код ошибки, но сведения не будут отображаться в **выходные данные** окно .

Разрешение для этой ошибки зависит от того, какое средство сгенерировал ошибку. Для MIDL вы получите представление о том, что пошло не так, если определен /o (перенаправление вывода).

Пакетный файл, например настраиваемого этапа сборки или события сборки, который не информативные сведения о сбоях, также может быть причиной этой ошибки.
