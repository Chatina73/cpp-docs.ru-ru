# <a name="project-build-error-prj0002"></a>Ошибка построения проекта PRJ0002

> результат ошибки, возвращенный из "*командной строки*".

Команда, *Командная строка*, сформированная на основе пользовательского ввода в диалоговом окне **страницы свойств** , возвращает код ошибки, но в окне **вывод** не отображаются сведения.

Решение этой ошибки зависит от того, какая программа вызвала ошибку. Для MIDL вы получите представление о том, что пошло не так, если определено/o (вывод перенаправления).

Пакетный файл, например настраиваемый этап сборки или событие сборки, который не является информативным в отношении состояния сбоя, также может быть причиной этой ошибки.
