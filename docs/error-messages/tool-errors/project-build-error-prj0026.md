# <a name="project-build-error-prj0026"></a>Ошибка построения проекта PRJ0026

> Файл ответов "*файл*", содержащий знаки Юникода, не может быть преобразована в кодовую страницу ANSI пользователя.
>
> *Содержимое файла в кодировке Юникод*

Система проектов найти содержимое в кодировке Юникод в файле ответов, не может правильно преобразован в текущую кодовую страницу ANSI пользователя.

Решение для этой ошибки заключается в том, для обновления содержимого файла ответа в формате ANSI или установите кодовую страницу на компьютере и установите ее в качестве системной настройки по умолчанию.
