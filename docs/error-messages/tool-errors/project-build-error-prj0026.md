# <a name="project-build-error-prj0026"></a>Ошибка построения проекта PRJ0026

> Файл ответов "*файл*" содержит содержимое в Юникоде, которое не может быть преобразовано в кодовую страницу ANSI пользователя.
>
> *Содержимое файла в Юникоде*

Система проектов обнаружила в файле ответов содержимое в Юникоде, которое не может быть правильно преобразовано в текущую кодовую страницу ANSI пользователя.

Решение этой ошибки заключается в обновлении содержимого файла ответов для использования ANSI или установки кодовой страницы на компьютере и установке ее в качестве системы по умолчанию.
