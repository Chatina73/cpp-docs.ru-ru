# <a name="project-build-error-prj0019"></a>Ошибка построения проекта PRJ0019

Средство вернуло код ошибки из

Уровень ошибки был ненулевым для настраиваемого этапа сборки или события сборки.

Вы также увидите PRJ0019, когда средство вернуло код ошибки, но не выдает сообщение об ошибке. Это может произойти, например, при перенаправлении выходных данных MIDL в NUL.

Дополнительные сведения см. в разделе [Устранение неполадок настраиваемых шагов сборки и событий сборки](../../build/troubleshooting-build-customizations.md) .

Эта ошибка также может возникать, если вы работаете в качестве члена группы пользователей и требуется административный доступ. Дополнительные сведения см. в разделе [выполнение от имени члена группы "Пользователи"](../../security/running-as-a-member-of-the-users-group.md).
