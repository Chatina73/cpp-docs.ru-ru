# <a name="expression-evaluator-error-cxx0041"></a>Ошибка вычислителя выражений CXX0041

элемент класса должен быть статическим членом или функцией члена

Нестатический член класса (или структуры или объединения) использовался без указания того, какой экземпляр класса следует использовать.

Без указания экземпляра можно использовать только статические элементы данных или функции элементов.

Эта ошибка идентична CAN0041.
