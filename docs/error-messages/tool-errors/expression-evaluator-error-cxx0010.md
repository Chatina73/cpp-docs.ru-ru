# <a name="expression-evaluator-error-cxx0010"></a>Ошибка вычислителя выражений CXX0010

отсутствует левая скобка

Выражение содержит непарные квадратные скобки. Введите выражение с сбалансированными квадратными скобками.

Эта ошибка идентична CAN0010.
