# <a name="expression-evaluator-error-cxx0058"></a>Ошибка вычислителя выражений CXX0058

перегруженный оператор не найден

Тип класса был указан в качестве левого операнда в выражении, но перегруженный оператор не был определен для класса.

Эта ошибка идентична CAN0058.
