# <a name="bscmake-error-bk1504"></a>Ошибка BSCMAKE BK1504

не удается разместить в файле "filename" [: причина]

BSCMAKE не удалось переместить в расположение в файле.

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Полный диск.

1. Аппаратная ошибка.

1. Усечение файла из-за нехватки места на диске или при прерывании компиляции компилятора при создании файла. SBR.
