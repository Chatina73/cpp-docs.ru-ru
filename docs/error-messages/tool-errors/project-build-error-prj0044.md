# <a name="project-build-error-prj0044"></a>Ошибка построения проекта PRJ0044

Свойство "Дополнительные зависимости" для настраиваемого правила сборки "правило", назначенное файлу "файл", является недопустимым. Свойство содержит строку "строка", которая принимает "значение".

**Дополнительное свойство зависимостей** вычисляется как пустая строка или строка, содержащая недопустимые символы (любой символ, который не может находиться в имени файла или каталога). Настраиваемым правилам сборки требуются выходные данные действия сборки.

Дополнительные сведения см. в разделе [Указание настраиваемых средств сборки](../../build/specifying-custom-build-tools.md).
