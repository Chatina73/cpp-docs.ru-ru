# <a name="nmake-fatal-error-u1035"></a>Неустранимая ошибка NMAKE U1035

Синтаксическая ошибка: требуется разделитель ":" или "="

Ожидался либо двоеточие ( **:** ), либо знак равенства ( **=** ).

### <a name="to-fix-by-checking-the-following-possible-causes"></a>Чтобы устранить ошибку, проверьте указанные ниже возможные причины ее возникновения.

1. Двоеточие не соответствует целевому объекту.

1. Двоеточие без пробела (например,:) за ним следует подписка с одной буквой. NMAKE интерпретирует его как спецификацию накопителя.

1. Двоеточие не соответствует правилу вывода.

1. За определением макроса не следует знак равенства.

1. Символ, за которым следует обратная косая черта ( **\\** ), которая использовалась для продолжения команды на новую строку.

1. Строка, которая не соответствует правилу синтаксиса NMAKE.

1. Файл Makefile отформатирован текстовым процессором.
