# <a name="resource-compiler-warning-rc4002"></a>Предупреждение компилятора ресурсов RC4002

слишком много фактических параметров для макроопределения "идентификатор"

Число фактических параметров, указанных с помощью заданного идентификатора, превышает число формальных параметров, заданных в определении макроса идентификатора.

Дополнительные фактические параметры были собраны, но проигнорированы во время расширения макроса.
