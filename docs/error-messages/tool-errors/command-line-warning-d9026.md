# <a name="command-line-warning-d9026"></a>Предупреждение командной строки D9026

параметры применяются ко всей командной строке

Для команды был указан параметр после указания имени файла. Параметр был применен к файлу, предшествующему ему.

Например, в команде

```
CL verdi.c /G5 puccini.c
```

файл Верди. c будет скомпилирован с помощью параметра/G5, а не/G4 по умолчанию.

Это поведение отличается от некоторых предыдущих версий, которые применяют только параметры, указанные до имени файла, что приводит к компиляции Верди. c с помощью/G4 и ПУКЦИНИ. c, компилируемого с помощью/G5.
