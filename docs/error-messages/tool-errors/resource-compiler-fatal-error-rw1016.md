# <a name="resource-compiler-fatal-error-rw1016"></a>Неустранимая ошибка компилятора ресурсов RW1016

Версия-кандидат завершается после ошибок препроцессора

Компилятор ресурсов остановлен из-за других ошибок. Устраните другие ошибки и выполните повторную компиляцию.
