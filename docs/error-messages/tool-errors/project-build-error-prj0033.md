# <a name="project-build-error-prj0033"></a>Ошибка построения проекта PRJ0033

Свойство "Дополнительные зависимости" для настраиваемого этапа сборки файла "файл" содержит "макрос", который вычисляется как "macro_expansion".

Настраиваемый этап сборки для файла содержал ошибку в дополнительной зависимости, вероятно, из-за проблемы с оценкой макроса. Эта ошибка также может означать, что путь неправильно сформирован, содержит символы или сочетания символов, которые не являются допустимыми в пути к файлу.

Чтобы устранить эту ошибку, исправьте макрос или исправьте спецификацию пути. Вычисленный путь представляет собой абсолютный путь из каталога проекта.
