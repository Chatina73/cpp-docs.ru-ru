# <a name="linker-tools-warning-lnk4205"></a>Предупреждение средств компоновщика LNK4205

в файле "filename" отсутствуют текущие отладочные данные для ссылки на модуль; Компоновка объекта как при отсутствии отладочной информации

PDB-файл содержит устаревшую информацию. Компоновщик продолжит компоновку объекта без отладочной информации. Возможно, потребуется перекомпилировать объектный файл с помощью параметра [/Zi](../../build/reference/z7-zi-zi-debug-information-format.md) .
