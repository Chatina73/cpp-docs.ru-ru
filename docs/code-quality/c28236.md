# <a name="c28236"></a>C28236

> Warning C28236: Аннотация не может использоваться в выражении

Это предупреждение означает, что была предпринята попытка использования функции, объявленной как Аннотация в контексте выражения. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается в типичном использовании.
