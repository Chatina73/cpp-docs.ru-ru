# <a name="c28244"></a>C28244

> Warning C28244: Аннотация для функции имеет неанализируемый параметр или внешнюю аннотацию

В настоящее время это невозможно, если стандартные макросы используются для заметок; Это предупреждение не ожидается в типичном использовании.
