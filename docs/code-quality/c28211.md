# <a name="c28211"></a>C28211

> предупреждение C28211: для SAL_context ожидалось статическое имя контекста

Это предупреждение означает, что операнд в `_Static_context_` заметке должен быть именем определенного средством контекста. Это предупреждение никогда не должно возникать при обычном использовании.
