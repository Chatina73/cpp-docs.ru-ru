# <a name="c28204"></a>C28204

> Warning C28204: \<function> : только одна из этих перегрузок и только одна из них \<filename> ( \<line> ) записываются в комментарии для \<paramname> следующих элементов: или ни одной, ни одной аннотации.

Это предупреждение появляется при возникновении ошибки в заметках.
