# <a name="c28288"></a>C28288

> предупреждение C28288: для функции, синтаксическая ошибка в \_ \_ заметке at () (недопустимое имя параметра)

Средство анализа кода сообщает об этом предупреждении, `SAL_at` когда `__drv_at` используется аннотация () и выражение параметра не может быть интерпретировано в текущем контексте. Это может быть неправильное написание имени параметра или члена или неправильное использование ключевых слов "return" или "this".
