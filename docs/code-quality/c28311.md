# <a name="c28311"></a>C28311

> Warning C28311: <annotation_name> Аннотация On \<function> \<parameter> является УСТАРЕЛА версией SAL.

Заметка является старой версией и должна быть обновлена до эквивалентной [аннотацией SAL2](../code-quality/using-sal-annotations-to-reduce-c-cpp-code-defects.md). Это предупреждение не выдается, если ранее было выдано предупреждение о несогласованном примечании; оно выводится только один раз для каждого объявления. Проверьте остальную часть объявления на наличие других устаревших SAL.
