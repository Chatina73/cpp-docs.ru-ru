# <a name="c28289"></a>C28289

> предупреждение C28289: для функции: ReadableTo или WritableTo не содержит спецификацию ограничения в качестве параметра

Средство анализа кода сообщает об этом предупреждении при некорректном кодировании аннотации функции или параметра, как указано.
