# <a name="c28195"></a>C28195

> предупреждение C28195: функция объявлена как получающая память в переменной, однако она не выполнила эту операцию до своего выхода

Это предупреждение означает, что прототип функции для анализируемой функции имеет `__drv_acquiresMemory` заметку. `__drv_acquiresMemory` Аннотация указывает, что функция получает память в указанном расположении результатов, но по крайней мере в одном пути функция не получала память. Обратите внимание, что средство анализа кода не распознает фактическую реализацию распределителя памяти (включающее арифметические действия с адресом) и не распознает выделенную память (хотя многие оболочки будут распознаны). В этом случае средство анализа кода не распознает, что память была выделена, и выдает это предупреждение. Чтобы отключить ложное срабатывание, используйте `#pragma` предупреждение в строке, предшествующей открывающей фигурной скобке `{` тела функции.
