# <a name="c28021"></a>C28021

> предупреждение C28021: параметр \<param> с заметками \<anno> должен быть указателем

Это предупреждение выводится в том случае, если объект с заметками не является типом указателя. Эту аннотацию нельзя использовать с **`void`** целочисленными типами или.
