# <a name="c28217"></a>C28217

> Warning C28217: Function. число параметров в заметке не совпадает с количеством аргументов, найденных в файле

Заметки в текущей строке и в строке сообщения не согласуются. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается при обычном использовании.
