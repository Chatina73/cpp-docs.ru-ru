# <a name="c28215"></a>C28215

> Warning C28215: typefix нельзя применить к параметру, который уже имеет typefix

Применение заметки `__typefix` к параметру, который уже имеет эту заметку, является ошибкой. Заметки `__typefix` используются только в нескольких особых случаях, и это предупреждение не ожидается при обычном использовании.
