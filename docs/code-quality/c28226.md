# <a name="c28226"></a>C28226

> Warning C28226: Аннотация не может также быть PrimOp (в текущем объявлении)

Это предупреждение означает, что именованная Аннотация объявляется как PrimOp, а также ранее была объявлена как обычная Аннотация. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается при обычном использовании.
