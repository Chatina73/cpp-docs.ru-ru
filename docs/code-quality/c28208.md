# <a name="c28208"></a>C28208

> Warning C28208: функция \<function> ранее была определена с другим списком параметров в \<file> ( \<line> ). Некоторые средства анализа выдают неверные результаты

Это предупреждение выводится, когда известное определение функции не соответствует другому вхождению.
