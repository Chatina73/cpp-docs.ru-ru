# <a name="c6031"></a>C6031

> предупреждение C6031: возвращаемое значение пропущено: *функция, вызываемая* , может возвращать непредвиденное значение

Это предупреждение означает, что вызывающий объект не проверяет возвращаемое значение функции для ошибки. В зависимости от того, какая функция вызывается, эта ошибка может привести к неверному случайному поведению программы. Это включает сбои и повреждения данных в условиях ошибки или в ситуациях с низкими ресурсами.

Как правило, нельзя считать, что вызовы функций, запрашивающих диск, сеть, память или другие ресурсы, будут выполнены. Вызывающая сторона всегда должна проверять возвращаемое значение и обрабатывали случаи ошибок соответствующим образом. Также рассмотрите возможность использования аннотации `_Must_inspect_result_`, которая проверяет, является ли значение полезным способом.

## <a name="example"></a>Пример

Это предупреждение создается в следующем коде:

```cpp
#include <stdio.h>
void f( )
{
    fopen( "test.c", "r" ); // C4996, C6031 return value ignored
    // code ...
}
```

Чтобы исправить это предупреждение, проверьте возвращаемое значение функции, как показано в следующем коде:

```cpp
#include <stdio.h>
void f( )
{
    FILE *stream;
    if ( (stream = fopen( "test.c", "r" )) == NULL )
        return;
    // code ...
}
```

В следующем коде для исправления этого предупреждения используется `fopen_s`ная функция.

```cpp
#include <stdio.h>
void f( )
{
    FILE *stream;
    errno_t err;

    if ( (err = fopen_s( &stream, "test.c", "r" )) !=0 )
    {
        // code ...
    }
}
```

Это предупреждение также создается, если вызывающий объект игнорирует возвращаемое значение функции, помеченной свойством `_Check_return_`, как показано в следующем коде.

```cpp
#include <sal.h>
_Check_return_ bool func();

void test_f()
{
    func(); //  Warning C6031
}
```

Чтобы исправить предыдущее предупреждение, проверьте возвращаемое значение, как показано в следующем коде:

```cpp
#include <sal.h>
_Check_return_ bool func();

void test_f()
{
    if ( func() ) {
        // code ...
    }
}
```
