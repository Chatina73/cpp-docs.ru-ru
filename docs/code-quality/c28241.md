# <a name="c28241"></a>C28241

> Warning C28241: Аннотация для функции в параметре не распознана

Использовано нераспознанное имя заметки. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается в типичном использовании.
