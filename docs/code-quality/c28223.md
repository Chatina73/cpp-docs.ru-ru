# <a name="c28223"></a>C28223

> предупреждение C28223: не найден ожидаемый токен или идентификатор для аннотации, параметр

Это предупреждение означает, что параметр в заметке должен быть идентификатором, а обнаружен другой символ. Обычно это означает неправильное написание макроса аннотации. Использование ключевых слов C или C++ в этой должности вызовет эту ошибку.
