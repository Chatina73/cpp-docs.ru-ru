# <a name="c28219"></a>C28219

> Warning C28219: ожидался член перечисления для аннотации параметра в заметке

Параметр в заметке должен быть членом именованного типа `enum`, и обнаружен другой символ. Используйте член этого типа `enum`. Обычно это означает неправильное написание макроса аннотации.
