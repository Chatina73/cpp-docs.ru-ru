# <a name="c28219"></a>C28219

> Warning C28219: ожидался член перечисления для аннотации параметра в заметке

Параметр в заметке должен быть членом именованного **`enum`** типа, и обнаружен другой символ; используйте член этого **`enum`** типа. Обычно это означает неправильное написание макроса аннотации.
