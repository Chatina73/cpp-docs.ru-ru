# <a name="c28194"></a>C28194

> Warning C28194: функция объявлена как псевдоним значения в переменной и завершила работу без этого

Это предупреждение означает, что прототип функции для анализируемой функции имеет `__drv_isAliased` аннотацию, которая указывает, что он будет иметь *псевдоним* для указанного аргумента (то есть присвоить значение таким образом, чтобы оно сохранило бы возврат из функции). Однако функция не применяет псевдоним аргумента вдоль пути, обозначенного заметкой. Большинство функций, которые имеют псевдоним для переменной, сохраняют свое значение в глобальной структуре данных.
