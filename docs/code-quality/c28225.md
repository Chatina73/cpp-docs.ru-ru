# <a name="c28225"></a>C28225

> предупреждение C28225: не удалось найти правильное число обязательных параметров в аннотации

Это предупреждение означает, что Именованная заметка используется с неправильным числом параметров. Это не должно быть возможным, если стандартные макросы используются для заметок; Это предупреждение не ожидается в типичном использовании.
