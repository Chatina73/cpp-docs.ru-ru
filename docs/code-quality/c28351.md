# <a name="c28351"></a>C28351

> Warning C28351: Аннотация \<annotation> описывает, где динамическое значение (переменная) нельзя использовать в условии.

Это предупреждение выводится при применении заметки при проверке типа C/C++.

Выражение в выражении `_When_` должно иметь константу. Параметр `_When_` игнорируется.
