# <a name="c28351"></a>C28351

> Warning C28351: Аннотация заметки \<> описывает, где динамическое значение (переменная) нельзя использовать в условии.

Это предупреждение выводится при применении заметки при проверке типа C/C++ .

Выражение в `_When_` должно оцениваться как константа. `_When_` игнорируется.
