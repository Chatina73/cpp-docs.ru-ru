# <a name="c28214"></a>C28214

> предупреждение C28214: имена параметров атрибутов должны быть P1... P9

Это предупреждение означает, что при создании объявления заметки имена параметров ограничены P1... P9. Это предупреждение никогда не должно возникать при обычном использовании.
