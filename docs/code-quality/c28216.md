# <a name="c28216"></a>C28216

> Warning C28216: Проверка \_\_вернуть\_ Аннотация применяется только к последующим условиям для конкретного параметра функции.

`_Check_return_` Аннотация применена в контексте, отличном от `__post`; может потребоваться модификатор `__post` (или `__drv_out`) или он может быть неправильно размещен.
