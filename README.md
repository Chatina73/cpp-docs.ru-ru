# <a name="visual-studio-documentation-for-microsoft-c"></a>Документация по Visual Studio для Microsoft C++

Добро пожаловать! Этот репозиторий содержит исходные файлы для технической документации по проводимым работам для Microsoft C++ (MSVC или Visual C++). Эти статьи опубликованы на [сайте документации по C++ в Visual Studio](https://docs.microsoft.com/cpp).

Документация для Visual Basic и Visual C# находится в отдельном репозитории по адресу [https://github.com/dotnet/core-docs](https://github.com/dotnet/core-docs), а документация по Visual Studio — в репозитории по адресу [https://github.com/Microsoft/visualstudio-docs](https://github.com/Microsoft/visualstudio-docs).

## <a name="contributing-to-the-documentation"></a>Участие в работе над документацией

Ждем ваших публикаций, призванных помочь улучшить документацию по MSVC. Полное руководство по отправке документации на сайт docs.microsoft.com см. в [руководстве участника для документации Майкрософт](https://docs.microsoft.com/contribute). Сведения о том, как отправить свою документацию по MSVC, см. в [руководстве участника](CONTRIBUTING.md).

Некоторые области MSVC имеют в этом репозитории собственные папки, например `standard-library` для стандартной библиотеки C++, `ide` для интегрированной среды разработки (IDE) Visual Studio и т. д. Вложенная папка `/media` в каждой из папок содержит ART-файлы для соответствующих статей. Дополнительные сведения см. в [руководстве по участию](CONTRIBUTING.md).
